Create a Code Table

You can create a code table and add codes to it.

To create a code table, do the following:

  1. Click Configure > Code Tables from the menu bar to display the Code Tables page.



  2. Click the New button to launch the New Code Table page.



  3. Enter the following information in fields:

    Field Description
    Name Enter a unique name for this code table. This field is for internal use and is not supported by language translation.
    Active This field indicates whether this table is active (selected, by default) or not (unchecked). An inactive code table does not appear in any lists or finders.
    Project Specify the project name that this code table belongs to.
    Description Description of this Code Table.
    Sort Order

    When you select this field, all codes appear alphabetically sorted. Otherwise, when this field is not selected, codes appear in the order in which they were added to the code table.

  4. Click the Save button. A confirmation message appears at the top of the page, indicating that your code table has been successfully created.



    The Code area appears, allowing you to add a new code.

Create a Code in a Code Table

To create a new code in your code table, do the following:

  1. In the Code area, click the New button.



  2. The New Code dialog appears.



    Complete the following fields:

    Field Description
    Code Enter a unique name for this code .
    Code Table Click the drop-down menu and select the code table name that this code belongs to.
    Description Enter a description for this code.
    Active This field indicates whether this code is active (selected, default value) or not. An inactive code does not appear in any lists or finders.

  3. Click the Save button to save your new code. A comfirmation message appears, indicating that your code has been successfully created. Click the Close button to close the dialog.



  4. The Details page reappears. You newly created code appears in blue font. Click the Save button to save your code table.



    The blue font of your newly created code changes to black font and a confirmation message appears, indicating that your changes have been successfully saved.
