Edit and Delete a Code Table

You can edit an existing code table. You can also delete codes within a code table and delete the code table itself.

Edit a Code Table

To edit a code table, do the following:

  1. From the Code Tables page, select the checkbox next to the code table Name that you want from the search results, and then select the Details button. Alternatively, you can double-click the code table that you want to launch the code table's Details page.
  2. From the code table's Details page, click the Edit button.



  3. You have the option of editing, creating, or deleting a code.



  4. Make the pertinent changes to your code or table, and then click the Save button to save your changes.

Delete a Code

To delete a code from a code table, complete these steps:

  1. From the code table's Details page, select the checkbox next to the Code that you want to delete from the list, and then click the Delete button.



  2. A confirmation message appears. Click the Yes button to delete the selected code. The page refreshes and your code appears in strikethrough font. Proceed to click the Save button.



  3. The code tables list automatically refreshes and your selected code has been deleted.

Delete a Code Table

To delete a code table, perform these steps:

  1. From the Code Tables page, select the checkbox next to the code table Name that you want to delete, and then click the Delete button.



  2. A confirmation message appears. Click the Yes button to delete the selected code table. The code tables list automatically refreshes and your code has been deleted.
