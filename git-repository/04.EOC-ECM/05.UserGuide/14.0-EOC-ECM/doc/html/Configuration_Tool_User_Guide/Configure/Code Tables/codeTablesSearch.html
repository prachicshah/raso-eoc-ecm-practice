Search for a Code Table

You can perform a code table search by following these steps:

  1. Click Configure > Code Tables from the menu bar to display the Code Tables page.



  2. Enter your search criteria in any of the following fields:

    Field Description
    Code Table Name Enter the name of the code table that you want to find.
    Code Table Description Enter keywords to search within the description field of the code table that you are searching for.
    Project Enter the project name that the code table belongs to.
    Code Name

    Enter the code that is found inside a code table.
    Code Description

    Enter keywords to search within the code's Description field.

  3. Click the Search button to view a list of code tables.



  4. To view a specific code table, double-click the one that you want from the search results are. The code table's Details page appears.
