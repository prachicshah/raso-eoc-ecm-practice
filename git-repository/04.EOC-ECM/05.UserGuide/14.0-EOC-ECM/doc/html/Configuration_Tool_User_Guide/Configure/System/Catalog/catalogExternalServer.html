Catalog External Server

Clicking Catalog > Catalog External Server allows you to define one or more catalog remote servers. A list of catalog servers appears, allowing you to add a new catalog server or remove any catalog server from this list.

The following fields are available on this page:

Field Description
Fetch Version Code Table from Server

By default, this field's checkbox is selected, meaning that the catalog client always reads the versioned code table from the server. Otherwise, the versioned code table defined in the catalog client overrides the versioned code table in the catalog server.

Note: A non-versioned code table always overrides that on the server.
Host This field denotes the catalog remote server name or IP address.
Port

This field represents the catalog server port. By default, this field is set to port 0.

You can perform the following tasks from the Catalog External Server page:

  • Add a new catalog server
  • Delete a catalog server

Add a Catalog Server

To add a catalog server, complete these steps:

  1. From the Catalog External Server page, click the Add button.



  2. A blank row appears in the table. Click the Host field and enter the name of your catalog server.



  3. Click the Port field and enter the port number for your catalog server.
  4. When you have finished making your changes, click the Save button. A confirmation message appears, indicating that you have successfully added your catalog server.

Delete a Catalog Server

To delete a catalog server, do the following:

  1. From the Catalog External Server page, select the checkbox next to the Host that you want to delete, and then click the Delete button.



  2. A dialog appears, prompting you to confirm deleting the catalog server. Click the Yes button.
  3. The page refreshes and your selected catalog server has been deleted.
