Logical Connection

Clicking Databases > Logical Connection displays the logical connections to the database, which are responsible for enabling the AVM to send database commands to the database, to carry out different functionalities.

Your logical connections must be defined to run your metadata. However, you must have properly defined in the Physical Connection tab before you manage your logical connections.

By default, the Logical Connection tab contains three logical connections in the table:

  • ARCHIVE, which is the connection that is used by the AVM to archive database records
  • ORDER, which is the main logical connection for the AVM
  • PE_ORDER, which is the connection that is used by process engines

Notes:

  • Logical connections are only editable at the cluster level. They are read-only at the node level.
  • Do not remove the three default logical connections.
  • When setting up the application under normal circumstances as described in the Installation Guide, you have one physical database connection. All you need to do is to associate all three logical connections to that physical connection. However, in advanced application development and optimization, you can have the flexibility of multiple physical connections to separate database operations based on functionality.

You can perform the following tasks from the Logical Connection page:

  • Add a logical connection
  • Edit a logical connection
  • Delete a logical connection

Note: It is recommended to restart JBoss if you make changes to configuration variables.

Add a Logical Connection

To add a logical connection, complete these steps:

  1. From the Logical Connection page, click the Add button.



  2. A blank row appears in the table. Click the Name field and enter the logical connection's name.
  3. Click the Physical Connection field's drop-down menu and select the physical connection that you want to associate to your logical connection from the list.



  4. When you have finished adding your logical connection, click the Save button to save your configuration.

Edit a Logical Connection

To edit a logical connection, click the field that you want and make your changes. When you have completed them, click the Save button to save your configuration.

Delete a Logical Connection

To delete a logical connection, follow these steps:

  1. From the Logical Connections page, select the checkbox next to the logical connection that you want to delete, and then click the Delete button.



  2. A confirmation dialog appears. Click the Yes button to confirm the logical connection's deletion.
  3. The selected logical connection no longer appears on the page. Click the Save button to save your configuration.
