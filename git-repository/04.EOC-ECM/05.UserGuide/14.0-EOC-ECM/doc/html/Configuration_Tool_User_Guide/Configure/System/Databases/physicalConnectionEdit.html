Edit a Physical Connection

To edit a physical connection, complete the following steps:

  1. From the Physical Connection page, select the checkbox next to the Connection Name that you want, and then click the Edit button.



  2. The Edit Physical Connection dialog appears, which allows you to make any changes to your connection settings.



  3. When you have finished making your changes, click the Apply button, and then click the Close button to return to the Physical Connection page.
  4. Click the Save button to save your configuration.
