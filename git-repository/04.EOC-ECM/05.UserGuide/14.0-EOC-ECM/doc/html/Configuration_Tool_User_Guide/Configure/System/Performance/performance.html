Performance

The Performance section contains the following pages:

  • AVM Cache
    This page displays the AVM cache list, which allows you to control all data caching stored in the AVM.

  • Catalog Cache
    This page shows the catalog cache list, which controls all data caching stored in the Catalog application.

  • Statistic Monitoring
    This page displays the list of formatted strings called the statistics monitoring specification, which controls statistics gathering and reporting.

  • Action Categories
    This page provides additional server-side control to prevent server overload.
