Statistic Monitoring

Complex parameter-driven objects implement the AVM's internal mechanisms. Information about the current state of these objects helps understanding of how healthy the AVM is, what are the current bottlenecks, if any exist, and what has to be done to increase the AVM's performance.

Each AVM cache listed in JConsole allows you to view, manage, and change statistic counters. Each counter provides data for one runtime operational parameter of each AVM cache. The counter values are positive integers. See AVM Caches Domain for the various counters that you can use for each AVM cache.

Clicking Performance > Statistic Monitoring displays the list of formatted string called the statistics monitoring specification or SM specification. This specification provides full control over the statistics gathering and reporting by specifying what statistics counters are to be displayed and how they are to be displayed. See the Statistics monitoring section for more information on the specification.

You can perform the following statistics monitoring-related tasks:

  • Enable statistics in JConsole.
  • Add a statistics monitoring specification
  • Delete a statistics monitoring specification

Enable Statistics Monitoring

By default, statistics monitoring is turned off. It can be turned on directly from JConsole.

Note: Benchmark counters can be increased. Calling a function with the argument (N <=0) adds a 1 to the existing counter value.

Add a Statistics Monitoring Specification

To add a statistics monitoring specification, complete these steps:

  1. Click the Add button to add an empty row to the table.



  2. Enter the Name of your statistics monitoring specification.
  3. Proceed to enter the Specification details. See the Statistics monitoring section for syntactical details.
  4. When you have finished making your changes, click the Save button to save your configuration.

Delete a Statistics Monitoring Specification

To delete a statistics monitoring specification, complete these steps:

  1. Select the checkbox next to the statistics monitoring specification that you want, and then click the Delete button.



  2. A confirmation message appears. Click the Yes button to confirm.
  3. Your selected statistics monitoring specification has been successfully removed from the list.
