Alerts Addresses

Clicking Processes > Alerts Addresses displays alerts that have been defined in Velocity Studio, their respective alert type, and the e-mail address specified for each alert.

Note: If no alerts have been defined in the metadata in Velocity Studio, this tab is disabled.

Configure and Activate an Alert Assignment

To configure and activate an alert assignment, do the following:

  1. Click the Type field next to the alert name that you want and complete the field.
  2. Click the Email drop-down menu and enter the e-mail address to send to when the alert is raised by that process type.
  3. When you have finished making your changes, click the Save button.

This page contains the following fields:

Field Description
Alert This field contains the alert label, consisting of your namespace and the name of your alert, separated by a period (for example, TestNamespace.TestAlert).
Type This field denotes the type of alert (for example, pager, e-mail, worklist, or external system).
Email This field contains the e-mail address specified for the alert. You can edit this value clicking this field's cell and entering an e-mail address, If you change this value, the value in the Configuration application overrides the one specified in Velocity Studio.
