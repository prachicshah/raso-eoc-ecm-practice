Alerts Assignment

Clicking Processes > Alerts Assignment displays alerts that should be raised (sent) when a process failure (error) occurs. See dispatching alerts for details.

Note: If no alerts have been defined in the metadata in Velocity Studio, this tab is disabled.

Activate an Alert

To activate an alert, do the following:

  1. Expand the Process that you want a list of alerts associated with it.
  2. To enable an alert, click the Enable checkbox next to the alert that you want.
  3. When you have finished making your changes, click the Save button.

This page contains the following fields:

Field Description
Process This field represents the process name. Click this field's node to see a list of alerts.
Alert This field contains the alert associated with the process.
Enable This field indicates whether the alert is enabled. By default, the alert is disabled. Select this field to enable the alert.
