Delete Messages

Clicking Processes > Delete Messages allows you to enable the deletion of participant messages in intervals, to maintain database efficiency.

Configure Message Deletion Parameters

To configure your parameters for deleting messages, do the following:

  1. Edit the pertinent parameters that you want.
  2. When you have finished making your changes, click the Save button.

This page contains the following fields:

Field Description
Enable If checked, this property allows you to specify parameters for deleting messages. By default, this property is unchecked, meaning that deleting messages parameters are disabled and are ignored.
Loop Delay This parameter specifies the wait period between each deletion loop or interval.
Max Messages / Loop This parameter indicates the number of messages to process during each interval.
Schedule

This parameter allows you to specify the message-deleting process (that is, when the message-deleting process starts). Click the drop-down menu and select from one of the following options:

  • Daily
  • Weekly
  • Monthly
Run at You can specify the hour and minute (from midnight) in the Hour and Min fields, respectively, to start the message-deleting process when scheduling is to run for that day.
Purge Delay You can specify the interval for the message-deleting process (that is, for how long the message-deleting process runs). Indicate the number of days, hours, and minutes in the Day, Hour, and Min fields, respectively.
