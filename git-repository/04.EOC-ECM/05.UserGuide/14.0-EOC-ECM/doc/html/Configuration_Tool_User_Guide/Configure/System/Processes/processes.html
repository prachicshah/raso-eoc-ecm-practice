Processes

The Processes section contains the following pages:

  • General
    This page allows you to view and configure general process parameters.

  • PE Processes
    This page displays and allows you to configure the workload allocated to the process engine for each process type.

  • PE Queues
    This page allows you to specify queue options, which the process engine uses queues to implement process activities.

  • Alerts Addresses
    This page displays alerts that have been defined in Velocity Studio, their respective alert type, and the e-mail address specified for each alert.

  • Alerts Assignment
    This page shows alerts that should be raised (sent) when a process failure (error) occurs.

  • Purge
    This page allows you to specify purging options. The process consists of deleting information from system and order tables.

  • Delete Messages
    This page allows you to enable the deletion of messages in intervals, to maintain database efficiency.

  • Dynamic PE Load Balancing
    Dynamic load balancing allows running process engines (PEs) to dynamically distribute the load of dead PEs between themselves, as well as dynamically reduce their load if new PEs are started.
