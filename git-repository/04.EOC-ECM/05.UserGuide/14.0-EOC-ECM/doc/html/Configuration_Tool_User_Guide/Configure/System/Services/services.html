Services

Clicking Services > Services displays the external services that may be invoked by runtime. This tab includes a service-port tree on the left. Clicking a port shows its details pane on the right.

You can perform the following tasks from the Services page:

  • Show the node label format in the left pane
  • Switch between the Services and Binding tabs using the Go to binding button
  • Add a new port to a node

Notes:

  • It is recommended to restart JBoss if you make changes to configuration variables.
  • With the migrate5config command line option, you can migrate the services from version 5.x applications.
