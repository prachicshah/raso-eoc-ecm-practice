Services

The Services section contains the following pages:

  • Services
    This page displays the external services that may be invoked by runtime and allows you to add a new port to the node.

  • Binding
    This page allows you to display the bindings that link service ports, service providers, and interfaces together. You can also edit existing bindings.

  • Listeners
    This page displays the metadata-defined listeners and allows you to change the number of threads allocated to each listener.

  • Queues
    This page allows you to view a list of queues, and add and delete queues.
