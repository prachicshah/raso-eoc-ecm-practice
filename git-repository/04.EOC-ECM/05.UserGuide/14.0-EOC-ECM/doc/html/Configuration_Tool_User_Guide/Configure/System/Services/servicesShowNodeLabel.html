Show Node Label

The left pane shows a list of nodes (external services). You can specify how the nodel label appears by clicking the Show Node Label drop-down field:

The Show Node Label drop-down field allows you to select one of the following options:

  • By Label
    Selecting this option shows the node's label as defined in the external service's Label field in Velocity Studio (for example, Security Service).
  • By Name
    Selecting this option shows the node's name as defined in the external service's Name field in Velocity Studio (for example, securityService).
  • By Full Name
    Selecting this option shows the node's full metadata name (for example, security.service.securityService).
  • By Label (Namespace)
    Shows the label, followed by the namespace to which in belongs in parentheses (for example, Security Service (securityService)).

Clicking the Node heading in the left pane sorts the nodes in either ascending or descending order.

You can also click the arrow on the Node heading to select other table options.
