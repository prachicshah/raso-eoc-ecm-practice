Switch Between Services and Bindings Tabs

When you select a port and its details appears, you can click the Go to binding button, which switches to the Binding tab.

The Binding tab shows the binding associated with the port.

The following fields are available on this page:

Field Description
Host This field denotes the catalog remote server name or IP address.
Port

This field represents the catalog server port. By default, this field is set to port 0.
