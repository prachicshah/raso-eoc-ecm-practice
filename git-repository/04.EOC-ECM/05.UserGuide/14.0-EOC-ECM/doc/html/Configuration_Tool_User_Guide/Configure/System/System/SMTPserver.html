SMTP Server

Clicking System > SMTP Server displays a list of SMTP server settings.

The following settings are available:

System Setting Description
Username

This property contains the username for the connection.
Password

This property denotes the password used for the connection, which is masked when you enter the password in this field.
SMTP Server Address This property specifies the SMTP server address of the e-mail server used to send e-mails.
SMTP Server Port

This property defines the SMTP server port used by alerts to send e-mails.
Secure Connection Select this property's checkbox if a secure connection is required.
Attempt SMTP SSL

By default, this property is unchecked. To attempt an SMTP SSL connection, select this property.
