System

The System section contains the following pages:

  • General
    This page allows you to view and configure various system settings.

  • SMTP Server
    This page provides SMTP server settings that you can manage.

  • Config Variables
    This page contains system-specific variables that you can define to adjust system settings.

  • Logging
    This page displays system logging levels, which allows you to specify the level of logging of various components.

  • Events
    This page allows you to disable and enable events.

  • Compatibility
    This page contains settings that support document and store procedure modes from product version 5.x.
