Tabbing

Clicking UI > Tabbing provides the ability to globally suppress tabbing to either any or all form elements, such as text fields, checkboxes, radio buttons, and more.

To exclude an element from the tab order, select the checkbox next to the element that you want, and then click the Save button to save your configuration.

Note: Suppressing tabbing for elements is not advised as it is not accessibility-friendly.

The following table describe further details about specific elements when you chose to suppress their tabbing functionality:

Element Notes
image

By default, image buttons are included in the tab order if the Can Focus property is set to true.

Note: Suppression is not applied to image buttons of date and reference fields.
layout

By default, a layout is included in the tab order if it has scrolling, which excludes all layouts in tabbing (that is, vertical and horizontal layouts, grid layouts, and vertical and horizontal stacks). Exclude each layout individually by setting its Include in Tab Order property.
menu Excludes all menu buttons from tabbing, but is not applicable to submenus. By default, menu items are included in the tab order if the Can Focus property is set to true.
radiobutton A known issue is that excluding the radio button works fine using Firefox as your Web browser. However, in Internet Explorer, Chrome, and Safari, it is impossible to completely exclude it from the tab order. It skips radio buttons from the tab order when moving from field to field, eventually tabbing to them after visiting all other items on the page.
table, tree By default, the table and tree bodies are focusable such that they can use keys for scrolling and other actions. As a result, they are included in the tab order.
tabset The default tab order also includes the tab buttons of the tab sets.
text Note: It is not recommended to disable tabbing for text items. Instead, exclude elements individually by setting their Include in Tab Order property.
textarea Note: It is not recommended to disable tabbing for text area items. Instead, exclude elements individually by setting their Include in Tab Order property.


Note: An element's Include in Tab Order property can override the global Suppress Tabbing setting. See the Elements section for information on this property.
