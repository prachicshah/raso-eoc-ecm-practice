iFrame

This page describes how to perform various tasks in iFrame and what user interface settings are needed to perform them.

Set Opacity for a Popup to Gray Out Browser When Page is in iFrame

To set the opacity for a popup to gray out the browser when the page is in iFrame, complete these steps:

  1. Set the Different Domain setting to false. The iFrame must be in the same domain.
  2. The main HTML body requires the following:
    • One div to represent the mask:
      <div id="cwShadow" class="opaqueLayer"> </div>
    • The other is the iFrame that actually displays the application:
      <iframe class="appLayer" src="http://localhost:8080/cwf" width="1500" height="700" scroll="auto"></iframe>
    Both HTML elements have a class property specified, which is used to set zIndex using the css stylename.
    .opaqueLayer 
            { 
                z-index:1;
                display:none; 
                position:absolute; 
                top:0px; 
                left:0px; 
                opacity:0.6; 
                filter:alpha(opacity=60); 
                background-color: #000000; 
            } 
            
     .appLayer 
                { 
                    position:absolute; 
                    z-Index:1001; 
                }
    Note: By default, the application has opacity set to 0. Set this property to the opacity that you want to use for the user action.

Custom Dialog Components

User messages can be replaced by custom dialog components by extending com.conceptwave.system.Dialog. You can control how to show your user messages by creating this object in your script and populating dialog.content.

Centering in iFrame

Centering based on the size of the root layout of your application does not always work. For example, if your application is 1000 px in height and 500 px in width, but your Web browser is only 500 px in height and 500 px in width, centering the popup would render it outside of the browser view, making it unusable. As such, whatever you are showing as a popup can be guaranteed to be visible at all times.

Dynamically Change the External Site Source

To dynamically change the source (src) of external sites in iFrame, you can use the HTML Content element by doing the following:

  1. Assign a script with a return type of String to the variable of the HTML content element.
  2. Have the script return an external site, such as <iframe src="http://www.ericsson.com" width="100%" height="100%" border="0">.
  3. Include statements in the script to determine which external URL to return.
