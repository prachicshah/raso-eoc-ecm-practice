System Configuration

The System Configuration application, also known as the Configuration application, allows you to configure across multiple application nodes when your application is deployed in a J2EE-cluster setting.

The System Configuration application contains the following menus:

  • System
    This section contains settings for configuration variables, event handlers, logging, and more.

  • Databases
    This menu allows you to view and configure both logical and physical database connections.

  • Processes
    This section allows for configuring, managing, and viewing processes, queues, alerts, and archives.

  • UI
    This menu contains user interface settings that you can configure.

  • Services
    This section allows you to view, change, and manage your listener and provider settings.

  • Catalog
    This menu contains setings to configure your Catalog application.

  • Performance
    This section allows you to fine-tune your application by adjusting your cache settings, setting statistical monitoring, and defining action categories.
