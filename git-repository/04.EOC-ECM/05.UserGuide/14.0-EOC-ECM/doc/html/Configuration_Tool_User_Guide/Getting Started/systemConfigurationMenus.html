System Configuration Menu Options

The System Configuration contains the following menu options:

  • Manage
  • Configure
  • Import
  • View

Note: To access all menus in the System Configuration application, your user ID must have the Runtime Administrator privilege set.

Manage Menu

The Manage menu contains the Manage Processes dialog, which allows you to change properties for all configuration node processes on one screen.

Configure Menu

The Configure menu has the following menu options:

Menu Item Description
System Configure application nodes from the System Configuration application.
Code Tables Search, add, edit, and delete code tables.

Import Menu

The Import menu provides controls for importing and exporting your configuration settings.

Menu Item Description
Export

Export your configuration settings.
Import Import your configuration settings.

View Menu

The View menu contains the Event Log, which Displays a list of messages logged by the User Profile Manager.
