System Configuration's User Interface

The System Configuration's user interface contains the following elements:

  • Preferences and logout bar
  • Menu bar
  • Node menu
  • Main area

Preferences and Logout Bar

This section contains the following components:

  • Indication of the user who has logged in to the System Configuration application
  • Number of assigned tasks and user availability for task assignment
  • Application preferences
  • Log out of the application
  • Exit the application and return to the Select Application page

Indication of Logged-in User

The top right corner of the user interface shows who has logged in to the application. In this example, upadmin is logged in.

Number of Assigned Tasks and User Availability for Task Assignment

The top right corner of the user interface shows the number of tasks (Tasks (number)) currently assigned to the logged-in user. It also shows whether the current user is available for task assignment.

In this example, the upadmin user has no tasks assigned. The green circle indicates that this user is available for task assignment.

For information on this feature, see the Number of Assigned Tasks and User Availability for Task Assignment section in the Worklist Management User Guide.

Application Preferences

Clicking your username and then clicking the Preferences hyperlink allows you to hide the System Configuration's banner.

To hide the banner, complete these steps:

  1. Click the Preferences hyperlink to launch the Preferences page.
  2. Select the Hide Banner setting and then click the Save All button.
  3. The banner disappears and a confirmation appears, indicating that your preference settings have been successfully saved.

To change the rows per page, complete these steps:

  1. Click the Preferences hyperlink to launch the Preferences page.
  2. Enter the number of rows in the Rows per Page field. This setting defaults to ten rows.
  3. Click the Save All button. A confirmation appears, indicating that your preference setting has been successfully saved.

Logout

Click the Logout hyperlink to end your current session.

To log back in, click the Click to Login or Login button, depending on whether you are using the classic or tiled version of the Select Application page.

Exit Application

Click the Exit Application hyperlink to close the application and return to the Select Application page. There are two types of Select Application pages:

  • Classic
  • Icon

See Select Application Page: Classic and Icon for more information.

Menu Bar

The menu bar contains the following options:

  • Configure
  • Import
  • View

Node Menu

The node menu appears on the left side of the page, which list the types of nodes in your application:

  • PE: Runtime instance runs as a Process Engine only
  • PE_UI: Runtime instance runs both UI server and Process Engine functions
  • UI: Runtime instance runs as a UI server only

You can expand and collapse these node types to select child nodes, if any. See the System Configuration user interface for more information about the node menu, and the details menu and tabbed pages in the main area.

Main Area

The main area appears under the menu bar, which contains information pertaining to the page that you are on. For example, the Event Log allows you to specify search criteria. Both the search critiera and your search results appear in the main area of the page.
