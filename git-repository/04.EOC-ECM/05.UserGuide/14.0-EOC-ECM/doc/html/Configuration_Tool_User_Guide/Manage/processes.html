Processes

Clicking Manage > Processes from the menu bar allows you to change properties for all configuration node processes on one screen. Each configuration node process appears, grouped by process name.

To edit the properties of a process, complete these steps:

  1. From the Manage Processes dialog, click the process that you want to make changes to.



  2. You can make changes to the following fields:

    Field Description
    Number of Processes This field represents the number of process instances for this PE instance, which is used for calculating the cache sizes and memory size.
    Serialization Click the drop-down menu and select from the available serialization options.

    For more information about these fields, refer to the PE Processes page.

    Notes:
    • When you edit a row, all text in the row appears in blue font to indicate unsaved changes.
    • If the node that you are editing has child nodes inheriting from it, its values change accordingly. If you want a different value for the child node processes, provide a different value, and then save the table.Â 
    • When you either set the number of processes to null or set the value with the same value as the parent, the value is inherited. As a result, whenever the parent node contains a change, that change appears at the child node level.

  3. When you have completed making your changes, click the Save button.
  4. A confirmation message appears, indicating that your changes have been saved.
