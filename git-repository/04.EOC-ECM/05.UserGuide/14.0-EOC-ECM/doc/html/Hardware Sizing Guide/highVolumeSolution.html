Note: There is no data available for the empty cells.

The production environment system hardware for a high volume solution includes three UI servers, two DB servers, three PE servers, and three API servers. Each UI server runs an instance of the application server that provides UI and API services. Each DB server provides common and aggregate DB services. Each PE server runs an instance of process engine service which also run Catalog server. Each API server runs an instance of application server that provides API services.
