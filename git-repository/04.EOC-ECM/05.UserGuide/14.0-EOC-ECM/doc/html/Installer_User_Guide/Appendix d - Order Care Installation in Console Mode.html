Appendix D: Work in Console Mode

When you cannot launch a graphical user interface, you can install the product using the console mode. In console mode, you are presented with the same options as in GUI mode.

Use the following command to start the installation process in console mode:

install.bin -i console

Notes:

  • In the Update Templates step of the product installation, there is neither a constraint, nor a dependency between templates.
  • Full installation mode is only supported in console mode.
