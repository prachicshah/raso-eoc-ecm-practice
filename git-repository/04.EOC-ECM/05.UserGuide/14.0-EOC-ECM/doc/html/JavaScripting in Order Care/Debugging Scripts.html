Debugging Scripts

After writing scripts, you should verify their correctness. Because of the nature in scripting, there is little validation check at design time. You should invoke the scripts at runtime to ensure they are correct. In addition to simply running the project, Velocity Studio provides script debugging capability at runtime with Rhino JavaScript debugger in two ways:

  • Using the Debug menu command, which provides script execution control for all scripts in the project.
  • For Global Scripts only, using a Debug Script command invoked from a right-click popup menu at a Global Script node in the Metadata tab of the Navigation Pane, which provides script execution control and input parameters' initialization for that particular Global Script.
