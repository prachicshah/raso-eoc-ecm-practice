Access the JavaScript Editor

You can launch the JavaScript Editor from Velocity Studio in one of the following ways:

  • Right-click the Script field, and then select the Editor Mode option from the popup menu.



  • Double-click any scriptable field to launch the JavaScript Editor. For example, double-click the Initialization field to launch the JavaScript Editor.
