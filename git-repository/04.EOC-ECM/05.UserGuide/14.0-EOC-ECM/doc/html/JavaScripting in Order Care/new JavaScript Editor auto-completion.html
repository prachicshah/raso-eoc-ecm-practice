JavaScript Editor - Auto-completion

Auto-completion means that the JavaScript Editor has the ability to open a popup list of all scripts (methods) while you code. In a couple of cases, the auto-completion function requires that you press the Ctrl + spacebar keys within the editor.

You can launch the auto-completion list panel as follows:

  • When there are no strings before the mouse cursor (that is, only a space, comma, or semicolon), press the Ctrl + spacebar keys to open the methods popup and select the method that you want.



  • Input a namespace in the editor, and then enter a period after the name to automatically launch the methods list, which shows all scripts that begin with your namespace's name.



  • Input the first few characters of the method name in the editor, and then press the Ctrl + spacebar keys. The list filters the method names such that the ones that match the characters you have entered display first.



    Note: To make auto-completion visible, continue to either input or backspace the first few letters of your method name. The list is then filtered by the word that you have entered.
