JavaScript Editor - Format Code

The JavaScript Editor allows you to automatically format your script code according to the editor's formatting rules. Using the Ctrl + I keys allows you to make formatting changes to your code, which include the following:

  • Indentation
  • Limit of one statement on a line
  • Proper continuation lines
  • Correct spacing between keywords and delimiters

To use this formatting feature, complete these steps:

  1. In the JavaScript Editor, use your mouse to highlight the code that you want to format.



  2. Press the Ctrl + I keys. The code automatically formats to the rules defined by the JavaScript Editor. In this case, the indentation for return true; changes, which uses an indentation of four spaces. The closing parenthesis and the return false; statement are left-justified.



    Note: When a Java keyword has no bracket (for example, if, for, while, and so on), using the Ctrl + I shortcut automatically indents the block statement.

  3. Click the OK button to save your formatting changes to your script.
