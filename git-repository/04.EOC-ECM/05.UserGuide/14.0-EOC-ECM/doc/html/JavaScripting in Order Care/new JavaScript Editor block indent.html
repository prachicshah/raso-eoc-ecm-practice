JavaScript Editor - Block Indentation

The JavaScript Editor contains a block indentation operation that allows you to indent a range of rows in your script. To use this operation, complete these steps:

  1. Highlight all lines of code that you want to indent.



  2. Press the Tab key to indent your highlighted block of code, moving it to the right.



  3. To indent your code to the left, highlight the text you want to indent and press the Shift + Tab keys.
