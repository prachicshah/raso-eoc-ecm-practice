JavaScript Editor - Comments in Code

The JavaScript Editor automatically generates a proper comment statement. To use the comment statement, do the following:

  1. Type /* on a line in your script where you want to add a comment.



  2. Continue writing your comment and then press the Enter key.



  3. The closing */ tag is automatically generated. If you have a longer set of comments to write, continue pressing the Enter key to generate comment line that begin with the asterisk (*).
