JavaScript Editor - Edit Menu

The JavaScript Editor allows you to perform the following editing functions:

  • Copy highlighted text within the JavaScript Editor
  • Cut highlighted text
  • Paste copied or cut text
  • Undo an action
  • Redo an action
  • Select all text in your script

You can access these editing functions by doing one of the following actions:

  • Use a keyboard shortcut combination specific to the editing function
  • Click any editing button from the toolbar that is specific to the editing function you want to use
  • Right-click the main editor area and select the Edit option, followed by the editing function you want to use

Copy Function

To copy text in your script, follow these steps:

  1. Highlight the text you that you want to copy in your script and then do one of the following:

    • Press the Ctrl + C keys.
    • Click the button from the toolbar
    • Right-click the text and select Edit > Copy from the menu.



  2. The text that you have highlighted has been copied to the clipboard.

Cut Function

To cut text from your script, complete these steps:

  1. Highlight the text you that you want to cut from your script and then perform one of the following actions:

    • Press the Ctrl + X keys.
    • Click the button from the toolbar
    • Right-click the text and select Edit > Cut from the menu.



  2. The text that you have highlighted has been cut and stored in the clipboard.

Paste Function

To paste text that you have previously copied or cut into your script, complete these steps:

  1. Place the cursor where you want to paste your text within your script and then perform one of the following actions:

    • Press the Ctrl + V keys.
    • Click the button from the toolbar
    • Right-click and select Edit > Paste from the menu.



  2. Your text is pasted into your script.

Undo Function

To undo a previous action, complete these steps:

  1. Do one of the following actions:

    • Press the Ctrl + Y keys.
    • Click the button from the toolbar
    • Right-click and select Edit > Undo from the menu.

Redo Function

To redo a previous action, complete these steps:

  1. Do one of the following actions:

    • Press the Ctrl + Z keys.
    • Click the button from the toolbar
    • Right-click and select Edit > Redo from the menu.



  2. In this example, performing the redo function continues deleting a character in the script. In this case, the redo function deletes the letter i.

Select All Function

To select all text in your script, complete these steps:

  1. Do one of the following actions:

    • Press the Ctrl + A keys.
    • Right-click and select Edit > Select All from the menu.



  2. All text in your script is selected.
