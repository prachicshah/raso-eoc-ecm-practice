Select JavaScript Methods to Add to a Script

While you are editing your script, you can open the Select Method dialog, which lists available JavaScript methods that you can insert into your code. To use the Select Method dialog, follow these steps:

  1. Click where in your script you want to insert your JavaScript method and then perform one of the following actions to launch the Select Method dialog:

    • Press the Ctrl + M keys
    • Click the button from the toolbar
    • Right-click your script text and select Functions > Select Methods from the menu

  2. A list of JavaScript methods appears. Click the drop-down box that contains the Global item, which contains a list of available object types and a global scripts item.



    In the Select Method dialog, the functions listed for the Global item and the object types display both the function name and the input parameters on the left side. The return type appears on the right side. Properties for object types also display the data type on the right side. A blue colour indicates a read-only property.

    Notes:

    • When the Global item is selected, a list of available global functions displays.
    • When the global scripts item is selected, a list of available functions defined in the Scripts folder in the metadata displays.
    • When any object type is selected (for example, Finder), the list contains two categories of data:
      • Object properties
      • Object functions

    Any field or method in the Select Method dialog has an icon and colour. The following table explains the codes:

    Icon or Font Style Meaning
    Left-arrow icon Getter function
    Right-arrow icon Setter function
    Double-sided arrow Getter/setter function
    Block icon Constructor
    Lightning-bolt icon Regular function
    Page icon Object property
    Script icon Script function
    Plain font Non-static function
    Bold font Static function
    Black-strike-through Deprecated function to be removed in a future release
    Red-Strike-through Deprecated function to be removed in the next major release

  3. Navigate through the lists by doing of the following:
    • Use your arrow keys
    • Press the first letter of your method to jump to a list of methods that begin with that letter
    • Use your mouse

    To select a method to use, highlight the row, and either double-click the selection or press Enter to insert the object into the script.

  4. The method that you have selected appears in your script.



    Notes:
    • After inserting a method into your code, the method's matching open and closing parentheses are highlighted in blue. The highlighting then disappears after a few seconds.
    • Ensure that the method parameters are defined in your script. Otherwise, they appear underlined in red. Use your mouse to hover over any underlined parameter, which provides information on correcting the problem.
