Insert Code Templates in a Script

The JavaScript Editor allows you to launch the JavaScript Code Template dialog, allowing you to insert code templates or standard code patterns into the JavaScript editor. To use the JavaScript Code Template dialog, follow these steps:

  1. Perform one of the following actions to launch the JavaScript Code Template dialog:

    • Click the button from the toolbar
    • Right-click your script text and select Functions > Code Template from the menu

  2. In the JavaScript Code Template dialog, double-click the code template that you want to insert into your script.



  3. The JavaScript Code Template dialog disappears and your code pattern appears inserted in your script.



  4. Change and customize the code pattern to suit your coding needs.
