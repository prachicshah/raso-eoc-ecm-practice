List Available Methods to Insert into a Script

While you are editing your script, you can open a list of methods and select the one that you want to insert into your code. To use this list, do the following:

  1. Click where in your script you want to insert your method and then perform one of the following actions to launch the methods list:

    • Press the Ctrl + Spacebar keys
    • Click the button from the toolbar
    • Right-click your script text and select Functions > List Methods from the menu

  2. A list of methods appears. Use the vertical scroll bar to locate the method that you want and then double-click it.



  3. The method that you have selected appears in your script.



    Notes:
    • After inserting a method into your code, the method's matching open and closing parentheses are highlighted in blue. The highlighting then disappears after a few seconds.
    • Ensure that the method parameters are defined in your script. Otherwise, they appear underlined in red. Use your mouse to hover over any underlined parameter, which provides information on correcting the problem.
    • When you enter this. in the editor and press the Ctrl + Spacebar keys, a list of methods, as well as local and inherited variables appears. You can then select the method or variable that you want from the list.
