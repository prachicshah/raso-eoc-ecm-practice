JavaScript Editor - Functions Menu

The JavaScript Editor allows you to perform the following functions when creating or editing your script:

  • List available methods that you can insert into your script
  • Select JavaScript methods that you can add to your script
  • Select a metadata element to add to your script
  • Insert code templates or standard code patterns in your script

You can access these functions by doing one of the following actions:

  • Use a keyboard shortcut combination specific to the function
  • Click any function button from the toolbar that is specific to the function that you want to use
  • Right-click the main editor area and select the Functions option, followed by the function that you want to use

You can also perform the following functions using the JavaScript Editor:

  • Use the auto-completion function to open a list of all methods while you code
  • Add a properly generated comment statement in your script
  • Select a node
