Select a Metadata Element to Add to a Script

The Select Element dialog allows you to select metadata elements that you can insert into your script. To use the Select Element dialog, perform these steps:

  1. Perform one of the following actions to launch the JavaScript Code Template dialog:

    • Press the Ctrl + E keys
    • Click the button from the toolbar
    • Right-click your script text and select Functions > Select Element from the menu

  2. In the JavaScript Code Template dialog, click the drop-down box and select a namespace. Once you have selected a namespace, a list of available elements belonging to that namespace appear.



  3. Double-click the metadata element that you want to insert into your script or press Enter to insert the full element name into your script.
