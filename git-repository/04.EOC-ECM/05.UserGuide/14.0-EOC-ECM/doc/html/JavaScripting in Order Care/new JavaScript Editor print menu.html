JavaScript Editor - File Menu

The JavaScript Editor allows you to print your script.

To print your script, follow these steps:

  1. Perform one of the following actions:
    • Use the Ctrl + P keyboard shortcut
    • Click the button from the toolbar
    • Right-click the main editor area and select File > Print from the menu

  2. A print dialog appears.



    Select the printer you want to send your print job to and then click the OK button.
