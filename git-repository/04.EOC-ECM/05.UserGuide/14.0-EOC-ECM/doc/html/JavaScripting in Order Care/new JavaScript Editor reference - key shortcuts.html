Reference - Keyboard Shortcuts

You can use the following functions in the JavaScript Editor to help you with writing your scripts. The following keyboard shortcuts are available.

Function Shortcut Description
Auto-completion Ctrl + Space List all scripts
Auto-formatting Ctrl + I Automatic script formatting, such as indentation, one statement on a line, proper spacing between keywords and delimiters, and more
Back indent Shift + Tab Tab indentation to the left
Copy Ctrl + C

Copy text in your script

Cut Ctrl + X

Cut text in your script
Find and replace Ctrl + F Find specific words or text within your script
Find next Ctrl + K Find the next occurrence of your keyword or text within your script
Find previous Ctrl + Shift + K Find the occurrence of your keyword or text by searching backward within your script
Forward indent Tab Tab indentation to the right
Go to line Ctrl + L Enter a line number to go to in within your script
Insert mode Insert Insert text while you edit your script in the editor
Open declaration F3 Open the declaration when the selected string is a metadata reference (for example, top-level metadata, variable, action, etc.)
Paste Ctrl + V

Paste cut or copied text in your script

Print Ctrl + P Print your script code from the editor
Redo Ctrl + Y

Redo the last action performed within the editor

Select all Ctrl + A Select all text in your script
Select element Ctrl + E Select an element from the list to add it into your script
Select method Ctrl + M Select a JavaScript function from the list to add it into your script
Show tree node Ctrl + T Display the tree node dialog, allowing you to select all metadata elements and their variables or scripts
Start runtime Alt + R Start runtime from your script
Undo Ctrl + Z Undo the last action performed within the editor
