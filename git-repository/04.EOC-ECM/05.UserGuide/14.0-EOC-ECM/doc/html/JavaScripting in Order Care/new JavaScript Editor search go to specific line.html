Go to a Specific Line in a Script

To go to a specific line number in your script, complete the following steps:

  1. Do one of the following actions to launch the Go to Line Number dialog:

    • Press the Ctrl + L keys.
    • Right-click and select Search > Go to Line... from the menu.

  2. In the Line number field, enter the line number that you want to go to, and then click the OK button.



  3. Your cursor appears on the line that you have specified going to. In this example, the cursor appears on line 14.
