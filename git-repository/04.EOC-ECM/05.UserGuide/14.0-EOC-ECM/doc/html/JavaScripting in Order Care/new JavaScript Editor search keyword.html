Search for a Keyword or Phrase in a Script

There are a number of ways in which you can perform a keyword search in the JavaScript Editor:

  • Use the Find or Replace Text dialog to perform a search
  • Use the toolbar's Search field to perform a search
  • Use previous search criteria to repeat a search
Use the Find or Replace Text Dialog to Perform a Search

To conduct a search in the JavaScript Editor, follow these steps:

  1. Perform one of the following actions to launch the Find and Replace text dialog:

    • Press the Ctrl + F keys
    • Click the button from the toolbar
    • Right-click your script text and select Search > Find from the menu

  2. The Find and Replace Text dialog appears.



    Fill in the following fields:
    1. Enter your search criteria in the Find field. You also have the option of clicking this field's drop-down menu and selecting previous search criteria.
    2. If you are conducting a search and replace, enter the text or keyword that will replace your search criteria in the Replace with field. See the Replace a keyword or phrase in your script for details.
    3. In the Options section, select from the following options:
      • Match case - Finds the exact text entered in the Find field in your script.
      • Match whole word - Finds the exact whole word entered as your search criteria within your script.
      • Warp search - Conducts a warp search based on the search criteria that you have entered.
    4. In the Direction section, select the Forward radio button if you want to search your script in a forward direction. Otherwise, select the Backward option to search in the reverse direction.

  3. Click the Find button to conduct your search. Any matching occurrence of your search criteria appears highlighted in purple in your script.

Note: When performing a search in the JavaScript editor, the search dialog remembers your previously specified search options.

Use the toolbar's Search field to perform a search

You can use the toolbar's Search field to quickly perform a search. Complete these steps:

  1. From the toolbar, enter your search criteria in the Search field.



  2. Proceed to click the Find Next Occurrence button to the right of the drop-down field to find the next occurrence of your search criteria in your script.

Use Previous Search Criteria to Repeat a Search

You can use search criteria that you previously used to conduct a search within your script by doing the following:

  1. From the toolbar, click the Search field's drop-down menu and then select search criteria that you have previously used.



  2. Proceed to click the Find Next Occurrence button to the right of the drop-down field to find the next occurrence of your search criteria in your script.
