JavaScript Editor - Search Menu

The JavaScript Editor allows you to perform the following search functions:

  • Search for a keyword or phrase in your script
  • Repeat the same keyword or phrase search in your script
  • Search backward for a keyword or phrase in your script
  • Replace a keyword or phrase in your script
  • Go to a specific line in your script

You can access these search functions by doing one of the following actions:

  • Use a keyboard shortcut combination specific to the search function
  • Click any search button from the toolbar that is specific to the search function that you want to use
  • Right-click the main editor area and select the Search option, followed by the search function that you want to use
