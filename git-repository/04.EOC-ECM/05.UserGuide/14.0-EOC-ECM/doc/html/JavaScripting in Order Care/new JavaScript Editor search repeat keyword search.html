Repeat the Same Keyword or Phrase Search

Clicking the Find button on the Find and Replace Text dialog only searches for the first occurrence of your search criteria. To continue searching through your script using the same criteria, do one of the following:

  • From the toolbar, click the Find Next Occurrence button as many times as required.



  • Press the Ctrl + K keys.
  • Right-click and select Search > Search Again from the menu.

Search Backward for a Keyword or Phrase

Clicking the Find button on the Find and Replace Text dialog only searches for the first occurrence of your search criteria. To continue searching through your script using the same criteria, but in a backward direction, do one of the following:

  • Press the Ctrl + Shift + K keys.
  • Right-click and select Search > Search Back from the menu.
