Replace a Keyword or Phrase in a Script

To search for a keyword or text, and replace the information with a different keyword or text, complete these steps:

  1. Perform one of the following actions to launch the Find and Replace text dialog:

    • Press the Ctrl + F keys
    • Click the button from the toolbar
    • Right-click your script text and select Search > Find from the menu

  2. The Find and Replace Text dialog appears.



    Fill in the following fields:
    1. Enter your search criteria in the Find field. You also have the option of clicking this field's drop-down menu and selecting previous search criteria.
    2. Enter the text or keyword that you want to replace in the Replace with field.
    3. In the Options section, select from the following options:
      • Match case - Finds the exact text entered in the Find field within your script.
      • Match whole word - Finds the exact whole word entered as your search criteria within your script.
      • Warp search - Conducts a warp search based on the search criteria that you have entered.
    4. In the Direction section, select the Forward radio button if you want to search your script in a forward direction. Otherwise, select the Backward option to search in the reverse direction.

  3. Click the Replace button to conduct your search and replace actions for each instance. Any matching occurrence of your search criteria appears highlighted in purple in your script.



    Note: Clicking the Replace All button replaces all instances of your search criteria in your script, without any confirmation dialog appearing.

  4. A dialog appears, asking you to confirm the replacement (in this case, order) from your script.



    You have the option of clicking the following four buttons:

    Button Action
    Yes Click this button to confirm text replacement. This dialog appears for every keyword or text instance that is found in your script.
    No Click this button to decline text replacement. This dialog appears for every keyword or text instance that is found in your script.
    All Click this button to apply the replacement text to your entire script. The dialog button automatically disappears.
    Done Click this button to exit the search and replacement process.

    To replace all occurrences of your search criteria (in this example, order) from your script, click the All button.

  5. Your script shows that all occurrences of order have been changed to item.



  6. Click the OK button to save your changes.

Note: When performing a search in the JavaScript editor, the search dialog remembers your previously specified search options.
