JavaScript Editor - Select Node

The JavaScript Editor allows you to select all metadata elements, and their variables and scripts, into your code. To use the Select Node dialog, complete these steps:

  1. In the JavaScript Editor, perform one of the following actions to launch the Select Node dialog:

    • Press the Ctrl + T keys
    • Click the button from the toolbar



  2. Select the metadata object name by doing one of the following:
    • Use the vertical scroll bar to find the object name that you want.
    • Enter the object name or a portion of it in the Enter object name prefix or pattern field.



  3. Highlight the metadata object name that you want under the Matching items heading, and then click the OK button to select all metadata elements, and their variables and scripts.

History of Node Selections

The Select Node dialog keeps a history of all nodes that you have selected in the past. To view the history, do the following:

  1. In the JavaScript Editor, press the Ctrl + T keys to launch the Select Node dialog.
  2. A list of previously selected metadata objects appears at the top of the Matching items section of the dialog.
