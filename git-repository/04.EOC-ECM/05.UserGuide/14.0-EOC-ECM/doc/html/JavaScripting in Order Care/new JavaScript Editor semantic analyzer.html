JavaScript Editor - Semantic Analyzer

The JavaScript Editor contains a built-in semantic analyzer that checks for correct variable and function definitions in your code. When you load a script or when a value in your script has changed, the semantic analyzer automatically analyzes the code and highlights errors in it. The JavaScript Editor underlines each word in red that has been flagged by the semantic analyzer to denote an error.

In editor mode, use your mouse pointer to hover over an underlined word that appears in red. An error message appears as a tooltip, to help you troubleshoot and debug your script.

JavaScript Parser

The JavaScript parser works with the semantic analyzer to analyze your script. When you load a script or when a value in your script has changed, the JavaScript Editor automatically analyzes the code by JavaScript grammar and highlights errors in your code. The JavaScript Editor underlines each word in red that has been flagged as an error. In editor mode, use your mouse pointer to hover over an underlined word that appears in red. An error message appears as a tooltip, to help you troubleshoot and debug your script.
