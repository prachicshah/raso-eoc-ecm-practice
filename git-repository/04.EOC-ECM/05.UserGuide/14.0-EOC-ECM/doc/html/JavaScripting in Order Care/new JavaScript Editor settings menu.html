JavaScript Editor - Settings Menu

The JavaScript Editor allows you to change a number of settings within the editor, including the following:

  • Change the font type and size used to code within the JavaScript Editor
  • Show or hide line numbers in the left margin of the editor
  • Map objects as fixed types
  • Change the tab size when indenting your code

You can also find a key map that describes the commands available in the JavaScript Editor and their respective keyboard shortcuts.

To change your editor settings and launch the Preferences dialog, right-click and select Settings > Preferences from the menu.

Change Font Style and Size

You can change the font family and size used to edit your script in the JavaScript Editor by completing these steps:

  1. In the left pane of the Preference dialog, click the Fonts option.



  2. To change the font style, click the Font family drop-down menu and select the font style that you want.



  3. To change the Font size, you can either use the up or down arrows, or enter the size in the field provided.

    Note: Changes to either the Font family or Font size fields can be previewed in the Sample area.

  4. Click the OK button to save your changes.

Show or Hide Line Numbers in the Editor's Left Margin

To change whether you see line number in the JavaScript Editor's left margin, complete these steps:

  1. In the left pane of the Preference dialog, click the Editor option.

  2. To display the line numbers in the JavaScript Editor, select the Show line numbers option and then click the OK button.



  3. In the JavaScript Editor, line numbers appear in the left margin.

Note: You can also enable the line numbers to appear in the left margin by right-clicking the main editor area, and selecting the View > Row number checkbox.

Map Objects as Fixed Types

You can define your variables as fixed types from the Object Mapping pane. In the left pane of the Preference dialog, click the Editor > Object Mapping option to access the Object Mapping pane.

The following are predefined objects in the JavaScript editor:

  • theAVM instance has AVM as its fixed type
  • theCatalog instance has Catalog as its fixed type
  • theItem instance has Catalog item as its fixed type
  • theBasket instance has Basket as its fixed type
  • theContext instance has Context as its fixed type

A column of action icons appears to the right of the table. The following table describes the functionality of each action icon.

Icon Description
Move Up Move the selected variable up one row in the table.
Move Down Move the selected variable down one row in the table.
Add Element

Add a variable to the table by doing the following:

  1. Enter the variable name in the Instance Name field.
  2. Click the Object drop-down menu and select the object type from the list.
  3. Click the OK button to save your changes.
Remove Element Delete the selected variable from the table.
Properties Show the properties of the selected variable in a Dialog box. You can edit the Instance Name field, if required. Click the OK button to save your changes.
Sort Sort the variables in the table in alphabetical order.

Change the Tab Size When Indenting Code

You can change the tab indentation size, complete these steps:

  1. In the left pane of the Preference dialog, click Editor > Formatting.

  2. The Tab indent size field defaults to 4 character spaces. To change this setting, either use the up or down arrows, or enter the size in the field provided.



  3. Click the OK button to save your changes.

Key Map of All Shortcuts in the JavaScript Editor

The key map allows you to view keyboard shortcuts that you can use while working with your script in the JavaScript Editor. To view the key map, in the left pane of the Preference dialog, click Editor > Key Assist.
