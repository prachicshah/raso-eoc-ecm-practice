JavaScript Editor - Token Usage

You can view how many times a keyword or token within your code has been used. This functionality is helpful when debugging your script.

To use the token usage functionality, follow these steps:

  1. In your script, double-click the keyword or token that you want. All instances of the keyword or token that you have selected are highlighted in grey throughout your script.
