JavaScript Editor - User Interface

The JavaScript Editor consists of three main areas:

  • Toolbar
  • Main area for script development
  • Status bar

Toolbar

The JavaScript Editor toolbar allows you to perform a number of actions to help you with writing your script. The following table describes what each button does within the script editor.

Button Shortcut Function
Ctrl + P Print the code displayed in the JavaScript Editor.
Ctrl + Z Undo the last action that you performed.
Ctrl + Y Redo an action that you performed.
Ctrl + C Copy highlighted text into your script.
Ctrl + V Paste cut or copied text into your script.
Ctrl + X Cut highlighted text into your script.
Ctrl + F Find specific words or text within your script. You also have the option of finding words or text, and replacing them.
Ctrl + K The drop-down box allows you to select from previous search criteria that you have used. Click the Find Next Occurrence button to the right of the drop-down field to find the next occurrence of your search criteria in your script.
Ctrl + Spacebar List methods that you can use in your script by clicking this button. Select the method you want to insert into your script by double-clicking your selection.
Ctrl + M Open the Select Method dialog, which lists the available JavaScript functions that can be selected and added into the script editor.
Ctrl + E Opens the Select Element dialog, which lists the available metadata elements that can be selected and added into the script editor.
Ctrl + T Opens the Select Node dialog, allowing you to select all metadata elements, and their variables and scripts, into your code.
 

Click this button to launch the JavaScript Code Template dialog, allowing you to insert code templates or standard code patterns into the JavaScript editor.

Main Area for Script Development

The JavaScript Editor's main area allows you to code and edit your scripts. You use this area with its toolbar, which provides you with a number of useful actions to help you develop and edit your scripts.

Status Bar

The JavaScript Editor's status bar shows a number of useful details:

  • Whether the script has been changed.
  • Whether you are in Insert mode (insert text as you use the JavaScript Editor) or in Overwrite (enter text over existing text) mode.
  • What line and character position your cursor is on. In the example that follows, your cursor is on line three, at character 50.


To the left of the status bar, you have two buttons:

  • The OK button saves your script changes and closes the JavaScript Editor window
  • The Cancel button does not save our changes and closes the JavaScript Editor window
