JavaScript Editor - View Menu

The JavaScript Editor allows you to view or hide the following user interface features:

  • Row numbers that appear in the left margin of the editor
  • The status bar that is located in the bottom right corner of the editor
  • Toolbar elements

To view or hide these user interface elements, right-click the main editor area, select the View option, and then select the user interface element that you want to view or hide.

Row Numbers

You can view or hide the row numbers in the left pane of the JavaScript Editor.

To view the row numbers, right-click the main editor area, and then select the View > Row number checkbox.

To hide the row numbers, ensure that the Row number checkbox is left unchecked.

Note: You can also show or hide row numbers in the left pane by right-clicking the main editor area, and then clicking Settings > Preferences. The Show line numbers checkbox under the Editor option allows you to perform the same action as clicking View > Row number from the right-click menu.

Status Bar

You can view or hide the row numbers in the bottom right corner of the JavaScript Editor.

To view the status bar, right-click the main editor area, and then select the View > Panes > Status Bar checkbox.

To hide the status bar, ensure that the Status Bar checkbox is left unchecked.

Toolbars

The toolbar consists of the following sections:

  • File
  • Edit
  • Search
  • Functions

You have the option of showing some, none, or all the toolbar selections. To customize your toolbar in the JavaScript Editor, follow these steps:

  1. Right-click the main editor area, and then select the View > Toolbars.
  2. By default, all four toolbar options are enabled:
    • File
    • Edit
    • Search
    • Functions



    To hide any of these toolbar section, deselect the toolbar options that you do not want. In the following example, only the Edit and Search options are selected.



  3. If you want to hide the toolbar, click View > Toolbars > Hide All.



    You can revert your changes by clicking View > Toolbars > Show All to view the entire toolbar.
