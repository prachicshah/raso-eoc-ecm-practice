JavaScript Editor

The JavaScript Editor allows you to create, write, and edit scripts within Velocity Studio. The JavaScript Editor contains the following areas:

  • The toolbar that allows you to perform a number of actions while writing your script, which includes selecting metadata elements for insertion into the development pane.
  • The main development pane, which allows you to create and edit your script.
  • The status bar at the bottom of the editor, which indicates whether you are changing your script, what character position your mouse pointer is located on a specific line of code, and whether you are inserting characters into your script.

You can perform the following tasks in the JavaScript Editor:

Access the JavaScript Editor
This section describes how to launch the JavaScript Editor from Velocity Studio.

JavaScript Editor's User Interface
The JavaScript Editor consists of three main areas:

  • Toolbar
  • Main area for script development
  • Status bar

File Menu in the Editor
The File menu contains printing options, allowing you to print your script from the JavaScript Editor.

Edit Menu
The Edit menu provides you with a number of editing functions within the JavaScript Editor, such as copying, cutting, and selecting all code within your script. This section also describes other editing functions you can perform in the editor, such as block indentation and code formatting.

View Menu
The View menu allows you to customize the JavaScript Editor's toolbar. You also have the option of viewing or hiding both line numbers in the editor and the status bar.

Search Menu
The Search menu contains the following search functions:

  • Search for a keyword or phrase in your script
  • Repeat the same keyword or phrase search in your script
  • Search backward for a keyword or phrase in your script
  • Replace a keyword or phrase in your script
  • Go to a specific line in your script

Functions Menu
The JavaScript Editor contains these functions to help you create and edit your script:

  • List available methods that you can insert into your script
  • Select JavaScript methods that you can add to your script
  • Select a metadata element to add to your script
  • Insert code templates or standard code patterns in your script

In addition to the functions within this menu, you can also perform these actions:

  • Use the auto-completion function to open a list of all methods while you code
  • Add a properly generated comment statement in your script
  • Select a node

Settings Menu
You can change a number of settings within the editor, including the following:

  • Change the font type and size used to code within the JavaScript Editor
  • Show or hide line numbers in the left margin of the editor
  • Change the tab size when indenting your code

You can also find a key map that describes the commands available in the JavaScript Editor and their respective keyboard shortcuts.

JavaScript Editor's Semantic Analyzer
The JavaScript Editor contains a built-in semantic analyzer, which checks for correct variable and function definitions in your script.

Token Usage
The token or keyword usage functionality allows you to view how many times a keyword or token has been used within your code. This feature helps you with debugging and troubleshooting your script.
