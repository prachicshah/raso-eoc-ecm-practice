Distributed Interface Deployment

You can deploy and dedicate your interfaces on different nodes through the Configuration application.

To do distributed interface deployment, you need at least two nodes defined in the Configuration application:

  • One node for the user interface
  • One node for the interface node

Under the Services tab in the configuration UI, do the following:

  1. For the UI node, all interface ports need to be disabled, and all listener threads are either left blank or set to 0.
  2. For the Interface node, interface ports need to be active and the listener thread numbers must be defined.

Note: For product version 5.x and higher, all nodes must be deployed in a J2EE application server.
