AVM Caches Domain

The AVMCaches domain allows you to view, manage, and change a number of attributes, including Size and Element count. It also provides you with operations that you can use to manage your AVM caches, such as the Clean the cache and Count removable elements operations.

Attributes

Expanding the AVMCaches' Attributes node in the MBeans tree provides you with all attributes that you can view, manage, and change.

As an example, clicking the Element count attribute displays pertinent information about it, including the attribute's description, whether it is readable or writeable, and its data type.

When you hover your mouse pointer over the Refresh button, a tooltip indicates the action that occurs after you click the button. You can also hover your mouse pointer over the Name and Value fields. In the following example, hovering the mouse pointer over the Value field displays a tooltip indicating that you can expand the field by double-clicking it.

Double-clicking the Value field in this example allows you to chart the Element count. You also have the option of clicking the Discard chart button to view the attribute's Value field.

Operations

Expanding the AVMCaches' Operations node in the MBeans tree provides you with operations that you can use to manage your AVM caches.

As an example, clicking the Count removable elements operation displays pertinent information about the operation, including its description, return type, impact, and any parameters that the operation has.

To invoke an operation, click the button in the right pane that is associated with the command you want to use. In this example, click the Count removable elements button to invoke the command.

Print Cache Info Operation

The Print Cache Info operation is available in AVM Caches and accepts the following parameters:

  • ID Only
    This property accepts Boolean values and the output only includes the cache elements's ID
  • ID
    This property accepts String values, with the output object having a specific ID

The following caches can have their memory queues exported using this operation:

  • Activities Cache
  • CodeTable Cache
  • DynamicParticipants Cache
  • Image Cache
  • InterfaceData Cache
  • Process Cache
  • ProcessData Cache
  • RecentlyChecked Cache
  • RecentlyFinished Cache
  • Resource Cache

Addtionally, the following Catalog caches can have their memory queues exported:

  • CatalogTranslations Cache
  • ChargeType Cache
  • DynamicRow Cache
  • InfoTable Cache
  • Item Cache
  • ItemParents Cache
  • QueryResult Cache
  • Tax Cache
  • TaxModel Cache
