AVM Domain

The CWAVM domain allows you to view, manage, and change a number of attributes, including Working thread count and Canceled processes. It also provides you with operations that you can use to manage the AVM, such as the Activity threads and Stop PE gracefully operations.

Attributes

Expanding the CWAVM's Attributes node in the MBeans tree provides you with attributes that you can view, manage, and change. As an example, clicking the Canceled processes attribute displays pertinent information about it, including the attribute's description, whether it is readable or writeable, and its data type.

When you hover your mouse pointer over the Refresh button, a tooltip indicates the action that occurs after you click the button. In this example, clicking the Refresh button refreshes the attribute Value field.

Operations

Expanding the CWAVM's Operations node in the MBeans tree provides you with operations that you can use to manage your AVM. As an example, clicking the Stop PE gracefully operation displays pertinent information about the operation, including its description, return type, impact, and any parameters that the operation has.

To invoke an operation, click the button in the right pane that is associated with the command you want to use. In this example, click the Stop PE gracefully button to invoke the command.

Notifications

Clicking the Notifications node in the MBeans tree allows you to subscribe or unsubscribe to AVM provides you with operations that you can use to manage your AVM. You also have the ability to clear notifications that you have viewed.

To subscribe to notifications, click the Subscribe button at the bottom of the right pane.

After clicking the Notifications button, a counter appears next to the Notifications node's label in the left pane, denoting how many notifications exist. Notification details appear in the right pane.

To unsubscribe to notifications, click the Unsubscribe button. You can also clear notifications by highlighting the notifications you want and then clicking the Clear button.
