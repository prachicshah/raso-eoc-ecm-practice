Process Engine Working Threads Domain

The PEWorkingThreads domain allows you to view, manage, and change a number of attributes, including Total activities and Longest activities.

As an example, clicking the Longest activity (ms) attribute within the ProcessActivities_01 Thread displays pertinent information about it, including the attribute's description, whether it is readable or writeable, and its data type.

When you hover your mouse pointer over the Refresh button, a tooltip indicates the action that occurs after you click the button. In this example, clicking the Refresh button refreshes the attribute Value field.
