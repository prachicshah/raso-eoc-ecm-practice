Monitor and Manage MBeans in JConsole

The MBeans tab allows you to view, monitor, and manage information on all registered MBeans on your server. Specifically, this tab allows you to view, change, and monitor the com.conceptwave.AVM class, which reveals caching, queuing, and performance monitoring properties.

Expanding the tree on the left shows all the MBeans, organized according to their object names. When you select an MBean in the tree, its attributes, operations, notifications, and other information is displayed in the right pane. You can access the following domains using the left tree navigation:

  • AVM caches
  • AVM
  • Process engine queues
  • Process engine working threads
  • Verbose

You can set the value of attributes if they are writeable (the value appears in blue).Â You can also invoke operations displayed in the Operations node within the tree.

Disconnect from a Process

To disconnect from a process, click the green icon in the right corner of the JConsole interface.

After clicking the icon, you have the option to immediately reconnect by clicking the Reconnect button.

Button and Field Tooltips

When you hover your mouse pointer over a button or a field on the MBeans tab, a tooltip appears, which provides additional information about the button or field in question. Descriptions are also available in both the MBean attribute and operation informational tables.
