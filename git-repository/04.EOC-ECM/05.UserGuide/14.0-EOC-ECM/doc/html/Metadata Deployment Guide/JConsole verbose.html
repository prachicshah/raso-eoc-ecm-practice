Verbose Domain

The Verbose domain allows you to view, manage, and change a number of attributes, including Profiler mode and Trace mask.

As an example, clicking the Trace mask attribute displays pertinent information about it, including the attribute's description, whether it is readable or writeable, and its data type.

When you hover your mouse pointer over the Refresh button, a tooltip indicates the action that occurs after you click the button. In this example, clicking the Refresh button refreshes the attribute. The Value field is blue, which allows you to edit this field and then refresh the attribute.

Example: Produce values of SQL parameters with a trace mask of 8

To view values of SQL parameters with a track mask of 8, do the following:

  1. In JConsole, click the Trace mask attribute and set its Value field to 8.
  2. For the Profiler mode attribute, set its value to OFF.
  3. Specify the Verbose file.
  4. In the verbose file, the values of SQL parameters are generated for data document database operations and SQLs with ? as parameters.
