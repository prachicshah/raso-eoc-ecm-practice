Include Oracle WebLogic-related JAR Files in Velocity Studio Path

To include WebLogic-related JAR files in your Velocity Studio path, refer to Oracle's procedure on creating a wlfullclient.jar for JDK 1.6 client applications.

For the last step of the procedure, ensure that you add wlfullclient.jar to the classpath within the <installation_folder>\designer\env\startDesigner.cmd file.
