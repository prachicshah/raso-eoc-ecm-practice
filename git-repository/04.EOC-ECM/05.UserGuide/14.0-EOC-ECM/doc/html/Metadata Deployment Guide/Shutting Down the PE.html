Shut Down the PE

Process engines can be gracefully shutdown by simply stopping their service at their hosted J2EE application servers. As an alternative approach, you can gracefully shut down a process engine through a JMX Remote Management application such as JConsole.

To use JConsole to shut down the process engine, do the following:

  1. Connect to the Process Engine using JConsole, and then click the MBeans tab.
  2. Navigate to com.conceptwave.AVM > CWAVM > Operations > Stop PE gracefully in the MBeans tree.

    .

  3. In the right pane, click the Stop PE gracefully button to issue the command and stop the process engine.

With JMX, the Process Engine can also be stopped using, for example, a Command-line JMX Client: http://crawler.archive.org/cmdline-jmxclient/index.html. The CW bean name is com.conceptwave.AVM:name=general, type=PE, and the Operation to stop the process engine is: stopPEGracefully. With Command-line JMX Client, a sample command would be as follows:

java -jar cmdline-jmxclient.jar -:- myhostname:10001 com.conceptwave.AVM:type=PE stopPEGracefully
