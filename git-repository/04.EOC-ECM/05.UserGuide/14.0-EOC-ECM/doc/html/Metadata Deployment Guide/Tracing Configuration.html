Tracing Configuration

AVM tracing configuration can only performed through the command line. AVM tracing configuration consists of the following components:

  • Tracing configuration command-line interface, which allows you to define local trace jobs for a specific application instance running on an AVM.
  • Simple Java JMX client, which details how tracing configuration scripts can use the com.conceptwave.runtime.cli.CwfInvokeJmx standalone class as a simple JMX client utility.
  • Tracing configuration JMX API, which lists a number of supported methods, along with their parameters and return values.
  • Tracing configuration Java model, which list the model classes, their fields, and their descriptions.
