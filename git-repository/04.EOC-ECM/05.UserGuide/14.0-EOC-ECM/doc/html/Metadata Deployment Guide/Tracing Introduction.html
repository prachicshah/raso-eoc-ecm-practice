Introduction to Tracing

Tracing is a technique that allows you to understand events that occur in a system to either debug or monitor it. Tracing is primarily intended for first-line support and troubleshooting activities.

This section contains the following information:

Tracing Configuration
AVM tracing configuration can only performed through the command line and consists of the following components:

  • Tracing configuration command-line interface
  • Simple Java JMX client
  • Tracing configuration JMX API
  • Tracing configuration Java model
