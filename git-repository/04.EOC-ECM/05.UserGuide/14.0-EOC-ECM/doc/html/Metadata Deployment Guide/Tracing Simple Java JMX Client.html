Simple Java JMX client

The tracing configuration scripts can use the com.conceptwave.runtime.cli.CwfInvokeJmx standalone class, which is meant to be a simple JMX client utility. It has minimal classpath dependencies and needs to be installed in JAR format with the command-line interfaces script on the application instances along with the AVM.

When this JAR file is installed, the command-line interface scripts need to be installed with the AVM JMX port number so that it can be provided to this class as a parameter.

The ordered arguments for its main method are as follows:

  1. JMX port
  2. MBean name on which to perform the operation
  3. Operation name
  4. Parameters (dependent on the operation)

This class does the following:

  1. Creates a JMX connection
  2. Fetches the MBean
  3. Invokes the specified operation
  4. Returns the result or propagates any exceptions
