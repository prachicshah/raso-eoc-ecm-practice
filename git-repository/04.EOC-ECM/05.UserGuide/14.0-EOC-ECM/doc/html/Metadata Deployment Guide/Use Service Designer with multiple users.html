Use Velocity Studio with multiple users

You can test your application with multiple users (for example, users > 1000) by adding the following option to your Velocity Studio startup script:

-Dcom.conceptwave.dontCheckForXSRF=true

If you do not use this option, the following error occurs:

ERROR /cwf - Exception while dispatching  incoming RPC call java.lang.SecurityException:  Blocked request without GWT permutation header (XSRF attack?)
