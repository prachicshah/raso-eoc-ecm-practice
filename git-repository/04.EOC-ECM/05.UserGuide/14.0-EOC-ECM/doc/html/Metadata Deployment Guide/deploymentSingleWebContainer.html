Deployment to a Single Web Container

This page provides you general information about deploying the product to a single Web container. The following describes the steps:

  1. Install Ericsson Order Care and Ericsson Catalog Manager.
  2. Locate the Disk1\InstData\Resource1.zip\$IA_PROJECT_DIR$\cd_source\build\ws\cwf.War file, and install it to the Web container.
    3. You can customize the web.xml with the use of Build Deployment WAR option of Velocity Studio, or command line option to generate the WAR file.Â 

    Note: The generated WAR file does not contain the metadata. If you are upgrading an existing deployment, replace the existing CWF Web application.

  3. Add the following system properties to the environment for the CWF Web application, or in the Web container’s startup scripts:

    -Dcom.conceptwave.appName=<Database application name>
    -Dcom.conceptwave.appVersion=<Database application version>
    -Dcom.conceptwave.nodeId=<Configuration Node Name>

    • For new deployments, create the database and run the required DDL scripts, and then use Velocity Studio or command-line options to generate the metadata upgrade script. Run the upgrade scripts.
    • For existing deployments, upgrade the database.
  4. Start the WEB container.
