Mapping Tab: Document to Text Mapping

The Text Map sub-tab in the Mapping tab allows the user to specify how to import/export Documents as text. Table below contains descriptions of the fields in the TextMap tab.

Field Description
Record type Specifies Fixed or Variable length records. If Fixed is selected, then every data field will have a data length defined. If Variable is selected, then the record fields will be separated by the separator string specified in the Separator field and the records will be separated by character specified in Terminator field.
Separator String used to separate the record fields for Variable record type format (for example, comma ,).
Quotation Character denoting quoted text (for example, single quote ' or double quote ). For Variable record format only
Terminator String appended to the end of each record during export that denotes the end of a record. (for example, new line \n). For Variable record format only.
List of fields The list of Document fields that will be exported/imported to/from the text file.  May not include all Document Variables.

List of Fields

The list of fields may be a subset of the Document variables In addition, a Field labeled none can be added to the list to allow fixed text output.

Click the Add button to add a field from the Document Variables. The Text map search dialog appears; highlight the item to be added and click Save" to add the field.

Click the Properties button, the Field Properties dialog will open where the user can enter the field length, format and null value (only the Length field will be visible for none field items). If the length and/or format is defined, it will override the values defined in the Data Types properties. The Length is mandatory for Fixed record types and for none field items.

The rows in the list of fields can be reorganized using the up and down arrow buttons located beside the table. Highlight an item, then click either the up or down button to move it. The order in the list will determine the order that the fields are imported/exported to/from the text file.
