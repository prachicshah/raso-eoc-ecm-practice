Mapping Tab: Document to XML Mapping

The XML Map sub-tab in the Mapping tab specifies how to export/import Documents to/from XML format. Table below contains a description of the fields in this tab.

Field Description
XML Name XML tag name for this element. Default value is the element name.
Length Visible for Variable elements only. Data length.
Format Visible for Variable elements only. Data format.
Exclude When checked for Variable elements, this Variable will be excluded when generating the XML.
When checked for branch elements, when generating the XML this element's XML tag will not be generated, but the XML tags for the children elements will be.
Exclude children Visible for branch elements only. When checked, this element, and all of its children, will be excluded when generating the XML.
Include namespace Visible for a Document root only. When checked, the xmlns attribute will be generated with the namespace value of the namespace this Document belongs to.
Attribute Visible for Variable elements only.  When checked, the data will be generated as an attribute of the Document tag when generating the XML.
CDATA When checked, CDATA will be generated when generating the XML.
Update Must be clicked for the data entered on the screen to be stored in the metadata as a XML mapping for the selected Variable.
Remove Cleans the screen and removes the XML mapping for the selected Variable.
