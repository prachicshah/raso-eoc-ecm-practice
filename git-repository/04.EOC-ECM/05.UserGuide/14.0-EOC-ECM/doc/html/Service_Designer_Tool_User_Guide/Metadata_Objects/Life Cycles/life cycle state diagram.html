Life Cycle State Diagram

The Diagram tab (see figure below) provides a graphical editor for drawing a Life Cycle state flow diagram. Once the Enumeration field in the General tab is filled in with a state element, the analogous icons will be presented automatically in the editor.

The Diagram tab is split into two panes:

  • The top pane contains a toolbar and a graphic diagram of the relationships between states. The table describes the functions of each button in the toolbar.
  • The bottom pane contains a toolbar and a JavaScript editing field that allows quick script editing for the selected state/link. The toolbar has a set of buttons on its left side that allows fast switching between the state/link scripts. The Save icon is used to save all changes. An editable combo box and a set of buttons on the right side can be used to search for particular words in the current state/link or in all Life Cycle states/links. The tooltips can be used to find out the button meanings.

In the top diagram pane, the green start state (the state with the NEW state code) and red end state (the state with the END state code) icons are mandatory states, representing a start and end point in the flow. The other icons are intermediate states. You can drag-and-drop these icons to any place in the editor to create a state flow.

To edit the state or link properties right-click the state icon and select the Properties command (refer to sections on Life Cycle State Properties and on Life Cycle Link Properties).
