Navigate Menu

The Navigate menu contains menu options as described in the following table:

Menu Shortcut Function
Open Type Ctrl + Shift + T This menu command provides a keyboard shortcut to navigating top-level metadata objects, an alternative to using the Navigation Pane. The Select Metadata dialog box appears.



The dialog box contains all metadata objects of the project, including both the application and library metadata. See instructions on how to use this dialog to search for the desired metadata object.
Forward Alt + Right Arrow Navigates to the next object in metadata object selection history.
Back Alt + Left Arrow Navigates to the previous object in metadata object selection history.
