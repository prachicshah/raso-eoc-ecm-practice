Define the Presentation

At the Presentation layer, a UserInterface object is defined containing a variable (testDocUI) that points to the document object (validate.TestDoc.UserInterface) created in the steps above and flagged for the Validate checkbox. The second variable, testDocUIArray, also points to the same document object, but this variable is an Array with the Array checkbox flagged.

Initialization

At runtime, the testDocUI user interface variable is initialized to the user interface (validate.TestDoc.UserInterface) by the onInit method. The onInit method of the TestUI object assigns the TestDocUI variable to the document user interface object (validate.TestDoc.UserInterface).
