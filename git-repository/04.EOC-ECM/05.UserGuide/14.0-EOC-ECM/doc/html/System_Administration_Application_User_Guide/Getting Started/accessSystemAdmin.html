Access the System Administration Application

By default, the System Administration application is available to the User Profile Administrator user with the Administration App privilege. With this privilege, you can log in to the application by completing these steps:

  1. In your Web browser, access the System Administration application by entering http://localhost:8080/cwf/systemAdministration as the URL.
  2. Enter upadmin as both your Username and Password, and then press the Enter key.

Note: The login screen differs, depending on whether you have selected the classic Select Application page or have opted to use the icon version of this page instead.

If you have administration privileges only, the classic Select Application page displays.
