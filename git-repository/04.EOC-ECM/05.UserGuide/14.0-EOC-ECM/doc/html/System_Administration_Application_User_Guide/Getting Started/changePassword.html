Change Password

The Change Password option allows you to change your password credentials. This option is also available from the icon version of the Select Application page, in the right corner of the menu bar.

To change your password, do the following:

  1. Enter your Current Password in the field provided.
  2. Enter your new password in the New Password field.
  3. Re-enter your new password in the Confirm New Password field.
  4. Click the Change Password button to change your password.
