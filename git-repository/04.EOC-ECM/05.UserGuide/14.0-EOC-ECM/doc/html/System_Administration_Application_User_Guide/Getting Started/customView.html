Custom Views

The custom view enables users to search, select, and save particular records of a finder. In System Administration, you can perform the following custom view actions for the current finder:

  • Add a new custom view
  • Change an existing custom view
  • Delete a custom view

Add a New Custom View

To add a new custom view, complete these steps:

  1. From your finder search results, click View and then select Add Custom View.



  2. The New Custom View dialog appears. The following fields are available:

    Field Description
    Label

    The name of the label, which is mandatory.

    Note: This field cannot be changed once you have entered and saved your changes.
    Max Rows This field saves the current finder result to the database with the Label field used as the identifier. It denotes the maximum number of rows to retrieve. The maximum rows allowed is 999. Setting the maximum number of rows avoids caching memory.
    Run query at start Select this field to search the database for existing favorites.
    Show search form at start By default, this field is selected, which opens the selected custom view and displays the report on screen.
    Save Search Criteria

    By default, this field is selected, meaning that the finder's search criteria are saved for custom views and favorites. Otherwise, when this field is not selected, the search criteria are cleared when you leave your finder results.
    Save Table State By default, this field is selected, indicating that you want to save the table layout for your custom view. This field is always visible.
    Default Select this field to indicate that this custom view is the default view for this finder.

    Enter your information in the field and then click the Save button.



  3. A message appears at the top of the dialog, indicating that your custom view has been successfully saved. Click the Close button (X) to close the dialog.
  4. To verify that your custom view works, click the Export button and select the name of your custom view to use this view.


Change an Existing Custom View

To change an existing custom view, complete these steps:

  1. From your finder search results, click View and then select Modify Custom View.



  2. From the Custom View Details dialog, make any pertinent changes to your custom view and then click the Save button.
  3. A message appears at the top of the dialog, indicating that your custom view has been successfully saved. Click the Close button (X) to close the dialog.

Delete a Custom View

To delete a custom view, follow these steps:

  1. From your finder search results, click View > Delete View, and then select the custom view that you want from the list.



  2. A confirmation message appears, asking whether you want to delete your custom view. Click the Yes button to confirm.
  3. To verify that your custom view has been deleted, click the View button. Your custom view name no longer appears in this menu and has been successfully deleted.
