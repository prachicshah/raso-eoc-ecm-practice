System Administration Menu Options

The User Administration contains the following menu options:

  • AVMs
  • Processes
  • Services
  • View

AVMs Menu

The AVMs menu contains the AVM and Process Engine interfaces. The AVM menu provides controls for maximum users, license violations, AVM, and Logged In Users. The Process Engine provides information about process definitions and process activities.

Processes Menu

The Processes menu displays options to view information about running engines, processes, global processes, worklists, queues, and more.

The following table describes each Processes menu option:

Menu Item Function
Information View information about the currently running engines, processes, global processes, worklist, and queues.
Process Manager View a list of all the processes in the system.
Global Proccesses View a list of all global processes that are running all the time in the system.
Milestones View a list of all milestones associated with the selected process.
Process Queue View a list of messages that are waiting to be processed.
Stale Jeopardy Report View a list of all stale-dated processes and processes that will be stale-dated within 30 days.

Services Menu

The Services menu displays options to view a list of message log items, view a Node ID's interfaces, and test connectivity to external services.

The following table describes each Interfaces menu option:

Menu Item Function
Message Log View a list of messages in a report, based on your search criteria.
Interface Statistics Provides a statistical report of a Node ID's interfaces.
Interface Status Test external services for connectivity, availability, and status.
Queue Monitor View details about a specified queue, based on the specified search criteria.

View Menu

The View menu provides controls for managing event logs, information, select application, logout, change password, and more.

A description of these options appears in this table:

Menu Item Function
Event Log Displays a list of messages logged by the System Administration application. 
Logged Users Info Lists users that are logged on to a specified Node ID.
Users for AVM Allows you to see how many users are active on a specific node.
Third-Party Licenses Displays a list of third-party licenses required to use software products.
External Scripts Allows you to search, create, detail, and delete an external script.
