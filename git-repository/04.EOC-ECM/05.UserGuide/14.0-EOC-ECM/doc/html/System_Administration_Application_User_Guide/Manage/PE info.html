Process Engines (PE) Information

To access the PE Info report, do the following:

  1. In the AVM search results, select the checkbox next to the PE that you want to view.
  2. Click the PE Info button to launch a summary for the selected process engine (applies only to process engines).

The PE Info page contains the following tabs:

  • General
  • Cache
  • Activity Threads
  • Queues
  • Messages

To refresh the information on any tab, click the Refresh button.

General

The PE Info page's General tab provides information about the number of process, pending process activities, and memory details.

The General tab contains the following field information:

Field Description
Total Number of Processes The total number of processes in this PE
Pending Process Activities Number of process activities in the wait Queue
Used Memory (KB) Memory currently used by this PE (in megabytes)
Total Memory (KB) Total Memory allocated to this PE (in kilobytes)

Cache

Clicking the PE Info page's Cache tab provides participant and order cache details.

The Participate Cache section contains the following fields:

Field Description
Last DB Read Time of the last database read of Participant messages
Queue Count Size of the participant messages in PE cache
Next DB Read Time of the next scheduled database read of Participant messages
Waiting Message Count Number of process requests waiting for participant messages

Activity Threads

Clicking the PE Info page's Activity Threads tab provides a summary of the activity threads in this process engine.

The Activity Threads summary contains the following fields:

Field Description
Threads ID This field denotes the identifier for the activity thread.
Process ID This field contains the process identifier.
Started This field indicates the time and date that the activity thread was started.
Activity This field represents the activity that is to be processes on the activity thread.
Executing This field indicates whether the activity thread is processing an activity.

Queues

Clicking the PE Info page's Queues tab provides a summary of data in the queue.

The Queues tab contains the following fields:

Field Description
Message ID This field contains the unique message ID.
Consumer ID This field denotes the consumer ID.
Sender Type This field represents the sender type.
Operation This field contains the operation identifier.

Messages

Clicking the PE Info page's Messages tab provides a summary of queue messages.

The Messages tab contains the following fields:

Field Description
Message ID This field contains the unique message ID.
Consumer ID This field denotes the consumer ID.
Sender Type This field represents the sender type.
Operation This field contains the operation identifier.
Activity This field describes the activity to be processed.

Clicking the Detail icon to the left of the Message ID field provides details about the selected queue message.
