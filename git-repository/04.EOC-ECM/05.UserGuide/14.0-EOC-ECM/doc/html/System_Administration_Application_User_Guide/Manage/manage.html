Manage

The Manage menu contains the following options:

  • AVMs
    This page allows you to view information about processes in the runtime environment. The AVMs page contains the following actions that you can perform:

    • PE Info
      The Process Engine Information report displays detailed information about the select process engine, including cache and activity threads.

    • Memory Processes
      The Memory Processes page displays a list of all processes in the system.

  • AVMs History
    This page allows you to view all the records of archived and currently running AVMs.

Note: To view the Manage menu, ensure that your user ID has the following privileges set using the User Profile Management application:

  • Process Manager Administrator
  • Runtime Administrator
