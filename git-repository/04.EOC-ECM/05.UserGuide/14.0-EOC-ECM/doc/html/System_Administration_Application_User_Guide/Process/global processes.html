Global Processes

The Global Processes report displays a list of all global processes in the system. Global processes are processes that are running at all times.

To access the Global Processes report, do the following:

  1. From the menu bar, click Processes > Global Processes.
  2. On the Global Processes page, click the Search button to display a list of all global processes in the system. You can also specify your search criteria.


The following search fields are available:

Field Description
Process Status Status of the global process.
Process Type Type of global process.
Priority Priority of the global process.
Process Engine ID ID of the Process Engine running the global process.
Start Time Date and time when the global process was started. Click the Calendar icon to view the calendar and select a date. You can also specify the time by using the Hour, Minute, and Second drop-down fields.
Status Changed Date and time when the status of the global process ended. Click the Calendar icon to view the calendar and select a date. You can also specify the time by using the Hour, Minute, and Second drop-down fields.

Action Menu

The Action menu allows you to perform a number of process functions. Select the checkbox next to the process that you want from the search results, click the Action button, and then select one of the following options:

Option Description
Suspend Suspends the selected process.
Resume Attempts to resume the selected process.

Detail Menu

The Detail menu allows you to view information about a specific process. Select the checkbox next to the process that you want from the search results, click the Detail button, and then select of the following options:

Option Description
Event Log Activates an Event Log dialog (refer to section Event Log), which displays a list of messages associated with the selected process.
Activities Activates an Activity Finder dialog, which displays a list of activities associated with the selected process.

Activities

The Activities dialog displays a list of activities associated with the selected global process. To access the activities dialog, do the following:

  1. Select the checkbox next to the global process that you want from the list and then click Detail > Activities.
  2. The Activities dialog, which lists all activities associated with the selected global process that have been started, but have not completed.



    The Activities dialog contains the following fields:

    Field Description
    Started Date and time when the activity was started.
    Completed Date and time when the activity completed.
    Activity Description of the activity.

  3. To view details about a specific activity, click the Details icon next to the activity that you want to view its details.



  4. When you are done reviewing the information, click the Close (X) button in the right corner of the dialog to return to the Global Processes page.
