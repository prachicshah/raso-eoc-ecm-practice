Process Queue

The Process Queue report displays a list of messages that are waiting to be processed. To access this report, click Processes > Process Queue from the menu bar.

The following fields are available:

Field Description
Message ID ID of the message.
Created Date and time when the message was created.
Action Message action (operation).
Sender Type Sender metadata type.
Sender ID Sender ID (process ID or user ID).
Consumer ID Consumer ID (process ID).

To refresh the list of messages in the queue that are waiting to be processed, click the Search button.

You can view details about each message by clicking the Details icon next to the Message ID that you want.
