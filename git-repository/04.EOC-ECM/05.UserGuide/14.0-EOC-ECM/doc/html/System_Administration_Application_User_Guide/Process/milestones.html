Milestones

The Milestones report displays a list of all milestones associated with the selected process.

To access the Milestone report, complete these steps:

  1. From the menu bar, click Processes > Milestones to launch the Milestones screen.



  2. You can specify your search criteria in the fields provided. Otherwise, click the Search button to view a list of milestones for each process.

The following fields are available:

Field Description
Process ID ID of the process the milestone belongs to.
Milestone Activity The type of activity (user or global).
Start Time Date and time when the process was started.
End Time Date and time when the process was ended.
Participant ID ID of the User Participant Operation that the activity belongs to.
Participant Type Label of participant involved with the activity.
Expected Completion Date and time when the activity is expected to be completed.
Order ID ID of the Order instance that the process belongs to.
Due Date Due date of the corresponding milestone activity.
Activity Index Corresponding internal activity index.
Duration (secs) Number of business seconds (based on calendars) taken to complete this activity.
Process Duration (secs) Number of business seconds (based on calendars) taken to complete this activity.
