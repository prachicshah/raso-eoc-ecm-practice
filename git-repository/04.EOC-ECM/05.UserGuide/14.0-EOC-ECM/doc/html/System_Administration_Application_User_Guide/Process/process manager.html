Process Manager

The Process Manager report displays a list of all processes in the system.

To access the Process Manager report, complete these steps:

  1. From the menu bar, click Processes > Process Manager to launch the Process Manager screen.



  2. You can specify your search criteria in the fields provided. Otherwise, click the Search button to view a list of processes.

The following fields are available in the search results:

Field Description
Status

Process status. You can select a status from the following list:

  • Not started
  • Executing
  • Suspended
  • Completed
  • Aborted
  • Error
  • Terminated
  • Cancelled
  • Stale
Process Type Metadata process label. Click the drop-down menu and select a process type from the list.
Priority Process priority.
Order Type Type of Order the process belongs to.
Start Time - From Date and time when the message was sent. Click the Calendar icon to view the calendar and select a date. You can also specify the time by using the Hour, Minute, and Second drop-down fields.
Start Time - To Date and time when the message was received. Click the Calendar icon to view the calendar and select a date. You can also specify the time by using the Hour, Minute, and Second drop-down fields.
Process ID The unique process identifier.
Metadata Version Metadata version number.
Process Engine ID The process engine identifier.
Order ID ID of the Order that the process belongs to.
Changed Status Time - From Starting date and time when the process status was changed. Click the Calendar icon to view the calendar and select a date. You can also specify the time by using the Hour, Minute, and Second drop-down fields.
Changed Status Time - To Ending date and time when the process status was changed. Click the Calendar icon to view the calendar and select a date. You can also specify the time by using the Hour, Minute, and Second drop-down fields.

Action Menu

The Action menu allows you to perform a number of process functions. Select the checkbox next to the process that you want from the search results, click the Action button, and then select one of the following options:

Option Description
Suspend Suspends the selected process.
Resume Attempts to resume the selected process.
Terminate Terminates the selected process.
Cancel Cancels the selected process.
Resume Stale Resumes a stale-dated process.

Detail Menu

The Detail menu allows you to view information about a specific process. Select the checkbox next to the process that you want from the search results, click the Detail button, and then select of the following options:

Option Description
Event Log Activates an Event Log dialog (refer to section Event Log), which displays a list of messages associated with the selected process.
Milestones Activates a Milestones dialog (refer to section Milestones), which displays a list of milestones associated with the selected process.
Document Activates a Process Document dialog displaying the Document associated with the selected process.
Order Activates a dialog displaying the Order associated with the selected process.
Worklist Activates a Worklist Manager dialog (refer to section Worklist Manager), which displays a list of worklist items associated with the selected process.
Child Processes Activates a Process Manager dialog, which displays a list of processes spawned by the selected process.
Activities Activates an Activity Finder dialog, which displays a list of activities associated with the selected process.

History Menu

The History menu allows you to view a list of completed activities or worklist items associated with a specific process. Select the checkbox next to the process that you want from the search results, click the History button, and then click one of the following options:

Option Description
Activities Archive Activates an Activity Archive dialog, which displays a list of completed activities associated with the selected process.
Worklist Archive Activates a Worklist History dialog, which displays a list of completed worklist items associated with the selected process.
