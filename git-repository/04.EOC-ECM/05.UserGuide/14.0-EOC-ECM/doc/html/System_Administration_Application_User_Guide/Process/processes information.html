Information

The Information report displays information about the currently running engines, processes, global processes, worklist, and queues. To access this report, click Processes > Information from the menu bar.

The following field appears in the Engines section:

Field Description
Cluster Size Number of process engines running.

The following fields are available in the Processes section for both User and Global process types:

Field Description
Executing Number of processes executing.
Error Number of processes with errors.
Aborted Number of aborted processes.
Suspended Number of suspended processes.
Completed Number of completed processes.
Terminated Number of terminated processes.
