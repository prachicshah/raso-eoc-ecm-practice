Processes

The Processes menu contains the following actions that you can perform:

  • Information
    Displays information about the currently running engines, processes, global processes, worklist, and queues.

  • Process Manager
    Provides a list of all processes in the system.

  • Global Processes
    Displays processes that are running at all times.

  • Milestones
    Provides a list of all milestones associated with the selected process.

  • Process Queue
    Displays a list of messages that are waiting to be processed.

  • Stale Jeopardy Report
    Displays a list of all stale-dated processes and processes that will be stale-dated within 30 days, with the number of days being configurable.

Note: To view the Processes menu, ensure that your user ID has one of the following privileges set using the User Profile Administrator:

  • Process Manager Administrator
  • Worklist Administrator
