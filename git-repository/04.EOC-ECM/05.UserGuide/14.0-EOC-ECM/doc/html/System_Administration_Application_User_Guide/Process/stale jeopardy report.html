Stale Jeopardy Report

The Stale Jeopardy report displays a list of all stale-dated processes and processes that will be stale-dated within 30 days. You can configure the number of days in your search by using the Days to Stale field.

To access the State Jeopardy report, complete these steps:

  1. From the menu bar, click Processes > Stale Jeopardy Report to launch the Stale Jeopardy Processes screen.



  2. You can specify the following search criteria fields:

    Field Description
    Stale Status The status of the process. Click the drop-down menu and select the process that you want from the list.
    Process Type Metadata process label. Click the drop-down menu and select the process type that you want from the list.
    Priority Click the drop-down menu and select the process priority from the list.
    Order Type Click the drop-down menu and select the type of order that you want to search for from the list.
    Start Time - From Date when the process started. Click the Calendar icon to view the calendar and select a date. You can also specify the time by using the Hour, Minute, and Second drop-down fields.
    Start Time - To Date when the process ended. Click the Calendar icon to view the calendar and select a date. You can also specify the time by using the Hour, Minute, and Second drop-down fields.
    Days to Stale This field defaults to 30 days. Click the drop-down menu and select a different stale-dated period.
    Metadata Version The metadata version number.
    Process Engine ID ID of the Process Engine running the stale process.
    Order ID ID of the Order instance that the process belongs to.
    Changed Status Time - From Start date when the status of the process changed. Click the Calendar icon to view the calendar and select a date. You can also specify the time by using the Hour, Minute, and Second drop-down fields.
    Changed Status Time - To End date when the status of the process changed. Click the Calendar icon to view the calendar and select a date. You can also specify the time by using the Hour, Minute, and Second drop-down fields.

  3. Click the Search button to view a list of stale-dated processes.

Action Menu

The Action menu allows you to perform a number of process functions. Select a state process from the search results, click the Action button, and then select one of the following options:

Option Description
Suspend Suspends the selected process.
Resume Attempts to resume the selected process.
Terminate Terminates the selected process.
Cancel Cancels the selected process.
Resume Stale Resumes the selected stale-dated process.

Details Button

The Details menu allows you to view information about a specific process. Select a stale process from the search results, click the Details button, and then select one of the following options:

Option Description
Event Log Activates an Event Log dialog (refer to section Event Log), which displays a list of messages associated with the selected process.
Milestones Activates a Milestones dialog (refer to section Milestones), which displays a list of milestones associated with the selected process.
Document Activates a Process Document dialog displaying the Document associated with the selected process.
Order Activates a dialog displaying the Order associated with the selected process.
Worklist Activates a Worklist Manager dialog (refer to section Worklist Manager), which displays a list of worklist items associated with the selected process.
Child Processes Activates a Process Manager dialog, which displays a list of processes spawned by the selected process.
Activities Activates an Activity Finder dialog, which displays a list of activities associated with the selected process.

History Menu

The History menu allows you to view a list of completed activities or worklist items associated with a specific process. Select a stale process from the search results, click the History button, and then select one of the following options:

Option Description
Activities Archive Activates an Activity Archive dialog, which displays a list of completed activities associated with the selected process.
Worklist Archive Activates a Worklist History dialog, which displays a list of completed worklist items associated with the selected process.
