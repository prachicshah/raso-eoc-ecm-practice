Import User Profile Groups and Users

From the Provider menu, you can import groups and users that were previously exported to an XML file.

Import Users

To import users that were exported to an XML file, do the following:

  1. Click Providers > Import Users from the menu bar to launch the Import Users page.



  2. Specify in the Domain field the domain where the XML file resides.
  3. Click the Search button to display a list of User IDs and accompanying names.
  4. Select the User IDs that you want to import and then click the Select button.

Import Groups

To import users that were exported to an XML file, do the following:

  1. Click Providers > Import Groups from the menu bar to launch the Import Groups page.



  2. Specify in the Domain field the domain where the XML file resides.
  3. Click the Search button to display a list of group names and their descriptions.
  4. To import users belonging to a group, select the Import users checkbox.
  5. Select the groups that you want to import and then click the Select button.
