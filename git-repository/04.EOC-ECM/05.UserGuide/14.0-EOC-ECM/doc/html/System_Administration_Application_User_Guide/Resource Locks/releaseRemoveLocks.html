Remove Locks

You can remove or release the locks from active locks, pending locks, and all locks options of the Resource Locks menu. The following are the steps to remove the locks form active locks page

Note: The steps to remove the locks are same for pending locks and all locks pages.

  1. Click Resource Locks > Active Locks menu; the Active Locks page appears.
  2. Search for a specific active lock or click the Search button to display all the active locks.
  3. Click the checkbox for one of the locks, and click the Release button.

  4. A confirmation dialog appears, prompting you to confirm whether you want to remove the selected lock. Click the Yes button to continue.
  5. The active locks result page refreshes and a confirmation message appears that release request has been submitted successfully.

Note: After the successful removal, the lock appears with a yellow dot for the Pending Release column.
