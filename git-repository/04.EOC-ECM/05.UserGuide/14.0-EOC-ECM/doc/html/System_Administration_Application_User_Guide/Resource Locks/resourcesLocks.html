Resource Locks

The resource locking feature provides a mechanism to place constraints onÂ the orders. This section provides information on how to get started with resource locks menu:

  • Overview
    This page gives you an overview of resource locks features.

  • Configure Resource Locks
    This page provides you the information on how to configure this menu

You can perform the following actions with this menu:

  • Search Locks
    This option allows you to perform a search on locks.

  • Remove Locks
    This option provides you the ability to remove or release the locks.

The following menu options are available for the resource locks:

  • Active Locks
    This menu option allows you to view all active locks.

  • Pending Locks
    This menu option allows you to view pending locks.

  • All Locks
    This menu option allows you to view all locks.

  • Lock History
    This menu option allows you to view history of all locks.
