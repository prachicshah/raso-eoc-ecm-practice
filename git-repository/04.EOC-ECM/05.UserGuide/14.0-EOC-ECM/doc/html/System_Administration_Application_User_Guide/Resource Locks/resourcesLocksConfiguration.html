Configure Resource Locks

The following are the steps to configure the resource locks:

  1. Follow the directions from the Velocity Studio’s Installer User Guide to install Velocity Studio, initialize the database, and configure the System Configuration application.
  2. Create a project in Velocity Studio and set the internal name of the project.
  3. Add the following library files for Resource Locks menu in Velocity Studio:
    • cwa_rslock.jar
    • cwl_rslock.jar
    • rslock.jar
  4. Run the models.sql and privileges.sql files found in the <EOC-ECM_installation_folder>\modules\rslock\DDL. These files are required to create indexes, tables, and provide the rslock administration privilege.
  5. Update the logical connection in the System Configuration application.
  6. Upgrade the database and run the associated SQL file.
  7. Assign the privileges required for resource locks menu in the User Profile Management application.
  8. Logout and log back in to the System Administration application.
