Search Locks

You can search the information on the locks from each option of the Resource Locks menu. The following steps to perform the search are same for each page of the resource lock menu:

  1. Click Resource Locks > All Locks menu; the Resource Locks page appears.

  2. Enter your search criteria in any of the fields provided (for example, you can search by resource type, system, or requester ID).

    3. Note: The fields for the search function have been described on the active.

  3. Click the Search button. The results displays in the results area.
