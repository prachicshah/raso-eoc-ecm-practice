Interface Statistics

Clicking Services > Interface Statistics from the menu bar allows you to view details about an interface, based on the specified search criteria.

To search for statistics on an interface, enter your search criteria, and then click the Search button. Your search results appear in the area under the search criteria.

The following search fields are available:

Field Description
Record Date The date and time of the record. The default is the current date, with the time set at midnight.
Record Type Click the drop-down menu and select from one of the record types from the list.
AVM Enter the node ID.
