Queue Monitor

Clicking Services > Queue Monitor from the menu bar allows you to view details about a specified queue, based on the specified search criteria.

To search for statistics on an interface, enter your search criteria, and then click the Search button. Your search results appear in the area under the search criteria.

The following search fields are available:

Field Description
AVM Enter the node ID.
Interval Specify the interval in which to monitor the queue, by default, the interval is set to 1 minute.
Group by Queue Name By default, this field is unchecked. Select this field to group the results by queue name.
Queue Name Click this field's drop-down menu and select the queue name that you want to monitor from the list.
Queue Direction By default, both is the field's value. Click the field's drop-down menu and select from one of the values.
Show Average Duration By default, this field is unchecked. Select this field to show the average duration in the results.
