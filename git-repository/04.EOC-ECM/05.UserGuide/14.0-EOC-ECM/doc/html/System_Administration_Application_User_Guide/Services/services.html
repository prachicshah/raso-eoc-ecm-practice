Services

The Interfaces menu contains the following actions that you can perform:

  • Message Log
    The message logs allows you to view a list of messages in a report, based on your search criteria.

  • Interface Statistics
    This page provides a statistical report of a Node ID's interfaces.

  • Interface Status
    This page allows you to test external services for connectivity, availability, and status.

  • Queue Monitor
    This page allows you to view details about a specified queue, based on the specified search criteria.
