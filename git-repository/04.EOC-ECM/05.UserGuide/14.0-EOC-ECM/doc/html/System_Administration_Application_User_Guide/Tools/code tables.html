Code Tables

Code tables allow you to maintain all enumeration lists used in the application. You can access the Code Tables page by clicking Tools > Code Tables from the menu bar.

Note: The code tables in a table result having the model as a data structure display the description and not the code, to align with the document as model behaviour.

For information on how to use the Code Tables page, refer to the Catalog User Guide, under the Catalog Setup and Configuration > Code Tables section.
