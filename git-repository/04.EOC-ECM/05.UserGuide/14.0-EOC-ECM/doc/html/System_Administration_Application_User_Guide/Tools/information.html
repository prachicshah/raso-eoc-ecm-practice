General Framework Information

From the Information menu, the General Framework Information report displays information about the current metadata, license, and servers.

The Information screen contains the following fields:

Field Description
Node ID Configuration file name, including path.
Version The product version number.
Session Timeout Maximum length of time a session can last with no activity.
Test Mode Indicates whether the framework is in test mode.
Metadata Host Database schema containing current metadata.
Metadata Label Description of the current metadata.
Metadata Name Name of the current metadata.
Activation Version Metadata activation version number.
Activated Date The date on which the metadata was activated.
License A list of the valid licenses.
