Tools Menu

The Tools menu contains the following actions that you can perform:

  • Information
    The General Framework Information report displays information about the current metadata, license, and servers.

  • Order Statistics
    This report displays a list of orders and associated statistics for metadata.

  • Code Tables
    Code tables allow you to maintain all enumeration lists used in the application. See the Catalog User Guide's Catalog Setup and Configuration > Code Tables section for details.

  • External Script
    In the system metadata, you can search, create, detail, and delete an external script.

  • User Actions
    The User Actions page allows you to view a list of actions that were performed by users and the duration of each action.
