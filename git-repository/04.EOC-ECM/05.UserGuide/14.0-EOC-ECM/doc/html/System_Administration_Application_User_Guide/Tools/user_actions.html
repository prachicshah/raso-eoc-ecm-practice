User Actions

The User Actions page allows you to view a list of actions that were performed by users and the duration of each action. To view this information, complete these steps:

  1. From the menu bar, click Tools > User Actions.



  2. Specify any of the following search criteria:
    • Enter the User ID of the user who performed the action.
    • Enter the Action Name what was performed.
    • Click the DateTime field's icon and specify the date start and end intervals.

  3. Click the Search button. The results appear under your search criteria.

The results contain the following fields:

Menu Item Function
User ID The Node ID that the user is logged on.
Action Name The action that the user was performing.
Date/Time (sec) The date and time that the user logged on and was performing the specified action.
