External Scripts

In the system metadata, you can search, view, create, and delete an external script. To open the External Script Search screen, click View > External Scripts from the menu bar.

You can perform the following tasks with an external script:

  • Search for an external script
  • Add an external script
  • Delete an external script
  • View an external script's details and make changes to it
  • Export a script into an existing external script
  • Import an external script

Search for an External Script

To search for an existing external script, follow these steps:

  1. In the Script Name field, enter the script name. You can enter the first few characters of the script name.
  2. If you know the Script Type, fill in that search field.
  3. If you want to specify the Application, click the field's drop-down menu and select the application that you want.
  4. Click the Search button. Your search results appear in the results area, under the search criteria section.

Add an External Script

To add an external script, do the following:

  1. From the External Scrips screen, click the New button. The New External Script screen appears.



  2. The following fields are mandatory on the External Script Details screen:

    Field Description
    Script Name Enter a name for your script in this field. The external script name must be unique for each script type.
    Script Type

    Enter the type of script in this field. The maximum number of characters allowed for the script type is two.

    Note: Types starting with a, b, and c are reserved for internal AVM purposes. Do not define external script types starting with these letters. Otherwise, using them may result in application crashes.
    Valid Dates - From Enter the date and time from which the script is available in this field.

  3. The following fields are optional:

    Field Description
    Application Click the drop-down menu and select the application associated with the external script for the list.
    Valid Dates - To This field denotes last date that the script is available.
    Is Constant By default, this property is not selected, indicating that the script is not a constant.
    Script Status This field represents the script's status.
    Import Script File

    If you have an existing script file that you want to use, you can import it by completing these steps:

    1. Click the Browse button and select the script file that you want to import.
    2. Click the Upload button. The Script field contains content from your script file.

  4. To create your script, enter your scripting code in the Script field.
  5. When you are finished completing the fields, click the Save button to save your script. You can locate your script by using the External Script page to conduct a search.

Delete an External Script

To delete an external script, complete the following steps:

  1. From the External Script Search screen, select the checkbox next to the script name that you want to delete from the search results, and then click the Delete button.



  2. A confirmation message appears, prompting whether you want to delete your script. Click the Yes button to continue.
  3. The results refresh and your script has been deleted.

View Details of and Edit an External Script

You can view the contents of an existing external script by doing the following:

  1. From the External Script Search screen, double-click the script name from the search results to launch the External Script Details screen.



  2. You can make changes to the fields on the External Script Details page. To make changes to your code, click the Edit button, and then make your changes.
  3. When you have completed your changes, click the Save button. A confirmation message appears at the top of the page, indicating that your changes have been saved..

Export an External Script

To export your external script to a JavaScript file, complete these steps:

  1. Open the external script that you want to export.
  2. From the External Script Details screen, click the Export Script button.
  3. Depending on the Web browser that you are using, you are prompted to either open or save your file. Choose either Save or Save as. For the latter option, specify the name of your .js file and where you want to save the file. The default filename is cwf.js.

Import an External Script

To import a script to your external script, do the following:

  1. Open the external script into which you want the script exported.
  2. From the External Script Details screen, locate the Import Script File field, click the Browse button, and then select your JavaScript file that you want to import.
  3. Click the Upload button. The Script field contains content from your script file.
  4. A dialog appears, prompting you to specify the .js file that you want to import. Click the Browse button to locate your JavaScript file, and then click the Upload button.
  5. The contents of your script appear in the script area of the External Script Details screen. Click the Save button to save your changes.
