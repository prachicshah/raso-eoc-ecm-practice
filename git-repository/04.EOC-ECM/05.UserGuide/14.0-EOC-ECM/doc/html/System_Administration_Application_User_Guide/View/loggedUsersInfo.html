Logged Users Info

The Logged Users Info page lists users that are logged on to a specified Node ID. To view information about logged users, do the following:

  1. From the menu bar, click View > Logged Users Info.
  2. On the Logged Users Info page, specify your search criteria. The following fields are available:

    Field Description
    User ID This field contains the user that is logged on to the node.
    Node ID This field represents the node set in the configuration application. Click the field's drop-down menu and select the node identifier from the list.

  3. Click the Search button to view a list of users logged on to the specified node.



    The results contain the following fields:

    Menu Item Function
    Node ID This field contains the node identifier that the user is logged on.
    User ID This field denotes the user who is logged on to the specified node.
    Login Time The field indicates the date and time that the user logged on.
    Email This field contains the user's e-mail address, if available.
    Logon (Minutes) The field contains the number of minutes that the user has been logged on to the application in minutes.
