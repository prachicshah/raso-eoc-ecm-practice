Third-Party Licenses

The Third-Party Licenses page allows you to view the details of each license. To view third-party license information, complete these steps:

  1. From the menu bar, click View > Third-Party Licenses.



  2. Select the third-party license file that you want from the left pane. Alternatively, you can enter your search criteria in the Search field and press the Enter key to narrow down your search results.



  3. Details of the third-party license you have selected appear in the right pane.
