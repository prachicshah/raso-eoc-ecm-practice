Users for AVM

The Users for AVM page allows you to see how many users are active on a specific node.

To access this functionality, complete these steps:

  1. From the menu bar, click View > Users for AVM.
  2. On the Users for AVM page, click the Node ID field's drop-down menu and select the node that you want.



    Click the Search button to view your results.

  3. In this example, the selected node has one active user.


The Users for AVM search results contain the following fields:

Field Description
Id Numeric event-message ID.
Node ID This field denotes the node set in the configuration application.
User Count This field indicates the number of users currently logged on to this node at runtime.
