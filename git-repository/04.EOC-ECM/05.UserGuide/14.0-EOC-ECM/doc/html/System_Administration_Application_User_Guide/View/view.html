View

The View menu contains the following actions that you can perform:

  • Event Log
    The Event Log report displays a list of messages logged by the System Administration application.

  • Logged in Users Info
    The Logged Users Info page lists users that are logged on to a specified Node ID.

  • Users for AVM
    The User Actions page allows you to view a list of actions that were performed by users and the duration of each action.

  • Third-Party Licenses
    The Third-Party Licenses page allows you to view the details of each license.

  • External Script
    In the system metadata, you can search, create, detail, and delete an external script.
