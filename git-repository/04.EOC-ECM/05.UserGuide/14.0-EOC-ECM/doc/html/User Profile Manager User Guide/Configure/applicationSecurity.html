Application Security

The Application Security page allows you to specify the security configuration for each application in your metadata project. To access this page, click Configure > Application Security from the menu bar.

To specify a security configuration, do the following:

  1. From the Application Security page, click the Security Configuration field next to the application that you want, and then select from the options available.



    Note: To create security configurations, you much click Configure > Security Providers from the menu bar before accessing the Application Security Configuration page.

  2. Your changes appear in blue font. When you have completed your changes, click the Save All button.



  3. A confirmation message appears, indicating that your changes have been saved.
