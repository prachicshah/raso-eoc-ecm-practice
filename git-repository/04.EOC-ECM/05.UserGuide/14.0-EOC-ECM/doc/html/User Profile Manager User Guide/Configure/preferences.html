Preferences

The Preference Configuration page allows you to configure the options available from Preferences page, which you can access from the preferences and logout bar. You can perform the following actions from the Preference Configuration page:

  • Create a new preference
  • Edit an existing preference
  • Delete a preference

Create a New Preference

To create a preference, complete these steps:

  1. From the menu bar, click Configure > Preferences to launch the Preferences Configuration page, and then click the New button.



  2. The New Preference Definition page appears.



    The following fields are available for entry:

    Field Description
    Name Enter a name for your preference setting. This field is mandatory.
    Type Click the drop-down menu and select the type from the list.
    Label Enter a label name. This field is mandatory.
    Default Value Enter a default value to which your preference is automatically set.
    Group Code If you require a group code for your preference, click the drop-down menu and select one from the list.
    Code Table Click the drop-down menu and select a code table from the list.

  3. Click the Save button to create your new preference setting. A confirmation message appears under the menu bar, indicating that your preference definition has been successfully saved.
  4. To view your new preference setting, click the Preferences link from the preferences and logout bar to launch the Preferences page. Your new setting appears on the page.


Edit an Existing Preference

To edit a preference, complete these steps:

  1. From the Preference Configuration page, select the checkbox next to the preference that you want to edit.
  2. All fields in that row, except for the Name, can be changed. Make the pertinent changes to your preference and then click the Save All button.



  3. A confirmation message appears under the menu bar, indicating that your changes have been successfully saved.


Delete a Preference

To delete a preference, complete these steps:

  1. From the Preference Configuration page, select the checkbox next to the preference that you want to remove and then click the Delete button.



  2. A confirmation dialog appears, prompting you to confirm deleting the selected preference. Click the Yes button.
  3. The Preference Configuration page refreshes and your selected preference has been successfully deleted.
