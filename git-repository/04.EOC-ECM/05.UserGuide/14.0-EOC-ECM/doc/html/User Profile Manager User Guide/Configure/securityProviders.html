Security Providers

You can set a security provider or service provider, or both, which differs from the default providers in the product by clicking Configure > Security Providers from the menu bar.

You can perform the following actions from the Security Providers page:

  • Create a new security provider
  • Edit an existing security provider
  • Delete a security provider
  • Set a security provider as the default

Create a New Security Provider

To create a preference, complete these steps:

  1. From the menu bar, click Configure > Security Providers to launch the Security Providers page, and then click the New button.



  2. The New Security Configuration page appears.



    The following fields are available for entry:

    Field Description
    Configuration Name Enter a name for your security provider configuration. This field is mandatory.
    Default Configuration Tthe product's security provider is the default. If you want to use this configuration as the default configuration, select this property.
    Profile Module Click this field's drop-down menu and select a product module from the list. This field is mandatory.
    Profile Data Enter your profile data in this field.
    Security Module Click the drop-down menu and select a security module from the list. Select a value from this field enables the Security Type field.
    Security Data

    Enter security data for your security module. For details, see the following sections:

    • Security provider characteristics
    • Set up an LDAP provider
    Security Type Click the drop-down menu and select a security type from the list.

  3. Click the Save button to create your new security provider. A confirmation message appears under the menu bar, indicating that your security provider configuration has been successfully saved.
  4. Click the Back button to return to the Security Providers page. Your newly created security provider configuration appears on the page.

Edit an Existing Security Provider

To edit a security provider, complete these steps:

  1. From the Security Providers page, double-click the Configuration Name that you want to edit to launch the Security Configuration Details page.
  2. Click the Edit button.



  3. Make any pertinent changes to the fields, and then click the Save button to save your changes.



  4. A confirmation message appears under the menu bar, indicating that your changes have been successfully saved.

Delete a Security Provider

To delete a security provider, complete these steps:

  1. From the Security Providers page, select the checkbox next to the Configuration Name that you want to delete, and then click the Delete button.



  2. A confirmation dialog appears, prompting you to confirm deleting the selected security configuration. Click the Yes button.
  3. The Security Providers page refreshes and your selected security configuration has been successfully deleted.

Set a Security Provider as the Default

To set a security provider as the default, follow these steps:

  1. From the Security Providers page, select the checkbox next to the Configuration Name that you want as the default, and then click the Set as Default button.



  2. The Security Providers page refreshes. A confirmation mesage appears, indicating that the default setting has been saved for your selected configuration.
