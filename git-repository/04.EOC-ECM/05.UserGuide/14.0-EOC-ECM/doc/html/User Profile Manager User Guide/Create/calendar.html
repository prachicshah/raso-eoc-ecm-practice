Create a Calendar

You can create calendar type objects based on one of the existing calendar templates. 

To create a new calendar, perform these steps:

  1. From the menu bar, click Create > Calendar to launch the New Calendar page.



  2. Complete the following fields:

    Field Description
    Calendar

    Enter the calendar type name. This field is mandatory. Once you have created this calendar type, it cannot be renamed. You can reference the name in your JavaScript scripts.
    Label

    Enter a label for the calendar type. This field is mandatory. The label appears in the user interface.
    Template

    Click the drop-down menu and select an existing calendar template from the list that can be used as a base for creating the current calendar type. This field is mandatory.

    Note: After creating a calendar template, it is recommended that you wait at least a minute to create a calendar. The cache needs this period to refresh the calendar template list and have your newly created calendar template appear in this field.

    Is Default Select this checkbox if you want to set this calendar as the default for groups. By default, this property is unchecked.
    First Day Click the drop-down menu and select the first day of the week to be used in the yearly calendars for this calendar type. If you do not select a day from the list, the default system setting is used.
    Work Hours

    Specify the starting work hour and ending work hour in the fields, respectively, to be applied by default for all working days in the yearly calendar.

  3. Click the Save button to create your new calendar type. A confirmation message appears under the menu bar, indicating that your calendar has been successfully saved.



  4. To create another calendar type, click the New button. Otherwise, you can configure your new calendar using the Years tab, or at a later time by clicking Manage > Calendars from the menu bar.

Years Tab

The Years tab allows you to create a yearly calendar object. This object presents a business calendar to be used by process engines.

The initial business calendar properties depend on the calendar template object that the calendar type object is based on. For example, the calendar template object specifies the recurring days (for example, Saturday and Sunday make up a weekend). These recurring days are related to the business calendar through the calendar template of the calendar type object that it falls under.

You can perform the following actions from the Years tab:

  • Add a yearly calendar object and configure calendar days
  • Remove a yearly calendar object

Add a Yearly Calendar Object

To add a new yearly calendar object and configure specific calendar days, do the following:

  1. Click the Years tab, and then click the Add button to create a yearly calendar object.



    The business calendar for the next year in the sequence appears in the results table. Before this new business calendar can be saved in the database, it must be customized by updating the properties of at least one day. Otherwise, an error message occurs.

  2. Double-click the newly created year object to launch the Calendar Year Details dialog, and then click the Edit button.



  3. Click the Add button to launch the New Calendar Day dialog, where you can add specific days to your calendar. These calendar customizations override any properties specified by the calendar template object on which the calendar is based.



  4. Complete the following fields:

    Field Description
    Day Name

    Enter the name of your day. This field is mandatory.
    Day Type

    Click the drop-down menu and select the day type from the list. This field is mandatory.

    Date

    Click the calendar icon to specify a specific date. This field is mandatory.
    Work Slots 1, 2, and 3 Specify the from and two periods for these work slots, if needed.

  5. When you have completed entering your information, click the Save button.



    A confirmation message appears, indicating that your calendar day has been successfully saved. Click the Close button (X) to close the dialog.

  6. Your new calendar day appears on the Calendar Year Details page.



    To add another calendar day, click the Add button and repeat the previous steps. Otherwise, to remove a calendar day, select the checkbox next to the day name that you want to remove, and then click the Remove button. Click the Yes button to confirm.

  7. Click the blue Back button to return to the Years tab of your calendar, where the Days field contains the number of days that you have configured.



  8. Click the General tab, and then click the Save button to save your changes. 

Remove a Yearly Calendar Object

To remove a yearly calendar object, complete these steps:

  1. From the Years tab, select the checkbox next to the Year that you want to remove, and then click the Remove button.



  2. The Years tab refreshes and a confirmation message appears, indicating that the yearly calendar object has been successfully removed from the calendar.
