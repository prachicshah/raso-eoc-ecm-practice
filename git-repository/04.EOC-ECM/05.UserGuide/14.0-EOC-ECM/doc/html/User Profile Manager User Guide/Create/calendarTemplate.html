Create a Calendar Template

The calendar template allows you to set up user availability, and contains default recurrences for USA and Canada. You can use this information when configuring calendars or creating a custom calendar template.

To create a new calendar template, perform these steps:

  1. From the menu bar, click Create > Calendar Template to launch the New Calendar Template Type page.



  2. In the Template field, enter the name of your template, and then click the Save button to create your new calendar type. A confirmation message appears under the menu bar, indicating that your calendar has been successfully saved.



Note: You can configure your new calendar template by complete the section steps that follow, or at a later time by clicking Manage > Calendar Templates from the menu bar.

Template Calendar Day

You can perform the following actions:
  • Add a template calendar day
  • Remove a template calendar day

Add a Template Calendar Day

To add a new template calendar day, do the following:

  1. From the Calendar Template Type Details page, click the Add button to launch the New Calendar Template Day page.



  2. The following fields are available:

    Field Description
    Day Name

    Enter the name of your day. This field is mandatory.
    Template This field displays the template name and is not editable.
    Day Type

    Click the drop-down menu and select the day type from the list. This field is mandatory.

    Recurrence Type

    Click the drop-down menu and select the type of recurrence for your calendar day. Depending on your selection from the drop-down list, additional drop-down fields display. In this example, Family Day occurs on the first Monday every February.

    The following options are available from this field:

    • Weekly - Specify the weekday that the recurrence takes place.
    • Monthly Every - Select the day of the month that the recurrence takes place.
    • Monthly The - Select the weekday within the month that the recurrence takes place.
    • Yearly Every - Select the year that the yearly recurrence takes place.
    • Yearly The - Select the parameters for the yearly setting that the recurrence takes place.
    • Script - In the Calendar Script field, specify the script that you want to use. You can create your scripts in Velocity Studio, in the Scripts folder, under the Business Processes folder.

    Note: Any changes made in this section are not saved to the database until you click the Save button.
    Date

    Click the calendar icon to specify a specific date. This field is mandatory.
    Working Day Select the field to indicate that this day is a working day. By default, the field is unchecked.




  3. When you have finished entering your information, click the Save button. A confirmation message appears, indicating that your calendar template day has been saved.
  4. Click the Back button to return to the Calendar template Type Details page. Your new calendar name appears on this page.



  5. Complete the following fields:

    Field Description
    Day Name

    Enter the name of your day. This field is mandatory.
    Day Type

    Click the drop-down menu and select the day type from the list. This field is mandatory.

    Date

    Click the calendar icon to specify a specific date. This field is mandatory.
    Work Slots 1, 2, and 3 Specify the from and to periods for these work slots, if needed.

  6. When you have completed entering your information, click the Save button.



    A confirmation message appears, indicating that your calendar day has been successfully saved. Click the Close button (X) to close the dialog.

  7. Your new calendar day appears on the Calendar Year Details page.

Note: Once you have created a new recurrence, you can change its name, day type, and recurrence pattern before clicking the Save button. Once your recurrences is stored in the database, its name cannot be changed.

Remove a Template Calendar Day

To remove a template calendar day, complete these steps:

  1. From the Calendar Template Type Details page, select the checkbox next to the Day Name that you want to remove from the calendar template, and then click the Remove button.



  2. The day that you have selected is removed from your calendar template.
