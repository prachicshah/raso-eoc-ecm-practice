Create

The Create menu contains the following actions that you can perform:

  • Group
    You can create a new group, and configure its users and privileges.

  • User
    This option allows you to create a new user and configure its groups to which it belongs.

  • Privilege
    You can create a new privilege and configure its groups to which it belongs.

  • Calendar
    This option allows you to create a new business calendar, as well as create calendar years and days that are used by process engines.

  • Calendar Template
    You can create a new calendar template and configure its days.

Notes:

  • To have full access to all functions under the Create menu, ensure that your user ID has the User Profile Administrator privilege set using the User Profile Management application.
  • If your user ID has only the Local Profile Administrator privilege set, you only have access to creating the following:
    • Group
    • User
    • Calendar
    • Calendar Template
