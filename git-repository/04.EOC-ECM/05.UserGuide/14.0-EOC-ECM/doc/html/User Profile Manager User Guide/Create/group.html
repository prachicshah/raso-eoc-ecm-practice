Create a Group

To create a new group, perform these steps:

  1. From the menu bar, click Create > Group to launch the New Group page.



  2. Complete the following fields:

    Field Description
    Group Name Enter a name for your group. This field is mandatory. Once you have created the group, it cannot be renamed.
    Description Provide a description of the group. This field is mandatory.
    Calendar Click the drop-down menu and select a calendar from this list, if required.

  3. Click the Save button to create your new group. A confirmation message appears under the menu bar, indicating that your group has been successfully saved.



  4. To create another group, click the New button. Otherwise, you can configure your new group using the Users and Privileges tabs, or at a later time by clicking Manage > Groups from the menu bar.

Users Tab

The Users tab shows a list of users associated with the group and the status of each user.

You can perform the following actions from the Users tab:

  • Add a user to a group
  • Remove a user from a group
  • Move users from one group to another

Add a User to a Group

To add a user to a group, complete these steps:

  1. Click the Users tab, and then click the Add button to launch the Search User dialog.



  2. Select the checkbox next to the User ID that you want to add to the group, and then click the Select button.



  3. The Users page appears. Your selected user has been added to your group.

Remove a User from a Group

To remove a user from a group, complete these steps:

  1. From the Users tab, select the checkbox next to the User ID that you want to remove from the group, and then click the Remove button.



  2. The Users tab refreshes and a confirmation message appears, indicating that the user has been successfully removed from the group.

Move Users from One Group to Another

To move users from one group to another one, do the following:

  1. From the Users tab, select the checkbox next to the User ID that you want to move users from, and then click the Move Users button.



  2. The Search Group dialog appears. Select the checkbox next to the group that you want to move your users to, and then click the Select button.



  3. The Users tab reappears, with the users that you have selected no longer appearing on this page. A confirmation message also displays, indicating that you have successfully moved your selected users to another group.

Privileges Tab

The Privileges tab shows a list of privileges for the selected group.

You can perform the following actions from the Users tab:

  • Add a privilege to a group
  • Remove a privilege from a group

Add a Privilege to a Group

To add a privilege to a group, complete these steps:

  1. Click the Privileges tab, and then click the Add button to launch the Search User dialog.



  2. The Search Privilege dialog appears. Select the checkbox next to the privileges that you want to add to your group, and then click the Select button.



  3. The Privileges tab refreshes with the selected privileges added to your group. A confirmation message appears, indicating that the selected privileges have been succesfully added to your group.


Remove a Privilege from a Group

To remove a privilege from a group, complete these steps:

  1. From the Privileges tab, select the checkbox next to the User ID that you want to delete from the group, and then click the Remove button.



  2. The Privileges tab refreshes and a confirmation message appears, indicating that the privilege has been successfully removed from the group.
