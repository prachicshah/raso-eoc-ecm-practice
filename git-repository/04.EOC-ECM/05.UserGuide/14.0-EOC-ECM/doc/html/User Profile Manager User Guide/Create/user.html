Create a User

To create a new user, perform these steps:

  1. From the menu bar, click Create > User to launch the New Group page.



  2. Complete the following fields:

    Field Description
    User ID

    Enter a username. This field is mandatory. Field restrictions are dependent on the security profile provider being used.

    The default security provider is an internal one, which states that this field cannot start with a number, although numbers are permitted after the first character, and cannot contain spaces. Once you have created this user, this field cannot be changed.
    Name Enter the name of the user. This field is mandatory.
    Email Enter the user's e-mail address in this field.
    Active By default, the new user is active in the system. Uncheck this field if you want your user to be inactive.
    Available By default, this field is selected and is not editable. It represents whether the user is available for task assignment.
    Minimum Effort

    This field indicates the minimum sum of all task efforts. This field is mandatory.

    When the value of the Current Efforts number is less than the value in this field, the system tries to assign the next task to the user. By default, this field has a value of 3. This field's value must be numerical.
    Maximum Effort

    This field denotes the maximum sum of all task efforts that can be assigned to the user by system distribution. This field is mandatory.

    By default, this field has a value of 10.This field's value must be numerical.
    Org Chart Position Click the Finder icon next to this field to launch the Organizational Chart menu and identify the user's position within the organization.

  3. Click the Save button to create your new user. A confirmation message appears under the menu bar, indicating that your user has been successfully saved.



  4. To create another user, click the New button. Otherwise, you can configure your new user using the Groups tab, or at a later time by clicking Manage > Users from the menu bar.

Notes:

  • The default values of user general properties can be changed by overriding the User Profile document and initialization rules for the appropriate fields.
  • Refer to the Velocity Studio User Guide > User Participant Tasks Distribution > User Participant section for more details on how the Minimum Effort and Maximum Effort  numbers are used by the system.

Groups Tab

The Groups tab shows a list of groups to which the user belongs.

You can perform the following actions from the Users tab:

  • Add a group to a user
  • Remove a group from a user

Add a Group to a User

To add a group to a user, complete these steps:

  1. Click the Groups tab, and then click the Add button to launch the Search Group dialog.



  2. Select the checkbox next to the Group Name that you want to add to the user, and then click the Select button.

    Note: Groups must be previously created to be shown in the Search Group dialog. 



  3. The Groups page appears. Your selected group has been added to your user.

Remove a Group from a User

To remove a group from a user, complete these steps:

  1. From the Groups tab, select the checkbox next to the Group Name that you want to remove from the user, and then click the Remove button.



  2. The Groups tab refreshes and a confirmation message appears, indicating that the group has been successfully removed from the user.
