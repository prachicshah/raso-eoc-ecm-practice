User Profile Manager Menu Options

The User Profile Manager contains the following menu options:

  • Create
  • Manage
  • Configure
  • Import
  • View

Create Menu

The Create menu allows you to create the following:

Menu Item Description
Group Create a new group.
User Create a new user.
Privilege Create a new privilege.
Calendar Create a new calendar.
Calendar Template Create a new calendar template type.

Manage Menu

The Manage menu contains the following menu options:

Menu Item Function
Groups Add, change, and delete groups, and configure a group's users and privileges.
Users Add, change, and delete users, and configure groups to which these users belong.
Privileges Change and delete privileges, and configure groups to which these privileges belong.
Organizational Charts Add, edit, and remove company organization charts, and configure users belonging to them.
Calendars Change and delete users, as well as create calendar years and days that are used by process engines.
Calendar Templates Add, change, and delete calendar templates, and configure a calendar template's days.

Configure Menu

The Configure menu has the following menu options:

Menu Item Description
Security Providers Configure security providers from the User Profile Manager application.
Preferences Add, edit, and delete preference options that appear on the Preferences page.

Import Menu

The Import menu provides controls for managing event logs, information, select application, logout, change password, and more.

Menu Item Description
Export

Allows you to export the following:

  • Groups
  • Users
  • Privileges
  • Organization charts
Import Provides a statistical report of a Node ID's interfaces.

View Menu

The View menu contains the Event Log, which Displays a list of messages logged by the User Profile Manager.
