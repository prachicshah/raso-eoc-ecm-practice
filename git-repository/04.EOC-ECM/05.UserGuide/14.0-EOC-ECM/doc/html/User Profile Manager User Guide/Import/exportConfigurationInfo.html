Export User Profiles

You can export the following configuration information from the User Profile Manager:

  • Groups
  • Users
  • Privileges
  • Organizational Chart

To export configuration information from the User Profile Manager, complete these steps:

  1. From the menu bar, click Import > Export.
  2. The Export dialog appears.



    By default, the following options are selected for export:
    • Groups
    • Users
    • Privileges
    • Organizational Chart

    Indicate the options that you want to export and then click the Export button.

  3. A popup message appears, prompting you to either open or save the file. By default, the application creates the cwSecurity.zip file that contains your confirmation information. Select whether you want to open or save your file.
