Import User Profiles

You can import the following configuration information from the User Profile Manager:

  • Groups
  • Users
  • Privileges
  • Organizational Chart

Notes:

  • Before you import a group, you must create the calendar record in the CWCALENDARS table, based on the calendar property value in the group XML.
  • Groups, users, organizational charts, and privileges are dependent on one another. As a result, if you want to import each of them individually, you must import each of them in the following order:
    1. Organizational charts
    2. Users
    3. Groups
    4. Privileges
  • After importing, if you attempt to export your configuration information and open the XML file, the following hold true:
    • Groups depend on both users and calendars
    • Users depend on organizational charts
    • Privileges depend on groups

To import configuration information from the User Profile Manager, complete these steps:

  1. From the User Profile Manager, click the Import button to launch the Import dialog.



  2. For the Select File field, click the Browse button and select the location of your XML file that you want to import.
  3. The Import Mode field defaults to Replace, which replaces your existing user profile configuration. Click the drop-down menu to change this mode to the following:
    • Create
    • Update

  4. The Max Error to Skip field defaults to 10 errors that are skipped. Specify a different value in this field, if required.
  5. By default, the following options are selected for import:
    • Groups
    • Users
    • Privileges
    • Organizational Chart

    Indicate the options that you want to import and then click the Import button.

  6. A confirmation message appears, indicating whether your import was successful, how many objects were imported, and whether any errors occurred. Click the Close button (X) to close the dialog.
