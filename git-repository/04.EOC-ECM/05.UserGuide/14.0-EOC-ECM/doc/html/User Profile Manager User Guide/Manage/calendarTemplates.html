Manage Calendar Templates

Clicking Manage > Calendar Templates from the menu bar provides a list of calendar templates.

You can manage calendar templates by performing the following actions:

  • Add a new calendar template
  • Delete a calendar template
  • Edit a calendar template

Add a New Calendar Template

You can click the New button from the Calendar Templates page to launch the New Calendar Template Type page.

See Create a calendar template for steps on how to create a new calendar template.

Delete a Calendar Template

To delete a calendar, complete these steps:

  1. From the Calendar Templates page, select the checkbox next to the Template that you want to delete, and then click the Delete button.



  2. A confirmation dialog appears, prompting you to confirm whether you want to delete the selected calendar template. Click the Yes button to continue.
  3. The page refreshes and your selected calendar template no longer appears on the Calendar Templates page.

Edit a Calendar Template

To edit a calendar template, complete these steps:

  1. From the Calendar Templates page, double-click the calendar template that you want to edit.
  2. Click the Edit button.



  3. Make any changes to your calendar, including double-clicking the Day Name and clicking that page's Edit button. Click the Save button to save your changes.
