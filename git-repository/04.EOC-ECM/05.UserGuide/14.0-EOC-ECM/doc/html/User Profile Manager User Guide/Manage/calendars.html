Manage Calendars

You can manage calendars by performing the following actions:

  • Search for a calendar
  • Add a new calendar
  • Delete a calendar
  • Edit a calendar

Search for a Calendar

To search for a calendar, perform these steps:

  1. From the menu bar, click Manage > Calendars to launch the Search Calendar page.



  2. To search for a calendar, do one of the following actions:
    • Enter your search criteria in these fields, and then click the Search button:

      Field Description
      Calendar

      Enter the calendar name.
      Label

      Enter the calendar's label.
      Template

      Click the drop-down menu and select the template name from the list.

    • Click the Search button without entering any information in the fields.

  3. A list of calendars appears in the search results area.

Add a New Calendar

You can click the New button from the Search Calendar page to launch the New Calendar page.

See Create a calendar for steps on how to create a new calendar.

Delete a Calendar

To delete a calendar, complete these steps:

  1. From the Search Calendar page's search results, select the checkbox next to the Calendar that you want to delete, and then click the Delete button.



  2. A confirmation dialog appears, prompting you to confirm whether you want to delete the selected calendar. Click the Yes button to continue.
  3. The page refreshes and your selected calendar no longer appears in the search results.

 

Notes:

  • Only those calendars that were created through the Create > Calendar menu will be permanently deleted. If the yearly calendar was the original default calendar of the calendar type object, a new standard calendar is recreated by restarting the User Profile Management application. The deletion of the default calendar results in the removal of all user-defined days, as the new standard calendar will not contain these previously defined properties.
  • When deleting a calendar type object, you must delete its yearly calendar objects first. Otherwise, an error appears.

Edit a Calendar

To edit a calendar, complete these steps:

  1. From the Search Calendar page's search results, double-click the calendar that you want to edit.
  2. Click the Edit button.



  3. Make any changes to your calendar, including details under the Years tab by clicking that page's Edit button. When you have finished, click the General Info tab, and then click the Save button to save your changes.
