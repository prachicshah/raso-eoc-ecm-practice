Manage Groups

You can manage groups by performing the following actions:

  • Search for a group
  • Add a new group
  • Delete a group
  • Edit a group

Search for a Group

To search for a group, perform these steps:

  1. From the menu bar, click Manage > Groups to launch the Search Group page.



  2. To search for a group, do one of the following actions:
    • Enter the Group Name in the field provided, and then click the Search button.
    • Click the Search button without entering any information in the Group Name field.

  3. A list of groups appears in the search results area.


Add a New Group

You can click the New button from the Search Group page to launch the New Group page.

See Create a group for steps on how to create a new group.

Delete a Group

To delete a group, complete these steps:

  1. From the Search Group page's search results, select the checkbox next to the Group Name that you want to delete, and then click the Delete button.



  2. A confirmation dialog appears, prompting you to confirm whether you want to delete the selected group. Click the Yes button to continue.
  3. The page refreshes and your selected group no longer appears in the search results.

Note: If an error appears, indicating that you cannot delete your group because there are dependent items, you must delete all users in your group first before you can delete the group itself.

Edit a Group

To edit a group, complete these steps:

  1. From the Search Group page's search results, double-click the group that you want to edit.
  2. Click the Edit button.



  3. Make any changes to your group, including details under the Users and Privileges tabs by clicking their respective page's Edit button. When you have finished, click the General Info tab, and then click the Save button to save your changes.
