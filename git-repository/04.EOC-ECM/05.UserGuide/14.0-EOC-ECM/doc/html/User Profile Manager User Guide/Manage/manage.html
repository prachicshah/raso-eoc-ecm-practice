Manage

The Manage menu contains the following actions that you can perform:

  • Groups
    You can add, change, and delete groups, and configure a group's users and privileges.

  • Users
    This option allows you add, change, and delete users, and configure groups to which these users belong.

  • Privileges
    You can change and delete privileges, and configure groups to which these privileges belong.

  • Organizational Charts
    This option allows you to add, edit, and remove company organization charts, and configure users belonging to them.

  • Calendars
    This option allows you to change and delete users, as well as create calendar years and days that are used by process engines.

  • Calendar Templates
    You can add, change, and delete calendar templates, and configure a calendar template's days.
