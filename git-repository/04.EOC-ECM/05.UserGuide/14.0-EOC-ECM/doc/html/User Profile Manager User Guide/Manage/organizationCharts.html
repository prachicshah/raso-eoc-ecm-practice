Manage Organization Charts

The Organizational Chart is the abstract definition of a company or organization. Users are later explicitly assigned to a specific position, and each position may determine application access (for example, performing order operations for limited products or regions). Records appearing under the Organization Chart node are sorted by name.

To manage your organization charts, you can perform the following actions:

  • Create a new organization chart
  • Create a new sub-organization chart under a top-level organization chart
  • Delete an organization chart
  • Add users to an organization chart
  • Remove users from an organization chart

Create a New Organization Chart

To create a new organization chart, perform these steps:

  1. From the menu bar, click Manage > Organization Charts to launch the Organization Charts page.
  2. Click the Add button (+) to launch the New Organizational Chart Node dialog.



  3. Enter the Name of your organization chart, and then click the Save button.



  4. Your organization chart name appears in the left pane. A confirmation message appears under the menu bar, indicating that the organization chart has been successfully saved.

Create a New Sub-Organization Chart Under a Top-Level Organization Chart

To create a new organization chart as a subchart of an existing organization chart, perform these steps:

  1. In the left pane, select the top-level organization chart that you want, and then click the Add button to launch the New Organizational Chart Node dialog.



  2. Enter the Name of your sub-organization chart, and then click the Save button.
  3. In the left pane, expand the node icon. Your newly created sub-organization chart appears under your top-level organization chart.

Delete an Organization Chart

To delete an organization chart, complete these steps:

  1. In the left pane, select the organization chart that you want, and then click the Delete button.



  2. A confirmation dialog appears, prompting you to confirm whether you want to delete the selected organization chart. Click the Yes button to continue.
  3. The page refreshes and your selected organization chart no longer appears in the left pane.

Add Users to an Organization Chart

To add users to an organization chart, complete these steps:

  1. From the left pane, select the organization chart that you want. From the Company Organization Chart Users section, click the Add button to launch the Search user dialog.



  2. Select the checkbox next to each user that you want to add to your organization chart, and then click the Select button.



  3. The Company Organization Chart page reappears, displaying your newly added users. A confirmation message appears, indicating that the selected users have been added successfully.



Remove Users From an Organization Chart

To remove users from an organization chart, do the following:

  1. From the Company Organization Chart Users page, select the checkbox next to the User ID that you want to remove, and then click the Remove button.



  2. A confirmation message appears, to verify that you want to delete the specified users. Click the Yes button to continue.
  3. The Company Organization Chart Users page refreshes. The users that you have selected have been removed from the Organization Chart.
