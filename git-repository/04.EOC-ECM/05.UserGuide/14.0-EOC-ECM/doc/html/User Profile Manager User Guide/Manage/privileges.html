Manage Privileges

You can manage privileges by performing the following actions:

  • Search for a privilege
  • Add a new privilege
  • Delete a privilege
  • Edit a privilege

Search for a Privilege

To search for a privilege, perform these steps:

  1. From the menu bar, click Manage > Privileges to launch the Search Privilege page.



  2. To search for a group, do one of the following actions:
    • Enter the Privilege in the field provided, and then click the Search button.
    • Click the Search button without entering any information in the Privilege field.

  3. A list of privileges appears in the search results area.

Add a New Privilege

You can click the New button from the Search Privilege page to launch the New Privilege page.

See Create a privilege for steps on how to create a new privilege.

Delete a Privilege

To delete a privilege, complete these steps:

  1. From the Search Privilege page's search results, select the checkbox next to the Privilege that you want to delete, and then click the Delete button.



  2. A confirmation dialog appears, prompting you to confirm whether you want to delete the selected privilege. Click the Yes button to continue.
  3. The page refreshes and your selected privilege no longer appears in the search results.

Edit a Privilege

To edit a privilege, complete these steps:

  1. From the Search Privilege page's search results, double-click the privilege that you want to edit.
  2. Click the Edit button.



  3. Make any changes to your privilege, including details under the Groups tab by clicking that page's Edit button. When you have finished, click the General Info tab, and then click the Save button to save your changes.
