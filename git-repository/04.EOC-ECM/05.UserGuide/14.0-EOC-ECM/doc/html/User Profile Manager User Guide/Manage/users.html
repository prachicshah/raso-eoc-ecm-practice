Manage Users

You can manage users by performing the following actions:

  • Search for a user
  • Add a new user
  • Change a user's availability status
  • Delete a user
  • Edit a user

Search for a User

To search for a user, perform these steps:

  1. From the menu bar, click Manage > Users to launch the Search User page.



  2. To search for a user, do one of the following actions:
    • Enter your search criteria in these fields, and then click the Search button:

      Field Description
      User ID

      Enter the user's ID.
      Name

      Enter the user's name.
      Email

      Enter the user's e-mail address.
      Active Click this field's drop-down menu and select whether this user is active (True) or not (False).
      Available Click this field's drop-down menu and select whether this user is availble for task assignment (True) or not (False).

    • Click the Search button without entering any information in the fields.

  3. A list of users appears in the search results area.

General Search

You can conduct a general search by hiding many of the search fields. To use the basic search, do the following:

  1. From the Search User page, click the Collapse button.



  2. The Search User page only contains a general search field. Enter your search criteria, and then click the Search button.

Add a New User

You can click the New button from the Search User page to launch the New User Profile page.

See Create a user for steps on how to create a new user profile.

Change a User's Availability Status

By default, when you create a user, that user's availability status is set to Available. This status means that the user can be assigned tasks.

To change a user's availability status, complete these steps:

  1. From the Search User page's search results, select the checkbox next to the User ID, and then click the Set Unavailable button.



  2. The search results refresh and a confirmation message appears, indicating that the user profile has been successfully updated. The selected user's Available status has changed to unavailable.

Delete a User

To delete a user, complete these steps:

  1. From the Search User page's search results, select the checkbox next to the Group Name that you want to delete, and then click the Delete button.



  2. A confirmation dialog appears, prompting you to confirm whether you want to delete the selected user. Click the Yes button to continue.
  3. The page refreshes and your selected user no longer appears in the search results.

Edit a User

To edit a user, complete these steps:

  1. From the Search User page's search results, double-click the user that you want to edit.
  2. Click the Edit button.



  3. Make any changes to your group, including details under the Groups tab by clicking that page's Edit button. When you have finished, click the General Info tab, and then click the Save button to save your changes.
