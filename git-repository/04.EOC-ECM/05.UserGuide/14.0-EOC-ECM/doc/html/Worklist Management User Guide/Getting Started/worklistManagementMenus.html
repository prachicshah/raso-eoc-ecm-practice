Worklist Management Menu Options

The Worklist Management contains the following menu options:

  • Create
  • Manage
  • View

Create Menu

By default, the Create menu does not contain any options. If your metadata application calls for creating an order, you can add this option to this menu.

Manage Menu

The Manage menu allows you to perform the following actions:

Menu Item Function
Worklist Manager Displays a list of pending worklist items that can require user action.
Worklist Users Provides a list of active users.
Worklist Participants Displays a list of participants with pending tasks.
Worklist Archive View a list of worklist items that have been completed.

View Menu

The View menu contains the Event Log, which Displays a list of messages logged by the Worklist Management application.
