Manage

The Manage page contains the following actions that you can perform:

  • Worklist Manager
    This pages displays a list of pending worklist items that can require user action.

  • Worklist Users
    This page provides a list of active users.

  • Worklist Participants
    This page shows a list of participants with pending tasks.

  • Worklist Archive
    This page allows you to view a history of worklist tasks in the system.

Note: To view the Manage menu, ensure that your user ID has the following privileges set using the Worklist Management application:

  • Process Manager Administrator
  • Runtime Administrator
