Worklist Archive

The Worklist Archive report displays tasks that are no longer pending and are open for work. This archive contains a record for every completed or closed action.

To access the worklist archive, complete these steps:

  1. From the menu bar, click Manage > Worklist Archive to launch the Worklist Archive page.



  2. On the Worklist Archive page, specify your search criteria. The following fields are available:

    Field Description
    Created Date

    This field allows you to specify the time and date when the tasks were created. There are two fields that you can specify:

    • From field
      Click the Calendar icon and select the date from which you want to search for tasks. You can also specify the time by using the Hour, Minute, and Second drop-down fields.
    • To field
      Click the Calendar icon and select the date to which you want to search for tasks. You can also specify the time by using the Hour, Minute, and Second drop-down fields.

    To search by either a before or after date, specify the corresponding From or To field, and leave the other field blank.
    Completed Date

    This field allows you to specify the time and date when the tasks were completed. There are two fields that you can specify:

    • From field
      Click the Calendar icon and select the date from which you want to search for tasks. You can also specify the time by using the Hour, Minute, and Second drop-down fields.
    • To field
      Click the Calendar icon and select the date to which you want to search for tasks. You can also specify the time by using the Hour, Minute, and Second drop-down fields.

    To search by either a before or after date, specify the corresponding From or To field, and leave the other field blank.

  3. When you have finished entering your search criteria, click the Search button to view your search results.



    The Worklist Archive search results contain the following fields:

    Field Description
    Due Date The field contains date and time when tasks had completed.
    Participant type This field denotes the participant type assigned to the tasks.
    User ID This field indicates the user assigned to the tasks.
    Task This field represents the task identifier.
    Details This field contains task details.
    Created Date

    This field contains the time and date when task creation occurred.

    Assigned Date

    This field denotes the time and date when tasks were assigned to a user.

    Start Date This field represents the date and time when tasks were started.

  4. To view and event's specific details, double-click the task that you want to view from the search results. The Worklist Archive Details page launches, with details pertaining to the selected task.
