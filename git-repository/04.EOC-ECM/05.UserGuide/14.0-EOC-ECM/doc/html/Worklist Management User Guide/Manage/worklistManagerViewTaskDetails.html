View Task Details

To view task details, do the following:

  1. From the Worklist Manager search results, click the Details icon next to the task that you want to view its details.



  2. The Task Details screen appears.



    The screen contains the following fields:

    Field Description
    Priority This field represents the task's priority.
    User ID This field denotes the ID of the user assigned to the worklist task.
    Due Date This field indicates the date and time when the task is due.
    Sender ID This field contains the ID of the sender process.
    Order Item ID This field denotes the ID of the order item.
    Order ID This field represents the ID of the order.
    Participant This field indicates the participant assigned to the task.
    Type This field denotes the task type.
    Task This field contains the task's description.
    Create Date This field indicates the date and time when the task was created.
    Assigned Date This field denotes the date and time when the task was assigned.
    Start Date This field represents the date and time when the task was started.
    Details This field contains details pertains to the task.

  3. When you have finished viewing the task, click the Back button in the right corner to return to the Worklist Manager.
