View the Change History of a Task

The Task History dialog displays a log of changes that have been made to the selected task. To access the History dialog, complete these steps:

  1. From the Worklist Manager search results, select the checkbox next to the task that you want, and then click the History button.



  2. The History Task dialog appears, which allows you to review all changes made to the task.



    The dialog contains the following fields:

    Field Description
    Action

    This field contains a brief description of the change (for example, creating the task).
    Performed By This field indicates the name of the user who made changes to the task.
    Changed On This field denotes the date and time when the change was made.
    User ID This field indicates the new task's user ID value.
    Due Date This field contains the task's due date.
    Priority This field represents the task's priority.
    Reason This field contains an explanation for the change.

  3. When you have finished reviewing the task history, click the Close button in the top right corner to close the dialog and return to the Worklist Manager.
