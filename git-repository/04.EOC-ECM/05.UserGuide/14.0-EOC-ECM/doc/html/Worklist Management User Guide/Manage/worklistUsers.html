Worklist Users

The Worklist Users report displays a list of active users. Based on configured processes, the Worklist Users report displays a list of all user tasks that are assigned to the current user. The worklist user interface is instantiated using the BaseWorklistFinder (Worklist) found in Velocity Studio's Library tab, under System > cwf_pm (Process Management Product) > Finders > Document Finders.

There are two ways to access the Worklist Users report:

  • By clicking Manage > Worklist Users from the menu bar
  • When reassigning or delegating a task to another user either from the Worklist Manager or the User Worklist

When accessed as a standalone finder, the Worklist Users report lists all users in the system. When you invoke Worklist Users from the context of reassigning a task’s owner, only those users who have the proper privilege for the particular tasks are listed.

To access the Worklist Users report, complete these steps:

  1. From the menu bar, click Manage > Worklist Users.
  2. The Worklist Users screen appears. Click the Search button to view a list of worklist users.



    You can also specify your search criteria in the fields provided and then click the Search button to view your results.

    The Worklist Users search results contain the following fields:

    Field Description
    User ID This field contains the ID of the user.
    Available This field indicates whether the user is available for task assignment.
    Active This field denotes whether the user ID is active.
    Tasks This field represents the number of tasks assigned to the user.
    Overdue This field contains the number of overdue tasks assigned to the user.
    Alerts This field indicates the number of alerts assigned to the user.


From the Worklist Users search results, you can perform the following tasks:

  • View user summary details
  • View user profile details
  • View a list of tasks assigned to a user
  • Set a user's availability for additional tasks

View User Summary Details

To view user summary details, do the following:

  1. From the Worklist Manager search results, click the Details icon next to the user that you want to view its details.



  2. The User Summary Details page appears.



    The page contains the following fields:

    Field Description
    User ID This field contains the ID of the user.
    Tasks This field represents the number of tasks assigned to the user.
    Overdue Tasks This field contains the number of overdue tasks assigned to the user.
    Alerts This field indicates the number of alerts assigned to the user.
    Available This field indicates whether the user is available for task assignment.
    Active This field denotes whether the user ID is active.

  3. When you have finished viewing the task, click the Back button in the right corner to return to the Worklist Users page.

View User Profile Details

To view specific information about a user, follow these steps:

  1. From the Worklist Users search results, select the checkbox next to the User ID that you want, and then click the User Profile button.



  2. The User Summary Details page appears.



    The User Profile Details page contains the following fields:

    Field Description
    User ID This field denotes the ID of the selected worklist user.
    Name This field represents the name of the selected user.
    Active This field indicates whether the selected user is active or not.
    Available This field indicates whether the selected user is available or not.
    Minimum Effort This field contains the smallest number of tasks that the selected user can have assigned.
    Maximum Effort This field denotes the largest number of tasks that the selected user can have assigned.
    E-mail This field represents the e-mail address of the selected user.
    Org Chart Position This field indicates the organizational chart to which the selected user belongs.

  3. When you have finished viewing the user profile details, click the Back button in the right corner to return to the Worklist Users page.

View a List of Tasks Assigned to a User

To view a list of tasks assigned to a user, follow these steps:

  1. From the Worklist Users search results, select the checkbox next to the User ID that you want, and then click the Tasks button.



  2. The User Tasks dialog appears, showing a list of tasks that are assigned to the selected user.



    The User Tasks contains the following fields:

    Field Description
    Due Date This field denotes the date and time when the worklist task is due.
    Task This field contains the description of the task.
    Details This field indicates the order details for the task.
    Assigned Date This field represents the date and time when the task was assigned.
    Started Date This field denotes the date and time when the task was started.

  3. When you have finished viewing the list of tasks assigned to the selected user, click the Close button in the right corner to return to the Worklist Users page.

Set a User's Availability for Additional Tasks

To set a user's availability for additional worklist tasks, complete these steps:

  1. From the Worklist Users search results, select the checkbox next to the User ID that is set to Available, and then click the Unavailable button.



  2. A confirmation message appears on the page, indicating that the selected user is unavailable for task assignment.



  3. Similarly, to set a user as available for additional tasks, select the checkbox next to the User ID that is set to Unavailable, and then click the Available button.
