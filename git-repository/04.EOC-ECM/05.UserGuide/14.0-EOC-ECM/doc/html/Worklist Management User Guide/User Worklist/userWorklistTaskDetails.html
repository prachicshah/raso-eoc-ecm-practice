Display Worklist Task Details

You can display the details for a worklist task by doing one of the following actions:

  • Double-click the task that you want from the My Tasks search results
  • Click the Details icon located at the beginning of the row, next to the Due Date field, of the task that you want to view its details.

The worklist user interface has a Tasks drop-down menu that contains actions that correspond to data displayed in the Worklist.

The Task Details page contains the following fields:

Field Description
Priority

This field represents the task's priority, which can be one of the following:

  • High
  • Low
  • Normal

Task priority is set when the task is created and ranges between 1 and 15. A priority of 1 is High, priority of 15 is Low, and everything in-between is Normal.
User ID This field denotes the ID of the user assigned to the worklist task.
Due Date This field indicates the date and time when the task is expected to be completed. This date is computed based on task participantâ€™s duration.
Sender ID This field contains the ID of the sender process that created the task.
Order ID This field represents the ID of the task's order.
Order Item ID This field denotes the ID of the task's order item.
Participant This field indicates the participant that created the task.
Type

This field denotes the task type, which can be one of the following:

  • Task
  • Alert
  • Taken task
Operation This field contains the participant operation that generated the task.
Create Date This field indicates the date and time when the task was created.
Assigned Date This field denotes the date and time when the task was assigned to the current user.
Start Date This field represents the date and time when the task was started. An empty start date means that a task has not started and there was no work performed on the task yet.
Details This field contains details pertains to the task (that is, the task's order visual key)..
