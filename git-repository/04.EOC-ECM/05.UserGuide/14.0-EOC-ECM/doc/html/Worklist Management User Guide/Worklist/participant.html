Participants

The Participants report displays a list of participants with pending tasks. A user participant represents a person who performs manual tasks within a process.

To access the Participants report, complete these steps:

  1. From the menu bar, click Worklist > Participants.
  2. The Worklist Users screen appears, which allows you to view a list of participants with pending tasks. To refresh the list, click the Search button.



    The Participants report contains the following fields:

    Field Description
    Participant Name of participant.
    Tasks Number of worklist items assigned to the participant.
    Overdue Number of overdue worklist items assigned to the participant.
    Unassigned Number of worklist items assigned to the participant, but not assigned to any users.
    Alerts Number of alerts assigned to the participant.

View a List of Users Associated with a Participant

To view a list of users that are associated with a participant, select a Participant from the Participants report and then click the Users button. The Worklist Users dialog appears, showing a list of users that are associated with the selected participant.

View a List of Tasks Associated with a Participant

To view a list of tasks that are associated with a participant, select a Participant from the Participants report and then click the Tasks button. The Worklist Manager dialog appears, showing a list of tasks that are associated with the selected participant.
