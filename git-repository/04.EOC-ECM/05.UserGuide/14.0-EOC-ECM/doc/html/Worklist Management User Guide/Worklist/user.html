Users

The Worklist Users report displays a list of active users. Based on configured processes, the Worklist Users report displays a list of all user tasks that are assigned to the current user. The worklist user interface is instantiated using the BaseWorklistFinder (Worklist) found in Velocity Studio's Library tab, under System > cwf_pm (Process Management Product) > Finders > Document Finders.

To access the Worklist Users report, complete these steps:

  1. From the menu bar, click Worklist > Users.
  2. The Worklist Users screen appears. Click the Search button to view a list of worklist users.


    You can also specify your search criteria in the fields provided and then click the Search button to view your results.

    The Worklist Users search results contain the following fields:

    Field Description
    Av Indicates whether the user is available.
    User ID ID of the user.
    Tasks Number of tasks assigned to the user.
    Overdue Number of overdue tasks assigned to the user.
    Alerts Number of alerts assigned to the user.

Note: The top two records of the Worklist Users report results display a summary that is unaffected by the search criteria entered. The summary always displays the total number of assigned or unassigned tasks after the <assigned> and <unassigned> user IDs.

User Profile

You can display specific information about a user by selecting a User ID from your Worklist Users search results and then clicking the User Profile button to launch the My User Profile dialog.

The General tab contains the following fields:

Field Description
ID ID of the selected worklist user.
Name Name of the selected worklist user.
E-mail E-mail address of the selected worklist user.
Active Indicates whether the selected worklist user is active or not.
Available Indicates whether the selected worklist user is available or not.
External User Indicates if the worklist user uses external systems.
Minimum Efforts Smallest number of tasks that the selected worklist user can have assigned.
Maximum Efforts Largest number of tasks that the selected worklist user can have assigned.
Current Efforts Number of tasks assigned to the user.
Org Chart (Reference) Indicates the organizational chart to which the selected user belongs.
Employee ID Contains the user's employee ID.

View a Lists of Tasks Assigned to a User

To view a list of assigned tasks that a user has, highlight the User ID that you want and click the Tasks button. The User Worklist dialog appears with all assigned tasks for the selected user.

The User Worklist dialog contains the following fields:

Field Description
Due Date and time when the worklist item is due.
Task Description of the worklist item.
Details Order details for the worklist item.
Assigned Date and time when the worklist item was assigned.
Started Date and time when the worklist item was started.

Set a User as Available or Unavailable for Additional Tasks

To set a user as available for additional worklist tasks, highlight the User ID that you want from the Worklist User results and click the Available button. Similarly, to set a user as unavailable for additional tasks, highlight the User ID that you want from the Worklist User results and click the Unavailable button.
