Worklist Manager

The Worklist Manager report displays a list of pending worklist items that can require user action. By default, the top pane of the Worklist provides end users with options to filter a returned list of user tasks.

To access this report, complete the steps:

  1. From the menu bar, click Worklist > Worklist Manager.
  2. The Worklist Manager screen appears. Click the Search button to view a list of worklist users.


    You can also specify your search criteria in the fields provided and then click the Search button to view your results.

    The Worklist Manager search results contain the following fields:

    Field Description
    Due Date and time when the worklist item is due.
    Participant Participant assigned to the worklist item.
    User ID of the user assigned to the worklist item.
    Task Description of the worklist item.
    Details Order details for the worklist item.
    Created Date and time when the worklist item was created.
    Assigned Date and time when the worklist item was assigned.
    Started Date and time when the worklist item was started.

    Note: The first column (that is, before Due) represents the priority of the task.

For more information about user tasks, see User Worklist. The Worklist Manager can be configured to display in the Administration application or in an application provided that you have the correct permissions.

Task Menu

The Task dialog displays the order item associated with the selected task. To access the Task dialog, do the following:

  1. From the Worklist Manager results, select the task that you want and then click the Task button.
  2. The Task dialog appears, which displays both the order and order item associated with the selected task.

Order

The Order dialog displays the order associated with the selected task. To access the Order dialog, complete these steps:

  1. From the Worklist Manager results, select the task that you want and then click the Order button.
  2. The Order dialog appears, which displays order details that are associated with the selected task.



  3. You can make any pertainent changes to your order. When you have finished making your changes, click the Save button.
  4. Clicking the Order button allows you to perform the following actions:
    • Select the Validate option to determine whether your order is valid. A dialog appears, confirming whether your order is valid. Click the OK button to return to the Order dialog.
    • Select the Save All option to save all changes that you have made.
    • Select the Tasks option to open the Task dialog, which allows you to search and select tasks for the selected order.
    • Select the Task History option to open the Worklist History dialog and view a log of changes that have been made to the selected order.

Assign

The Assign dialog displays a list of users that can be associated with the selected task. To access the Assign dialog, complete these steps:

  1. From the Worklist Manager results, select the task that you want and then click the Assign button.
  2. The Assign dialog appears, which displays a list of user IDs that you can assign the task to. Select the User ID to assign your task to and click the Select button.

    3.

    You can also specify your search criteria in the fields provided and then click the Search button to view your results.

    The Assign search results contain the following fields:

    Field Description
    Av Indicates whether the selected worklist user is available.
    User ID Denotes the ID of the selected worklist user.
    Tasks Displays the number of tasks that the worklist user has.
    Overdue Shows the number of overdue tasks that the worklist user has.
    Alerts Displays the number of alerts that the worklist user has.

  3. The Worklist Manager appears, which shows your task assigned to the User ID that you have specified.

Note: Selecting multiple work items belonging to different participants and then clicking the Assign button is not supported. 

Modify

The Modify Task dialog allows you to change the due date and priority of the selected task. To access the Modify Task dialog, complete these steps:

  1. From the Worklist Manager results, select the task that you want to change and then click the Modify button.
  2. The Modify Task dialog appears, which allows you to change the Due date and Priority fields. Make any pertinent changes and then click the Apply Changes button.



    The following table describes the fields that are available from the Modify Task dialog:

    Field Description
    Due Indicate the date and time that the task is due.
    Priority Click the drop-down menu and select from one of the priority settings.

  3. The Worklist Manager reappears. You can verify that your changes have been made by locating your task from the Worklist Manager and reviewing the task's details.

History

The Task History dialog displays a log of changes that have been made to the selected task. To access the History dialog, complete these steps:

  1. From the Worklist Manager results, select the task that you want and then click the History button.
  2. The History Task dialog appears, which allows you to review all changes made to the task.



    The dialog contains the following fields:

    Field Description
    Changed by Name of the user who made changes to the task.
    Changed on Date and time when the change was made.
    User ID New task user ID value.
    Due The task's due date.
    Priority The task's priority.
    Operation Brief description of the change.

  3. Click the Search button to refresh the task history. Otherwise, click the Close button in the top right corner to close the dialog and return to the Worklist Manager.
