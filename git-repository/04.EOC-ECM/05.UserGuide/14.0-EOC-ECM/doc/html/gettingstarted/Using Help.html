Using Help

The help documentation may be accessed in two ways:

  • As standalone HTML
  • Within Velocity Studio

Standalone HTML

The help documentation as standalone HTML is accessible through \Disk1\doc\index.html of the product installation.

Note: Your browser must be JavaScript-enabled.

Within Velocity Studio

Once you have installed the product, the help documentation is accessible within Velocity Studio by clicking Help > Help Contents from the menu bar. The help documentation is powered by Eclipse Help system, which supports documentation searching, breadcrumbs, synchronization between the table of contents and contents panes, documentation printing, and other capabilities.

Browse topics in the Contents frame () on the left. Click a topic to have it displayed. Use the Back and Forward buttons to navigate within the history of viewed topics.

Synchronizing

Clicking the Show in Table of Contents button () will select that topic in the navigation tree. The Link with Contents button () keeps the navigation tree synchronized to the current topic.

Enabling Scripts

The Velocity Studio Help system requires running JavaScript in your browser. In the event that your browser has disabled JavaScript, a basic version of the Help system is presented, with the URL parameter noscript=1 auto-appended.

If you have subsequently allowed your browser to run scripts (for example, in Internet Explorer, choose Allow Blocked Content... from the Information Bar that is displayed), you can enable the full version to run again by removing the noscript=1 parameter from the URL.
