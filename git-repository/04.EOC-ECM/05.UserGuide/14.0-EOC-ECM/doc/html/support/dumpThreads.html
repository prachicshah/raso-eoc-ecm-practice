Use the Log File to Provide Dump Details

You may encounter problems with process threads, process engines (PEs), and more. Often times, error dumps occur, which are captured in an error log file. The following is an example of locating a thread error and how to capture its details from the log file, which can provide the support team with valuable information to troubleshoot your problem.

  1. In Windows Explorer, navigate to <product_installation_folder>\designer\env\log to view a list of Velocity Studio log files.



  2. Double-click the log file that you want to view. The following is a thread error in the log.



  3. Highlight the error and details that follow it. Proceed to copy the information by pressing the Ctrl + C keys, and do one of the following actions:
    • Paste (press the Ctrl + V keys) the information in your e-mail to the support team.
    • In a text editor, such as Wordpad, open a new document, paste the information, and then save the file. You can then send your file as an attachment in your e-mail to the support team.
