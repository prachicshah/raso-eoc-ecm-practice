Error Codes

A list of error messages, a description of each message, and how to resolve the error can be found in Velocity Studio by clicking the Resource tab.

The following example shows a list of error messages for User Resources. If you know the error code ID, you can perform a search:

  1. Click the Search by ID button and enter your error ID in the ID field.



  2. Click the OK button to conduct your search.

Notes:

  • The screenshot in step 1 contains the ID and Message columns, as the Language is set to en-xx (English). If you choose a different Language for this field, a third column, Default Message, appears to the right of the Message column.
