Find and Extract a Stack Trace from an Error Message

A support team representative may ask that you locate the error message that you are experiencing from the event log in the System Administration application. The representative may also ask that you find and extract a stack trace from the error message. Perform these steps:

  1. Open your project in Velocity Studio and click Runtime > Run from the menu bar.

    Note: You must be connected to your database and have selected the corresponding UI configuration node. 
  2. In your Web browser, log on to the System Administration application by entering http://localhost:8080/cwf/systemAdministration as the URL.
  3. Enter upadmin as both your Username and Password, and then press the Enter key.
  4. In the System Administration application, click View > Event Log from the menu bar.
  5. In the Event Log, click the Search icon at the bottom of the screen to display a list of events. Locate the error by double-clicking the row that you want.



  6. Details about the event appear.



  7. Highlight the information in the Description field. Proceed to copy the information by pressing the Ctrl + C keys, and do one of the following actions:
    • Paste (press the Ctrl + V keys) the information in your e-mail to the support team.
    • In a text editor, such as Wordpad, open a new document, paste the information, and then save the file.
  8. Highlight the information in the Stack Trace field, copy the information, and paste the information in your e-mail or text file as in the previous step. You can then send your file as an attachment in your e-mail to the support team.
