Export Your Database

The support team may need to duplicate the problem you are experiencing by using a copy of your database. To export your database, you can use either Oracle's exp command (for Oracle 10g and earlier versions) or expdp (for Oracle 10g and higher).

Note: You can only use the expdp command if you are using Oracle 11g. The exp command has been deprecated.

Examples of exporting a database with some parameters for the exp and expdp commands are as follows:

exp system/password@mydb file=c:\exportdmp\exp_full_30Nov11.dmp full=y log=c:\exportdmp\30Nov11.log

expdp system/password directory=dir1 dumpfile=exp_dbase_30Nov11.dmp content=metadata_only

The exp and expdp command files appear in your $ORACLE_HOME/bin directory. You can run them interactively, using command line parameters, or by using parameter files. These parameters can be listed by running exp help=yes or expdp help=yes from the command line.

For more information about the exp and expdp commands, consult Oracle's user documentation for your specific version.
