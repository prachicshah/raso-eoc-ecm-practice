Diagnostic and Support Guide

This guide describes how to provide relevant information about your problem while using the product to create a support ticket with our support team. These steps help the support team diagnose and resolve your problem quickly and thoroughly. This guide contains the following sections:

Find and Extract a Stack Trace from an Error Message
This section describes how to locate the error message that you are experiencing from the event log in the System Administration application, followed by how to find and extract a stack trace from the error message.

Use the Log File to Provide Dump Details
You may encounter problems with process threads, process engines (PEs), and more. You can locate error dumps in a log file, which you can send to the support team for analysis.

Export Configuration
Use the Configuration application to export your configuration for our support team to analyze.

Export Your Database
This section describes the commands you can use to export your database, depending on your Oracle version. Exporting your database allows the support team to duplicate and troubleshoot your problem.

Interface Statistics
This section indicates how you can enable statistics for a given client or listener port, and then view the statistical data in either the System Administration application or through JMX.

Error Codes
This section indicates how you can use Velocity Studio's Resource tab to view a list of error messages and a description of each message.

Contact Support
You can contact support by telephone or by e-mail. This section describes support procedures during business and off-normal hours.
