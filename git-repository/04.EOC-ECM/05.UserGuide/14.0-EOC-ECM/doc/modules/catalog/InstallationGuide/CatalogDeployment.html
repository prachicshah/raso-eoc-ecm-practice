Catalog Deployment

The Catalog Management application is packaged in a WAR file that can be deployed on a catalog server. The cwf.war file version for catalog can be deployed standalone without Velocity Studio. The catalog WAR file is located in modules\catalog\Deployment directory of your installation disk or in folder the product is installed. The Deployment and Operations Guide > Deploy in J2EE Application Server section contains information on deploying the catalog WAR file using JBoss, Oracle WebLogic, and IBM WebSphere.
