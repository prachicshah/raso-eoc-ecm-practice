Installation through Velocity Studio

This guide describes how to install the Catalog Management application through Velocity Studio. The standard catalog application is available through the Velocity Studio library files. These files can be extended to customize the Catalog Management application. It is written for individuals with technical background who are familiar with Windows or UNIX operating systems. Some knowledge of related technologies such as database administration is assumed.

The following steps are required for the successful installation of Catalog Management application:

  1. Prerequisites
  2. Configuration - Catalog Parameters
  3. Velocity Studio Configuration
  4. Create the Catalog tables
  5. Assign Catalog Privileges

Note: If you have a valid Product Lifecycle Designer (PLD) license, see Product Lifecycle Designer - Velocity Studio Configuration for details on adding a PLD library file to your project.
