Introduction

The Catalog Designer allows a user to define products and services and their logical and physical components, eligibility and availability rules, pricing, and any offers on these products.

This guide describes how to install the Catalog Designer through Velocity Studio, through SUF, and through a standalone WAR file deployment method. Velocity Studio allows for the customization of the standard catalog application by extending the catalog library files. The catalog WAR file allows for a standard standalone Catalog application deployment without the need for installing Velocity Studio. The installation methods are described in the following documents:

  • Installation through Velocity Studio
  • Installation through System Upgrade Framework (SUF)
  • Installation through WAR file deployment
