Product Lifecycle Designer - Velocity Studio Configuration

To configure Product Lifecycle Designer (PLD), you must do the following:

  • Obtain a valid PLD license. For more information, see the product documentation's Installation User Guide > Appendix C - License Files Explained.
  • Update the Velocity Studio Preferences to reflect the catalog Node created in the configuration application
  • Add catalog library files

Add PLD Library File

The PLD library file, serviceOrchestrationFramework.jar, contains the metadata that enables PLD to run. This file must be added to Velocity Studio's library. Here are the steps to complete this process:

  1. Open Velocity Studio and click File > Open Project from the menu bar.
  2. Click the metadata root node. In the right pane, click the Library tab.
  3. Click the Add button.
  4. Add the serviceOrchestrationFramework.jar file, which is located in the <product_installation_folder>\modules folder.
  5. Once the files are added, Save the project metadata, and reload or open the project for the library files to take effect.

Note: For more information, see Add Library files.

Once all the library files have been successfully added, the Library folder contains all the files.
