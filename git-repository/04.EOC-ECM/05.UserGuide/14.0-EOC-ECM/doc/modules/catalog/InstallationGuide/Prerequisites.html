Prerequisites

Before installing Catalog Designer, you must ensure the following prerequisites have been fulfilled.

  • Install the product – see the Installer User Guide
  • Create a database schema – see the Installer User Guide > Standard Installation > Database Initialization
  • Create a project in Velocity Studio – see Velocity Studio User Guide > Velocity Studio Menu Actions > File > New Project
  • Connect to database – see Velocity Studio User Guide > Velocity Studio Menu Actions > Database > Connect to Database
