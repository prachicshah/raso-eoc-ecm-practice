Installation through SUF

ECM installation and configuration through SUF should be performed by users who have good skills in RHEL 6 and SUF.

The target group of the installation instructions includes:

  • System integrators
  • System administrators

Before starting the installation procedure, ensure the following prerequisites are met:

  • Access to all required software; refer to prerequisite sections of the Execution Environment Installation and SUF - ECM Installation.
  • Access to target machines on which the software is to be installed
  • Access to a machine with Management Server and SUF 3.5
  • The network configuration files for the target machines are available on the local machine
  • Access to SUF documentation
  • Access to Execution Environment on eForge.

If there are any issues with the installation, refer to the installation troubleshooting section.
