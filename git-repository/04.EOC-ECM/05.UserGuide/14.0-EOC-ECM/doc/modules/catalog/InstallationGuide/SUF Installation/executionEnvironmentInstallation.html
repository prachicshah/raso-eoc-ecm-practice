Execution Environment Installation

This procedure installs Execution Environment, including OS, LCF, Execution Environment Base and the JBoss Application Server on the target machine. All steps in this procedure refer to chapters in the Execution Environment Installation Guide.

Prerequisites

  • Access to the SUF Server
  • Access to the Execution Environment eForge site where the required software and Execution Environment Installation Guide resides. The following is a list of the required software:
    • rhel-server-6.4-x86_64-dvd.iso
    • rhel-server-supplementary-6.4-x86_64-dvd.iso
    • install-redhatVirtual-x.x.x.tar.gz
    • jboss_eap_61-x.x.x.tar.gz
    • lcf_23-x.x.x.tar.gz
    • ngee_base-x.x.x.tar.gz
    • shared-resources-x.x.x.tar.gz.

Installation Procedure

Install Execution Environment OS and 3PP and follow the instructions of the Execution Environment Installation Guide sections listed below:
  1. Installation on Virtual Server
  2. Prepare SUF for LCF Component Install from SUF Server
  3. Installation of LCF and SUF Components through SUF Playlist
  4. Installing Execution Environment Base
  5. Installation of JBoss Application Container.
