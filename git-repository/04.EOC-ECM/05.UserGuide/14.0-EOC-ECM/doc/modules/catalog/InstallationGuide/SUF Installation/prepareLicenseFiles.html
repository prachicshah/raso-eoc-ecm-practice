Prepare License Files

The AVM installation requires a license for the SUF server and a license for each ECM application server being installed.

Upload the AVM license for the SUF server

Prerequisites: The AVM license file is available with file name {license}.lic.

  1. Create an avm‚Äù directory: #mkdir /var/lib/sufstorage/repository/avm
  2. Upload the AVM license file to the SUF server location /var/lib/sufstorage/repository/avm.

AVM License File for Application Server

Upload the AVM license for the ECM application server(s).

Prerequisites: The AVM license file is available with file name {hostname}_avm_license.lic.

  1. Upload the AVM license file to a user specified folder (for example, /root/lics) on the SUF server.

Note: SUF will copy the AVM license file from the user specified license folder to each JBoss (ECM application) server. If this step fails (for example, the AVM license file is missing for a particular JBoss (ECM application) server), the AVM license file can be copied manually at a later time, thus allowing to ability to proceed to the next step.
