Upgrade Database and Change the Node

To make the database compatible with the System Configuration settings, you need to upgrade the database. The following steps describe how to upgrade the database and change the node:

  1. In Velocity Studio, click the Database > Upgrade Database from the menu bar.
  2. The Upgrade SQL dialog appears; specify the directory and the name of the generated file.
  3. Click the Save button to create the file.
  4. Run the upgrade SQL file using same database schema used for database initialization in the connect database step.
  5. Click Runtime > Select Application from the menu bar.
  6. Click the Node field and select the node configured for catalog (for example, Catalog\PE_UI).

For more information see, Velocity Studio User Guide.
