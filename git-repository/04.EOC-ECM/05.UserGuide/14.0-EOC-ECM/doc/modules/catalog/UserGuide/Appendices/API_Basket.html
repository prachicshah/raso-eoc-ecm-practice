External Services: Catalog Basket API

The Catalog Basket API is to be invoked through Services for an external system to create and maintain Catalog Basket instance. This includes browsing, adding and removing items, pricing, validating, and getting item information.

Note: For more details on the Catalog API's, refer to JavaScript documentation.
