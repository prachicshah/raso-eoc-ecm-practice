External Services: Maintenance API

The maintenance API is to be invoked through Services for an external system to perform maintenance on the Catalog data. This includes adding, changing, and deleting the listed objects and object types. The following is an explanation of maintaince APIs, the parameters, and the description:

AttributeMaintenance

API Name Input Output Description
CreateAttribute

cwt_pcmaintapi.attribute (Attribute)

cwt_pcmaintapi.response (Reponse) This API creates a new catalog attribute object, and saves the record to the database. Input parameter is a datastructure of attribute object with values. Output parameter is a datastructure that contains the response upon a successful operation or the fault in case of failure.
DeleteAttribute cwt_pcmaintapi.attributeSearchDS (Attribute Search) cwt_pcmaintapi.response (Reponse) This API deletes the catalog attribute record. Input parameter is a datastructure with values to find the catalog attribute. Output parameter is a datastructure that contains the response upon a successful delete operation, or containing the fault in case of failures (for example, data not found or violation integrity constraints).
UpdateAttribute cwt_pcmaintapi.attribute (Attribute) cwt_pcmaintapi.response (Reponse) This API updates the catalog attribute record.  Input is a datastructure with values to find and update the catalog attribute. Output is a datastructure containing the response in case of successful update operation, or containing the fault in case of failures (for example, data not found or violation integrity constraints).
ReadAttribute cwt_pcmaintapi.attributeSearchDS (Attribute Search) cwt_pcmaintapi.attribute (Attribute) This API read the catalog attribute record.  Input parameter is a datastructure to find the catalog attribute using the attribute key. Output parameter is a datastructure that contains the attribute record value, or contains the fault if the data is not found.
SearchAttribute cwt_pcmaintapi.attributeSearchDS (Attribute Search) cwt_pcmaintapi.attributeList (Attribute List) This API returns a list of catalog attribute records matching the search criteria provided in the fields of datastructure as input parameter. This API returns null, if the matching result is not found.
