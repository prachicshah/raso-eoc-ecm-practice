API and Object Reference

The external Catalog contains APIs to assist users in configuring data. The following external interfaces and objects are available:

  • Catalog Basket: The Catalog Basket API is to be invoked through Services for an external system to create and maintain Catalog Basket instance. These tasks include browsing, adding and removing items, pricing, validating, and getting item information.
  • Maintenance: The maintenance API is to be invoked through Services for external systems to perform maintenance on the Catalog data. These maintenance tasks include adding, changing, and deleting the listed objects and object types.
