AddItemToBasket SID Interface

The AddItemToBasket interface adds an item to the catalog basket by item code and returns the resulting catalog basket after adding the item. The AddItemToBasket interface is available in the Library files of Velocity Studio, and contains input, output, and fault data structures defined as follows:

Input

The input file contains the following elements and data types defined in Velocity Studio's data dictionary's cwt_pcsid.addRequest data structure:
