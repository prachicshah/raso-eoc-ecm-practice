GetBasket SID Interface

The GetBasket interface validates the catalog basket for errors, and returns a list of validation error codes and messages. The GetBasket interface is available in the Library files of Velocity Studio, and contains input, output, and fault data structures defined as follows:

Input

There is no input file required for the GetBasket interface.

Output

The output file contains the following elements and data types defined in the cwt_psid.basket data structure:

Output DataType Element Notes
basket    

Contains the following elements:

  • basketInstance
    • basketId (string 10)
    • cwCreated (DateTime)
    • cwCreatedBy (string (64))
    • lastUpdateDate (DateTime)
    • updatedBy (string (64))
    • cwOrderID (string (16))
    • state (string (16))
  • basketItemRelations (array)
    • basketItemRelation
      • basketItemIdFrom (string 10)
      • basketItemIdTo (string 10)
      • relationName (string 32)
      • basketId (string 10)
      • cwCreated (DateTime)
      • cwCreatedBy (string (64))
      • lastUpdateDate (DateTime)
      • updatedBy (string (64))
      • basketItemRelationId (string)
  • basketItems (array)
    • basketItem
      • basketItemId (string 10)
      • basketId (string 10)
      • itemcode (string 32)
      • cwCreated (DateTime)
      • cwCreatedBy (string (64))
      • lastUpdateDate (DateTime)
      • updatedBy (string (64))
      • parentbasketitemid (string (10))
      • relationname (string (32))
      • sequenceNo (decimal (4))
      • basketcreationdate (DateTime)
      • requesttime (DateTime)
      • state (string (16))
      • basketItemCharges (array)
        • basketItemCharge
          • basketId (string (10))
          • basketItemId (string 10)
          • chargeTypeCode (string (32))
          • amount (decimal (10.2))
          • sequenceNo (decimal (4))
          • currency (string(3))
          • frequency (string (1))
          • displayOnly (boolean)
          • taxCode (string (32))
          • label (string (128)
          • cwCreated (DateTime)
          • cwCreatedBy (string (64))
          • lastUpdateDate (DateTime)
          • updatedBy (string (64))
          • isDiscount (boolean)
        • dynamicLeaves (array)
          • dynamicLeaf
            • leafName (string)
            • leafValue (string)
