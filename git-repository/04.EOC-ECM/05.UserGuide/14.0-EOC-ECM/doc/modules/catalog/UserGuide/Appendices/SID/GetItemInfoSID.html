GetItemInfo SID Interface

The GetItemInfo interface gets detailed information for a specified catalog item and returns a catalog item in the version determined by the given date. The GetItemInfo interface is available in the Library files of Velocity Studio, and contains input, output, and fault data structures defined as follows:

Input

The input file contains the following elements and data types defined in Velocity Studio's data dictionary's cwt_pcsid.getItemRequest data structure:

Input DataType Element Notes
itemCode String    
date DateTime   Refer to Velocity Studio's documentation for more details on the DateTime format.
language String    

Output

The output file contains the following elements and data types defined in the cwt_psid.item data structure.

Output DataType Element Notes
code String (32)

 cwt_pcdd.catalogCode
type String (8) cwt_common.cwtct_itemType
label String    
startDate Date cwt_pcdd.startDate  
endDate Date cwt_pcdd.endDate  
itemAttributes Array cwt_pcoeapi.itemAttribute XML name is itemAttribute.
itemRelations

Array

 cwt_pcsid.itemRelation

XML name is itemRelation. Contains the following elements:

  • accessoryType (array)
    • codeTableElement (array)
      • code (string 32)
      • label (string 128)
  • relationType (array)
    • codeTableElement (array)
      • code (string 32)
      • label (string 128)
itemRules Array cwt_pcsid.itemRule

XML name is itemRule.

itemGroups Array cwt_pcsid.itemGroup

XML name is itemGroup. Contains the following elements:

  • name (string 32)
  • minCardinality(integer (4))
  • maxCardinality(integer (4))
  • targets (array) (extends from cwt_pcsid.groupTarget)
  • label (string)
itemImages Array cwt_pcsid.itemImage

XML name is itemImage.
itemDescriptions Array cwt_pcsid.itemDescription

XML name is itemDescription. Contains the following elements:

  • name (string (32))
  • description (string (10))
  • type (string (32))
itemItentifiers Array cwt_pcsid.itemItentifier

Contains the following elements:

  • systemId (array)
    • codeTableElement (array)
      • code (string 32)
      • label (string 128)
itemAttachments Array cwt_pcsid.itemAttachment

XML name is itemAttachment. Contains the following elements:

  • mimeType
  • attachmentName
itemCharges Array cwt_pcsid.itemCharge

XML name is itemCharge.
