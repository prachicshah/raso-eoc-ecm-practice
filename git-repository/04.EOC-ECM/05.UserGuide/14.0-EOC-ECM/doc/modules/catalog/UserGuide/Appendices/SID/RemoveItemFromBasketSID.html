RemoveItemFromBasket SID Interface

The RemoveItemFromBasket interface removes an item from the catalog basket by item ID and returns the resulting catalog basket after removing the item. The RemoveItemFromBasket interface is available in the Library files of Velocity Studio, and contains input, output, and fault data structures defined as follows:

Input

The input file contains the following elements and data types defined in Velocity Studio's data dictionary's cwt_pcsid.removeRequest data structure:
