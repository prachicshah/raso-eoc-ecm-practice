ValidateBasket SID Interface

The ValidateBasket interface validates the catalog basket for errors, and returns a list of validation error codes and messages. The ValidateBasket interface is available in the Library files of Velocity Studio, and contains input, output, and fault data structures defined as follows:

Input

The input file contains the following elements and data types defined in Velocity Studio's data dictionary's cwt_pcsid.priceBasketRequest data structure:

Input DataType Element Notes
basket

Contains the following elements:

  • basketInstance
    • basketId (string 10)
    • cwCreated (DateTime)
    • cwCreatedBy (string (64))
    • lastUpdateDate (DateTime)
    • updatedBy (string (64))
    • cwOrderID (string (16))
    • state (string (16))
  • basketItemRelations (array)
    • basketItemRelation
      • basketItemIdFrom (string 10)
      • basketItemIdTo (string 10)
      • relationName (string 32)
      • basketId (string 10)
      • cwCreated (DateTime)
      • cwCreatedBy (string (64))
      • lastUpdateDate (DateTime)
      • updatedBy (string (64))
      • basketItemRelationId (string)
  • basketItems (array)
    • basketItem
      • basketItemId (string 10)
      • basketId (string 10)
      • itemcode (string 32)
      • cwCreated (DateTime)
      • cwCreatedBy (string (64))
      • lastUpdateDate (DateTime)
      • updatedBy (string (64))
      • parentbasketitemid (string (10))
      • relationname (string (32))
      • sequenceNo (decimal (4))
      • basketcreationdate (DateTime)
      • requesttime (DateTime)
      • state (string (16))
      • basketItemCharges (array)
        • basketItemCharge
          • basketId (string (10))
          • basketItemId (string 10)
          • chargeTypeCode (string (32))
          • amount (decimal (10.2))
          • sequenceNo (decimal (4))
          • currency (string(3))
          • frequency (string (1))
          • displayOnly (boolean)
          • taxCode (string (32))
          • label (string (128)
          • cwCreated (DateTime)
          • cwCreatedBy (string (64))
          • lastUpdateDate (DateTime)
          • updatedBy (string (64))
          • isDiscount (boolean)
      • dynamicLeaves (array)
        • dynamicLeaf
          • leafName (string)
          • leafValue (string)
date DateTime
Refer to Velocity Studio's documentation for more details on the DateTime format.
context Array

Contains the following elements:

  • code (string)
  • attributes(array)
    • nameValuePair
      • name (string)
      • value (string)
  • label (string)

Output

The output file contains the following elements and datatypes defined in the cwt_psid.messages data structure.

Output DataType Element Notes
message    

Contains the following elements:

  • message (string/array)
  • itemCode (string/array)

Input and Output File Examples

These input and output files are examples using a SoapUI tool.

InputXML: 
<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/">
   <soapenv:Header/>
   <soapenv:Body>
      <priceBasketRequest>
         <basket>
            <basketInstance>
               <basketId>FGb000</basketId><cwCreated>2011-05-12T00:00:00.000-04:00</cwCreated>
               <cwCreatedBy>?</cwCreatedBy>
               <lastUpdatedDate>2011-05-13T00:00:00.000-04:00</lastUpdatedDate>
               <updatedBy>?</updatedBy>
               <cwOrderId>?</cwOrderId>
               <state>?</state>
            </basketInstance>
         </basket>
         <date>2011-05-12T00:00:00.000-04:00</date>
      </priceBasketRequest>
   </soapenv:Body>
</soapenv:Envelope>
Output XML: 
<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/">
   <soapenv:Body>
      <messages>
         <message>No validation errors</message>
      </messages>
   </soapenv:Body>
</soapenv:Envelope>
