Configure PLD

You can access the Product Lifecycle Designer (PLD) menus using the Quick Start menu on the Ericsson Catalog Manager home page or by clicking the Configuration > Configure PLD option from the menu bar to perform the following actions:

  • Configure Templates
    The Configure Template page allows you to configure the templates that can be used for several project types and different life cycle stages.

  • Configure Stages
    The Configure Stages page allows you to define new stages for projects and participants.

  • Configure Participants
    The Configure Participants page allows you to create new participants.

  • Configure Microflows
    The Configure Microflows page allows you to add, delete, and copy microflows. Each microflow in the product life cycle requires approval for action. That is, for every stage, there must be someone who needs to approve.

  • Request Type Template Configuration
    The Request Workflow Configuration page allows you to associate a request type with a workflow template.
