Template File

The <metadata home>/resources/Excel Template/Template.xml file is used to Import and Export data into the Catalog. This section contains a listing of the fields used by the system for importing and exporting data. The excel template file has the following sheets containing the fields below. This file is stored as an XML Spreadsheet 2003 (*.xml) format. This file can also be obtained by using the Catalog export function. The export file can be used to input your data for Catalog Import.
