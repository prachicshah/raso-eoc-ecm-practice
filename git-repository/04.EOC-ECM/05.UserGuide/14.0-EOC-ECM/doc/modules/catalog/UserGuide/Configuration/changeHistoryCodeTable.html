Change History of a Code Table

This section describes the following:

  • Change History of a Code Table
  • Change History of a Code
  • Show Usage of a Code Table or Code

To view the change history of a version code table, open the Code Tables page. See the Search Code Tables section.

Click on the Details icon (magnifying glass) next to Name of the code table or double-click on the code table row. This opens the code table details dialogue.

Click on the Change History button.

Note: A project must be opened to see the history of a version code table. Non-version code tables do no have a Change History feature.

The Change History page provides information about all changes done for the selected code table along with type of changed operation.

Change History of a Code

To view the change history of a version code, open the Code Details page of a code. Click the Change History button.

The Change History page provides information about all changes done for the selected code along with type of changed operation.

Show Usage of a Code Table or a Code

From the Code Tables page, click the Show Usage button to show the usage of the code table.

From the Code Details, click the Show Usage button to show the usage of the code.
