Search for an Association Type

To search for an association type from all projects available in the Catalog Designer, follow these steps:

  1. From the menu bar, click Designer > Association Types to launch the Association Type Finder page.
  2. Specify your search criteria. The following fields are available:

    Field Description
    Association Type Code This field denotes the code that is used to identify the assocation type.
    Category This field references the association type's category (for example, Attachment, Item, Group, and so on).
    Name This field indicates a unique name for the association.
    Project This field indicates with which project the association type is associated.
    Status This field denotes a list of available statuses for the association (for example, Definition and Active).
    Type This field denotes the association's data type. Click the drop-down menu and select one of the types from the list.

  3. When you have finished entering your search criteria, click the Search button. All associations from all projects appear in the search results.
