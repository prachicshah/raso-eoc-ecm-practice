Debug a Catalog Rule

You can debug a Catalog rule by using the Rhino JavaScript Debugger. You can access this tool in Velocity Studio by clicking Runtime > Debug from the menu bar. See the Velocity Studio User Guide >SVelocity Studio Menu Actions > Runtime > Debug for information on using the Rhino JavaScript Debugger.

Once you have the debugger started, you can insert a breakpoint by doing the following:

  1. Click Search > Scripts from the Rhino JavaScript Debugger's menu bar.
  2. Enter your Catalog rule name in the Enter script name field, and then click the Search button.
  3. Open the Catalog rule that you want. You can insert a breakpoint by clicking the code's line number. A brown dot next to the line number indicates that you have set a breakpoint.
