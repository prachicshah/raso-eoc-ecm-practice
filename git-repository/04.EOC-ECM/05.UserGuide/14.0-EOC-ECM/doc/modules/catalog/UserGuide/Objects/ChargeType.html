Charge Types

The Charge Type is the collection of formulas and price tables, driven by a specific product or category, or family of products. For example, Private Line pricing is different from Wireless pricing. Price Model Charge Types can be associated at different levels in the Product Catalog.

Clicking the Charge Types option from the Catalog Designer menu opens the Charge Types Search. Existing Charge Types, including pricing and discounts, displays.

The relationship for Charge Types, Pricing, and Discounts to Products is a one to many relationship. A product may have many price points, for example different prices for hardware or installations (standard or express). Charge Types uses versions to represent different pricing and discount options that are available.

The Charge Types page has the following buttons that you can click after you select at least one charge type from the search results::

Action Description
Add Opens the Charge Type Detail dialog for creating a new Charge Type.
Delete Deletes the selected Charge Type.
Detail Opens the Charge Type Detail form for editing the selected Charge Type (a row in the search results table must be selected).
Show Usage Performs a search displaying the objects containing the selected Charge Type within the Catalog.
Change History Displays a history for the selected Charge Type.
