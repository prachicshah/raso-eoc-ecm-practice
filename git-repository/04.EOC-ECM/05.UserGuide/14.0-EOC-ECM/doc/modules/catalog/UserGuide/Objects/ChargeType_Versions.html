Charge Type Versions

A Charge Type can contain several versions. Versions define different instances of an association with different life span (start and end date). A Charge Type Version holds pricing or discount information for the Charge Type Items.

After configuring a Charge Type, click the Save button. The Charge Type version appears under the Versions tab.

When you select a charge type from the Versions tab, the following buttons are available:

Action Description
Add Add a new version.
Delete Deletes the selected version.
Copy Create a new version with the same values of the selected version; this copies the Price List in the selected version as well.
Detail View the properties of the selected version.
The Versions section contains the following fields:
Field Description
Name Displays the identifying code for this charge type.
Status Displays the status for this charge type.
Start Date This field identifies the charge type's effective date. It must be less than or equal to the End Date field.
