Component Group

A Component Group is a means to classify Items within an Item. Items of similar type can be clustered together as a Group in a Item to do the following:

  • Form a package (for example, add-on voicemail / call display / call-forwarding package of a VoIP access Product)
  • Provide selection among Components (for example, choose among various VDSL speeds in high-speed Internet Product)
  • Provide restriction in pick-and-choose choices (for example, choose up to a maximum of three channels in a theme pack for Digital TV subscription Product)
