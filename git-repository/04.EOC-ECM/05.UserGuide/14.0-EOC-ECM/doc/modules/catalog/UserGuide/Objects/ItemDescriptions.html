Item Descriptions

To create a new item descriptions, follow these steps:

Note: A project must be opened to create an item descriptions.

  1. Navigate to Designer > Items menu.
  2. Expand the Item tree and select one item to which you want to add the descriptions (for example, Internet Offer).
  3. Select Descriptions from associated item objects in the left pane.

  4. The Description Detail dialog appears in the right pane.
  5. Enter the information in the following fields:

    6. Field Description
    Code This is the code used to identify the item descriptions. This code is not restricted to alphanumeric characters, it can also contain a combination of spaces, hyphens, etc. Strings can be up to 32 characters long.
    Type This is a list of association types of category Item descriptions.
    Start Date The effective date of this descriptions. Refer to Start and End dates for details. Defaults to the project's start date.
    End Date The end date of this descriptions represents the date when this descriptions is no longer effective. Refer to Start and End dates for details.
    Status A list of available Status for this descriptions (for example, Definition and Active).
    Cancel Objects that are extended have this Cancel flag checked. If the flag is checked, then the extended item does not inherit that association.
    Description Enter the description for the Item in this field.
    Project Each object is assigned to a Project. This defaults to the active or opened project when the descriptions is first created.
    Classification Types Select different classification types to identify item under different environments.

  6. Click the Save () button.

  7. The Versions tab appears on the Descriptions Detail dialog. You can create versions of this description.
  8. The saved item descriptions (OfferDescription) appears under the Descriptions object.
