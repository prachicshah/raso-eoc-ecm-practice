Items

The Catalog Items occupy central place among the Catalog objects. All item properties are expressed through associations. As a result, Items and their associations usually account most of the Catalog elements. The items represent the products and services which the company offers. Items support versions, support inheritance where all characteristics of the parent are inherited when an item is extended, and have relations with other catalog objects.

  • Add an Item
    This option allows you to add new items

  • Change History
    This option allows view the change history of the items.

  • Copy an Item
    This option allows you to copy an item.

  • Change End Date
    This option provides you the ability to change the end date of an active an item.

  • Delete an Item
    This option provides you the ability to delete the items.

  • Export Items
    This option allows you to export items in different file formats.

  • Extend Items
    This option allows you to extend an item.

  • Item Palette
    This option allows you to add item associations using drag-and- drop functionality.

  • Search an Item
    This option allows you to perform a search on items

  • Show Usage
    This option allows see the usage of the items

  • View
    This option allows you to create a custom view for the items page.
