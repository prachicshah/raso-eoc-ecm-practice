Create a Project

To create a new project, complete these steps:

  1. From the menu bar, click No Current Project > Manage to launch the Project Search Results page.
  2. Click New to launch the Project page.



  3. Enter your project information in the fields provided. The following fields are available:

    Field Description Rules
    Code The code used to identify this Project. This code is used through the Catalog to associated Catalog changes to a project. Mandatory, Unique
    Status A list of available Status (for example, Definition, Ready, Review, Approved and Active). The default status is Definition. Changing the status to Active results in all Catalog changes under this project to be committed. Mandatory
    Name A unique name for this Project. This field is for internal use when designing your catalog model. This field is not supported by language translation. Mandatory
    Effective Date The Project changes are effective on this date/time once the project is committed (that is, status is Active). The start dates of the revisions must be after this effective date. Mandatory
    Parent Project This field is automatically populated with the Parent Project object. If a parent project does not exist, then this field remains blank. All the child projects must be active before the parent project can be active. Mandatory
    Label The visual Label used for this Context, which is used in the application user interface. This field is a translational string that supports language translations (refer to Velocity Studio's Translation Element for more details). Optional
    Description A description of this Project. Optional
    Owner This is the position of the user who created the project and only users with the same position or higher can access the project data. This value cannot be changed. Refer to Domains for more information. Mandatory
    Error Correction This option allows you to determine whether errors appear in your project. By default, this option is not selected. Optional

  4. Click Save. A confirmation message appears, indicating that your project has been created. Click Project to return to the Project appears in the left pane.
  5. Click the Refresh button to verify that your new project appears in the complete list of projects available.


Note: The Domains tab appears after you have created and opened your project. Domains allow you to grant additional access to your project data. Refer to Domains for more information.
