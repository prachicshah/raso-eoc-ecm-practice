Business Dates Association Category

The itemBusinessDatesVersion is an association category in the Catalog application. This association category serves to store the date's value.

This association category comprises the following details:

  • itemBusinessDatesVersionId
  • itemBusinessDatesCode
  • itemCode
  • datesModelCode
  • catalogObjectType[Item, ItemMSRule]
  • catalogObjectCode
  • datesValueDocId
  • startDate
  • endDate
