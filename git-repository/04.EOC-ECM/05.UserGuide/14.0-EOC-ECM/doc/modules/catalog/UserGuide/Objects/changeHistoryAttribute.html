Change History of an Attribute Type

To view the history of an attribute, follow these steps:

Note: A project must be opened to see the history of an attribute.

  1. From the menu bar, click Designer > Attribute Types.
  2. Search for all the attributes and select the one you want to delete, or search for a particular attribute.
  3. Select the attribute (for example, bandwidth) and click the Change HIstory button.

  4. The Change History page provides information about all changes done to the selected attribute.
