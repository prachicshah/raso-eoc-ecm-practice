Change History of an Info Model

To view the change history of an info model, follow these steps:

Note: A project must be opened to see the history of an info model.

  1. In Catalog Designer application, navigate to Designer > Info Models menu.
  2. The Info Models Search Criteria dialog appears. Search for an info model.
  3. Select an info model from the result list in the Results area.
  4. Click the Change History button located at the bottom of the dialog.

  5. The Change History page provides information about all changes done to the selected info model.
