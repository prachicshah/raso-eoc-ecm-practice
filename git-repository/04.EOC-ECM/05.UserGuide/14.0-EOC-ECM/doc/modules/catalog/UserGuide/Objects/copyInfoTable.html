Copy an Info Table

To copy an info table, follow these steps:

Note: A project must be opened to copy an info table.

  1. On the Info Model Detail dialog, click the Info Tables tab.
  2. The list of info table details appears; select one from the list (for example, 8/14/2013 18).
  3. Click the Copy button.

  4. The Info Table dialog appears with the details of selected info table. You can add new data or change the details in the fields.

  5. Click the Save button on Info Table dialog.
  6. A copy of info table appears in the list (for example, 8/14/2013 16 with an end date).
