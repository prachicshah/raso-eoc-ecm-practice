Delete an Attribute Type

To delete an attribute, follow these steps:

Note: A project must be opened to delete an attribute type.

  1. From the menu bar, click Designer > Attribute Types.
  2. Search for all the attributes and select the one you want to delete, or search for a particular attribute.
  3. Select the attribute (for example, y11) and click the Delete button.

  4. A confirmation dialog appears. Click the Yes button.
  5. The selected attribute is successfully deleted from the project.

Notes:

  • You cannot delete an attribute that has its Status field set to Active.
  • You can select multiple attributes from the search results, and then select the Delete button.
