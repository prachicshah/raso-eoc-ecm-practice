Delete an Info Model

To delete an info model, follow these steps:

Note: A project must be opened to delete an info model.

  1. In Catalog Designer application, navigate to Designer > Info Models menu.
  2. The Info Models Search Criteria dialog appears; search for an info model.
  3. Select an info model from the result list (for example, infomodel_price_disc), in the Results pane.
  4. Click the Delete button located at the bottom of the dialog.

  5. A confirmation dialog appears; select the Yes button.
  6. The selected info model is deleted from the project.

Note: You cannot delete an info model that has its Status field set to Active.
