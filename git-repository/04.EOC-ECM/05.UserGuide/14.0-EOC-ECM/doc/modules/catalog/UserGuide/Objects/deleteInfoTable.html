Delete an Info Table

To delete an info table, complete these steps:

Note: A project must be opened to delete an info table.

  1. On the Info Model Detail dialog, click the Info Tables tab.
  2. The list of info table details appears; select one from the list (for example, 8/14/2013 18)
  3. Click the Delete button.

  4. A confirmation dialog appears; select the Yes button.
  5. The selected info table is deleted from the project.

Note: You cannot delete an info table that has its Status field set to Active.
