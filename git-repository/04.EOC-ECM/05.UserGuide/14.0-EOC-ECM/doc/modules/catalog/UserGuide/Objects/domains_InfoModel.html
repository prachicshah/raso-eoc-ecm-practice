Domains for Info Model

To add domains for info model, complete these steps:

  1. From the menu bar, click Designer > Info Models.
  2. Search for the info model and select one.
  3. Click the Domains tab from Info Model detail page
  4. Click the Add button; the Domain dialog appears on the right side.

  5. Click the Search button next to the Org Chart field, and selcet one org chart to associate with the info model.
  6. Click the Save button.

Delete a Domain

To delete a domain related to info model, complete these steps:

  1. Click Domains tab from Info Model Detail page to display list of added domains.
  2. Select a domain from the list, and click the Delete button.
  3. A confirmation dialog appears; select Yes to delete the selected domain.

View details of a Domain

To view details of a domain related to info model, complete these steps:

  1. Click Domains tab from Info Model Detail page to display list of added domains.
  2. Select a domain from the list, and click the Detail button.
  3. The details of the selected domain appear.
