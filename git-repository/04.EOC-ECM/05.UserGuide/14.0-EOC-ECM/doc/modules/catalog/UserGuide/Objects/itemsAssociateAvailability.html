Associate an Availability Tree

To associate an availability tree to an item, complete these steps:

  1. Open your project and click Commercial Modelling > Items from the Overview page.
  2. From the Item page, perform a search to a list of items, and then double-click the item that you want.
  3. Click the Availability tab, and then click New.
  4. The Availability page appears.



  5. You can specify the following fields:

    Field Description
    Name Enter the name of your availability tree.
    Status This field denotes a list of available statuses for the availability tree (for example, Definition and Active).
    Start Date This field represents the start (effective) date that the availability tree is available. Refer to Start and End dates for details.
    End Date This field indicates the date when the availability tree is no longer effective. Refer to Start and End dates for details.
    Project This field indicates with which project the availability tree is associated.
    Type Indicate the type of operator to use for availability (that is, Include or Exclude).
    Tree Document You can expand this field and specify the tree documents that you want to include in your availability.

  6. Click the Save button. A confirmation appears, indicating that you have successfully saved your information. When you return to the Availability page for your item, the List of Availability section contains your newly created availability.
