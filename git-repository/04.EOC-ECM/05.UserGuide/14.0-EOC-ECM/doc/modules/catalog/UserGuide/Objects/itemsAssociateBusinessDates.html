Associate a Business Date

To associate a business date to an item, complete these steps:

  1. Open your project and click Commercial Modelling > Items from the Overview page.
  2. From the Item page, perform a search to a list of items, and then double-click the item that you want.
  3. Click the Business Dates tab, and then click New.
  4. The New Item Business Date Version dialog appears. Click the Date Model Code field's drop-down menu and select the business date model code that you want. Additional fields appears in this dialog.



  5. You can specify the following fields:

    Field Description
    Status This field denotes a list of available statuses for the business date (for example, Definition and Active).
    Start Date This field represents the start (effective) date that the business date model is available. Refer to Start and End dates for details.
    End Date This field indicates the date when the business date model is no longer effective. Refer to Start and End dates for details.
    Advertising Start Date Specify the date controlling the earliest date that an item may be advertised.
    Start Selling Date Indicate the date controlling the earliest date than an item can be sold. This item cannot be ordered before this date. However, orders may be taken.
    Start Servicing Date Specify the date controlling the earliest requested due date for an item. The earliest indicates the due or installation date for orders.
    Stop Selling Date Indicate the date controlling when an item is pulled from active sale. This field represents the last date for which orders are taken.
    Stop Servicing Date Specify the date controlling when an item is pulled from active service. This field contains the last date for which service is provided using the item.

  6. Click the Save button to save your information. Proceed to click the Close button to return to the Item page.
  7. The business date that you have created appears in the list of business dates versions.
