Associate an Image

To create a new item image, follow these steps:

  1. From the Overview page, click Commercial Modeling > Items.
  2. The Item page appears; perform a search and then double-click one item from the results list (for example, Phone Offer).
  3. Click the Images tab and then click the New button.
  4. The Item Image page appears. Prior creating an image item association, you must create an association of type image.

  5. Enter the information in the following fields:

    6. Field Description
    Code This is the code used to identify the item image. This code is not restricted to alphanumeric characters, it can also contain a combination of spaces, hyphens, etc. Strings can be up to 32 characters long.
    Type This is a list of association types of category Item image.
    Status A list of available Status for this image (for example, Definition and Active).
    Cancel Objects that are extended have this Cancel flag checked. If the flag is checked, then the extended item does not inherit that association.
    Name The name of the image can be used as a visual label for internal use when designing your catalog model (for example, Internet Offer Bandwidth). This field is not supported by language translation.
    Start Date The effective date of this image. Refer to Start and End dates for details.
    End Date The end date of this image represents the date when this image is no longer effective. Refer to Start and End dates for details.
    Label The visual label used for this image, which is used in the application user interface. This field is a translational string that supports language translations (refer to Velocity Studio's Translation Element for more details).
    Size Select a size either Normal or Large from this drop-down list.
    Project Each object is assigned to a Project. This defaults to the active or opened project when the image is first created.
    Classification Types Select different classification types to identify item under different environments.

  6. Click the Browse button; the File Upload dialog appears.
  7. Choose an image file (for example, offer.png) and then click the Open button.
  8. Click the Upload button.
  9. The image appears on the item image page and a confirmation message displays that the item image has been saved successfully.

Alternatively, you can also use the item palette to create an item association.
