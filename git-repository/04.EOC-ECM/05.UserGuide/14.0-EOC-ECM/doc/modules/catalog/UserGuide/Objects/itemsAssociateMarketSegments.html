Associate a Market Segment

To associate a market segment to an item, complete these steps:

  1. Open your project and click Commercial Modelling > Items from the Overview page.
  2. From the Item page, perform a search to a list of items, and then double-click the item that you want.
  3. Click the Market Segments tab, and then click New.
  4. The Market Segment page appears.



  5. You can specify the following fields:

    Field Description
    Rule Code Enter the rule code for your market segment.
    Project Code By default, this field contains the project code for the market segment.
    Start Date This field represents the start (effective) date that the market segment is available. Refer to Start and End dates for details.
    End Date This field indicates the date when the market segment is no longer effective. Refer to Start and End dates for details.
    Status This field denotes a list of available statuses for the market segment (for example, Definition and Active).
    Rule operator Specify the rule operator type (for example, Include or Exclude).
    Location You can add a location by clicking the Add button to the right of this field, selecting the location from the Availability section's Favourite or All tab, and dragging the location to this field. To remove values in this field, click the Close button to the right of this field.
    Sales Channel You can add a sales channel by clicking the Add button to the right of this field, selecting the sales channel from the Availability section's Favourite or All tab, and dragging the location to this field. To remove values in this field, click the Close button to the right of this field.
    Customer Segment You can add a customer segment by clicking the Add button to the right of this field, selecting the customer segment from the Availability section's Favourite or All tab, and dragging the location to this field. To remove values in this field, click the Close button to the right of this field.
    Customer Type You can add a customer type by clicking the Add button to the right of this field, selecting the customer type from the Availability section's Favourite or All tab, and dragging the location to this field. To remove values in this field, click the Close button to the right of this field.

  6. Click the Save button to save your information. Proceed to click the Close button to return to the Item page.
  7. The market segment that you have created appears in the list of business dates versions.
