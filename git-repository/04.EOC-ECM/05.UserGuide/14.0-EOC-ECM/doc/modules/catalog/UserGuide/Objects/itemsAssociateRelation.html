Associate a Relation

To associate a relation to an item, complete these steps:

  1. Open your project and click Commercial Modelling > Items from the Overview page.
  2. From the Item page, perform a search to a list of items, and then double-click the item that you want.
  3. Click the Relation tab to launch the Relation page.
  4. From the Item Relations section, select the relation that you want and then drag it to the List of Relations section.



  5. The Relation page appears.



    You can specify the following fields:

    Field Description
    Target This field represents the target item.
    Code This field denotes the code associated with the target item.
    Type This field indicates the type of relation.
    Status This field denotes a list of available statuses for the relation (for example, Definition and Active).
    Name This field contains the unique name for the relation.
    Start Date This field represents the start (effective) date that the relation is available. Refer to Start and End dates for details.
    End Date This field indicates the date when the relation is no longer effective. Refer to Start and End dates for details.
    Label This field contains the field's visual label displayed at runtime, which is used in the application user interface.
    Cancel By default, this property is unchecked. To cancel an inherited relationship, select this checkbox.
    Min Quantity This field contains the minimum item quantity.
    Max Quantity This field denotes the maximum item quantity.
    Default Cardinality You can provide a default cardinality, allowing you to specify the number of items added to the basket with a number in the item relation. If the parent item is added into the basket and the item relation association type is contains, the same number of child items is automatically added into the basket.
    Project This field indicates with which project the relation is associated.

  6. Click the Save button to save your information. A confirmation appears, indicating that you have successfully saved your relation. When you return to the Relations page for your item, the List of Relations section contains your newly created relation.
