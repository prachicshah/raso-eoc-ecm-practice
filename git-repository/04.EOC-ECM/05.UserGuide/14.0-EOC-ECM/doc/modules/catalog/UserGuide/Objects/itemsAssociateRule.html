Associate a Rule

To associate a rule to an item, complete these steps:

  1. Open your project and click Commercial Modelling > Items from the Overview page.
  2. From the Item page, perform a search to a list of items, and then double-click the item that you want.
  3. Click the Rules tab to launch the Relation page.
  4. From the Item Rules section, select the rule that you want and then drag it to the List of Conditions section.



  5. The Item Rule page appears.



    You can specify the following fields:

    Field Description
    Code This field denotes the code associated with the item rule.
    Rule Name This field contains the rule name.
    Type This field indicates the type of rule.
    Status This field denotes a list of available statuses for the item rule (for example, Definition and Active).
    Start Date This field represents the start (effective) date that the rule is available. Refer to Start and End dates for details.
    End Date This field indicates the date when the rule is no longer effective. Refer to Start and End dates for details.
    Label This field contains the field's visual label displayed at runtime, which is used in the application user interface.
    Cancel By default, this property is unchecked. To cancel an inherited relationship, select this checkbox.
    Message Code This field contains the error message code to invoke when the rule returns false.
    Sequence This field contains a number value used for condition sorting.
    Project This field indicates with which project the item rule is associated.

  6. Click the Save button to save your information. A confirmation appears, indicating that you have successfully saved your rule. When you return to the Rules page for your item, the List of Conditions section contains your newly created item rule.
