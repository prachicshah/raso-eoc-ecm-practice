Item - Change History

You can view the change history of an item. To access the change history of an item, follow these steps:

  1. From the Overview page, click Commercial Modeling > Items in the Quick start section.
  2. The Item page appears; enter the search criteria.
  3. Click the Search button; the item results displays in the results area.
  4. Select an item from the list (for example, Phone Offer) and then click the Change History button.
  5. The Change History Results page displays a summary of the changes for the selected Item.

Refer to the Change History document for more details.
