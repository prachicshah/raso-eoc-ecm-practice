Item - Show Usage

To view the usage of an item, follow these steps:

  1. From the Overview page, click Commercial Modeling > Items in the Quick start section.
  2. The Item page appears; click the Search button to search the items.
  3. Select one item (for example, Phone Offer) from the list, and then click the Show Usage button.
  4. The Usage Finder displays which Items are associated with the selected item.

  5. The Usage Finder displays which Items or objects are associated with the selected item.
