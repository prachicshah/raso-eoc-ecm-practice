View InfoTable Details

To view details of an info table, follow these steps:

Note: A project must be opened to view details of an info table.

  1. On the Info Model Detail dialog, click the Info Tables tab.
  2. The list of info table details appears; select one from the list (for example, 8/14/2013 16).
  3. Click the Detail button.

  4. The Info Table dialog appears with the details of selected info table.

  5. Click the Save button, if any changes are done.
