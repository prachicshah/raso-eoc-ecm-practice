Catalog Designer

The Catalog Designer allows you to define products and services (referred to as Products), their logical and physical components, eligibility and availability rules, pricing, and any offers on these products. The key differentiating factor is the ability to drive not only order entry, but also fulfillment activities, from the catalog.

This guide describes how to use the Catalog Designer to define, configure, and maintain your products and services. The guide's information is organized as follows:

Catalog Overview
This section describes new features that have been added to Catalog Designer and the benefits of using the application. It also provides a high-level overview of client and server applications, as well as catalog implementation.

Legal Matters
See this section for information about trademarks and ancillary software before using Catalog Designer.

Getting Started
The getting started section shows you how to create a project to begin using the Catalog Designer. It also allows you to become familiar with the application's user interface and menu options.

Catalog Designer
The options within the Catalog Designer menu allow you to define, control, and maintain catalog objects to represent the products and services offered by your company.

Catalog Setup and Configuration
The Catalog Designer allows you to configure various aspects of your catalog, including configuring catalog privileges, and importing and exporting catalog data.

Catalog Tools
Catalog Designer contains a number of tools as you design and configure your catalog, such as the tools event log and the ability to switch between the Catalog Designer and Administration applications.

Test Mode
Test mode provides you with a test interface to evaluate offer and product selection, as well as charging. This section shows you how to start and use test mode.

Catalog Rules
The catalog rules section contains an overview of the types of rules used in Catalog Designer and describes how to use the catalog rule language to create customized catalog rules.

Appendices
