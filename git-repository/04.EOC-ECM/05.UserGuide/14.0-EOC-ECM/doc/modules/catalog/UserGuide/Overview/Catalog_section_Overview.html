Catalog Overview

The Catalog Designer provides you with a number of tools to design and implement a catalog model. The Catalog Designer allows you to define, introduce, manage, and retire product offerings. It also helps you to define, control, and maintain Catalog objects, which have properties and attributes associated with them.
