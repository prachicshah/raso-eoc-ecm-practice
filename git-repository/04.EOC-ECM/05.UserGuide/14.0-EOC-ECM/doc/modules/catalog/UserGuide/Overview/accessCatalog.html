Access the Catalog Application

By default, the Catalog application is available to the User Profile Administrator user with either the Catalog Management Administrator or Catalog Management User privilege.

To log in to the application, complete these steps:

  1. In your Web browser, access the Catalog application by entering http://localhost:8080/cwf/ as the URL, and then selecting the Catalog Designer application.
  2. Enter upadmin as both your Username and Password, and then press the Enter key.

Note: The login screen differs, depending on whether you have selected the classic Select Application page or have opted to use the icon version of this page instead. See the Velocity Studio User Guide > Metadata Objects > Metadata Root > Metadata Root Application Page > Select Application Page: Classic and Icon for details.

Open a Project

You must have an existing project open to configure catalog elements and objects.

  1. Click No Current Project from the menu bar and select Manage to open the Project page
  2. Select the checkbox next to the project and click Open.
  3. The project opens and its name appears after Current Project: in the menu bar.
  4. You are now ready to begin defining the catalog by either clicking the Overview link to return to the main page or by select objects from the Designer menu on selected pages.
