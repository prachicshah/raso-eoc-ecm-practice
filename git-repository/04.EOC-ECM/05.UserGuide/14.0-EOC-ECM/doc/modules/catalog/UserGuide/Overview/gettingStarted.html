Getting Started Overview

This section provides information to familiarize yourself with Catalog Designer, and to get started with designing and implementing a catalog model. This overview section contains the following information about working with Catalog Designer:

Access Catalog
This page provides key steps to using Catalog Designer, including logging in to the application, and opening an existing project.

Catalog User Interface
This page describes Catalog Designer's toolbar, which features four main areas:

  • Catalog Designer- contains catalog definitions
  • Configuration - contains items used to configure your catalog project and data
  • Tools - contains options for managing your catalog
  • Test Mode - contains menu controls when testing the catalog

Catalog Controls
This page explains the different icons and buttons that are found within different Catalog dialogs. This section also provides a description of each button or icon's functionality.

Catalog Search
This page describes how to conduct an informational search on an object using the Search dialog and where to access this dialog within Catalog Designer.

Item Tree User Interface
This page describes the items that appear in the item tree node and how to view these items graphically in a model.

Export Results to a File
This section describes how you can export finder results to a file.
