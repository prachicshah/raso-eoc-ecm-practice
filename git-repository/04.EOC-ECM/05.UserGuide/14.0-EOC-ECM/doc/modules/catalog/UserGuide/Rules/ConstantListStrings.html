Constant list of strings

The constant list of strings does not have any methods.

Number

The number data type does not have any methods. It supports all arithmetic and comparison operations.

Constant list of numbers

The constant list of numbers does not have any methods.

Boolean

The Boolean data type does not have any methods. It supports all Boolean operations, including the equal and not equal operations.

Row

The info row object is result of the Info table select operation. It is a read-only object that allows access to the corresponding Info table column values by name using dot notation. The columns are simple String or Number data types. See Info models and Info tables for details.

Context

Context is a generic object that contains other objects accessible by name using dot notation. The catalog application that uses the catalog basket APIs assigns context variable values.
