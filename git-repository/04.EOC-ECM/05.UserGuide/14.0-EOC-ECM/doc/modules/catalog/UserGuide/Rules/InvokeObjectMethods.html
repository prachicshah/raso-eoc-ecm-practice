Invoke object methods

Objects have methods (object functions) that perform specific actions. The object methods may or may not have parameters. When a method is invoked, parameters have to be given in the order they are specified in the method definition.Â All parameters must be specified.

The method invocation format is as follows:

identifier.methodname(expression,  expression, ...)

where each expression gives a value for the corresponding parameter.

If the method does not have any parameters, parentheses still have to be used:

identifier.method name()

The following example uses the length method that does not have any parameters:

  s = s.append("ABC");
  len = s.length() + 5;
