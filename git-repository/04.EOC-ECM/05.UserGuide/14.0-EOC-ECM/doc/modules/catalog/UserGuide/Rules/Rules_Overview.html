Catalog Rules

Rules that can be called from other rules are called catalog rules, which are global in nature and must have unique names. The Catalog Designer allows you to define catalog rule parameters and return values.

The Catalog Designer allows you to use to create customized catalog rules. This section contains the following information to help create your catalog rules:

Catalog Rule Types
There are two types of catalog rules:

  • Object rules, which are strictly called by the Catalog product
  • Catalog rules, which define a rich set of business rules that can be attached to the Catalog elements

Catalog Tasks
You can perform these catalog rule tasks:

  • Open the Catalog Rules page
  • Create a new catalog rule
  • Edit a catalog rule
  • Delete a catalog rule
  • View a catalog rule's change history

Catalog Rule Language
The Catalog Rule Language allows programmers and nonprogrammers to create and customize catalog rules. The language offers alternative forms of rule statements, and introduces many non-procedural constructs that reduce rule size and complexity.
