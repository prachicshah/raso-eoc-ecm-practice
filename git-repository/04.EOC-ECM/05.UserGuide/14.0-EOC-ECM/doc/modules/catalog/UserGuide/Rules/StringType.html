String data type

The String data type has the following methods:

Method Description
length() Returns the number of characters in the string.
substring(start position, end position)  Returns a new string, which is the substring of the given one. The start and end positions start from 0. The start position is the position of the first character of the substring; the end position is the position of the character after the substring's last character. If the end position is not specified, the substring takes all characters from the start position to the end of the string.
append(string). Returns a new string, which concatenates the current string and the string specified as the parameter.
appendObject (any object). Returns a new string, which concatenates the current string and the string presentation of the parameter.
indexOf(string)  Finds the position of the parameter string within the string (starting from 0) or returns a negative number if no match is found.

The string data type supports only the comparison operations equal and not equal.
